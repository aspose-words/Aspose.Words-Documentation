Ujistěte se, že váš stroj splňuje [požadavek](/words/cpp/system-requirements/) než začnete.

Tento článek vysvětluje, jak nainstalovat Aspose.Words pro C++ do počítače.

## Nainstalujte nebo aktualizujte Aspose.Words pro C++ pomocí balíčků Manage NuGet {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet je nejjednodušší způsob stažení a instalace [Aspose.Words pro C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. Chcete-li to provést, postupujte takto:

1. Otevřete *Microsoft Visual Studio* a *Manage NuGet Packages* z nabídky a otevřete správce balíčků
2. Vyhledejte "aspose" nebo " aspose.slova " k nalezení požadovaného Aspose API<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. Vyberte požadovaný Aspose API a klikněte na "Instalovat"

Vybraný API bude stažen a odkazován ve vašem projektu.

Můžete také aktualizovat Aspose.Words pro C++ na požadovanou verzi pomocí *Manage NuGet Packages*.

## Nainstalujte nebo aktualizujte Aspose.Words pro C++ Pomocí Package Manager Console

Můžete také nainstalovat nebo aktualizovat Aspose.Words pro C++ APIs Pomocí *Package Manager Console*. Chcete-li to provést, postupujte takto:

1. Otevřete *Microsoft Visual Studio* a *Package Manager Console* z nabídky a otevřete konzolu správce balíčků
2. Zadejte příkaz `Install-Package Aspose.Words.Cpp -Version 21.3.0` a stisknutím klávesy Enter nainstalujte nejnovější úplnou verzi do své aplikace
3. Po stažení se zobrazí potvrzovací zprávy<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. Nyní můžete zjistit, že Aspose.Words byl úspěšně přidán a odkazován ve vaší aplikaci pro vás

V *Package Manager Console* můžete také použít příkaz `Update-Package Aspose.Words.Cpp` ke kontrole aktualizací balíčku Aspose.Words a jejich instalaci, pokud jsou k dispozici.

## Nainstalujte Aspose.Words pro C++ ručně

Můžete si stáhnout [Aspose.Words pro C++ z NuGet Galerie](https://www.nuget.org/packages/Aspose.Words.Cpp/) kliknutím na odkaz "ruční stahování" v sekci Informace. Offline API dokumentaci najdete také ve staženém balíčku ZIP.

Chcete-li ve svých aplikacích použít Aspose.Words pro C++, postupujte takto:

1. Stáhněte si nejnovější soubory
2. Rozbalte složku do produkčního prostředí
3. Chcete-li použít Aspose.Words pro C++, Podívejte se do složek Include a Lib v projektu
