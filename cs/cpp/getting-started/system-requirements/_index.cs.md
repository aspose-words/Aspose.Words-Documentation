## Rozvoj

Aspose.Words pro C++ můžete použít při vývoji aplikací pro Windows, Linux nebo macOS.

Aspose.Words pro C++ lze použít k vývoji aplikací v jakémkoli vývojovém prostředí, které podporuje [Microsoft Visual Studio V143 sada nástrojů platformy](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017), ale prostředí uvedená v následující tabulce jsou výslovně podporována:

### Další závislost na Linux

Aspose.Words pro C++ na Linux závisí na [náběh](https://www.freedesktop.org/wiki/Software/fontconfig/) binární soubory dynamická knihovna i nástroj. Před použitím jej nainstalujte:

1. Instalace fontconfig na Ubuntu nebo Debian</br>
`sudo apt install libfontconfig fontconfig`
2. Instalace fontconfig na Fedora nebo CentOs</br>
   `sudo yum install fontconfig`
