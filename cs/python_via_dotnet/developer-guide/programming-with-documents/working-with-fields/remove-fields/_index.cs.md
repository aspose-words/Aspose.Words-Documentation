Někdy je nutné z dokumentu odstranit pole. K tomu může dojít, pokud má být nahrazen jiným typem pole nebo pokud pole již není v dokumentu zapotřebí. Například `TOC` pole při ukládání do HTML.

Odstranit pole vložené do dokumentu pomocí [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/), použít vrácené [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/) objekt, který poskytuje pohodlné [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) způsob snadného odstranění pole z dokumentu.
