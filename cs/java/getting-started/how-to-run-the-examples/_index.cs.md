## Stáhnout z GitHub

Všechny příklady Aspose.Words místo Java jsou hostovány na [Github](https://github.com/aspose-words/Aspose.Words-for-Java). Můžete buď naklonovat repozitář pomocí svého oblíbeného Github klienta, nebo stáhnout ZIP soubor z [Tady](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

Extrahovat obsah ZIP souboru do jakékoli složky ve vašem počítači. Všechny příklady jsou umístěny v **Examples** Složka.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

Projekt využívá Maven vybudovat systém. Jakýkoli moderní IDE může projekt a jeho závislost snadno otevřít nebo importovat. Níže vám ukážeme, jak používat populární IDE ke stavbě a spuštění příkladů.

## IntelliJ IDEA

Klikněte na **File** menu a vybrat **Open**. Procházet se do složky projektu a vybrat **pom.xml** Složka.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

Otevře projekt a automaticky stáhne závislosti. Z záložky Projekt prohlédněte příklady v **src/main/java** Složka. Chcete-li spustit příklad, klikněte pravým tlačítkem myši na soubor a zvolte "Run .." příklad bude proveden a výstup bude zobrazen vestavěném výstupním okně konzole.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

Klikněte na **File** menu a vybrat **Import**. Vybrat **Maven** - Existující Maven Projekty.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

Prohlédněte si složku, ze které jste naklonovali nebo stáhli GitHub a vybrat **pom.xml** Složka.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

Otevře projekt a automaticky stáhne závislosti. Z záložky Package Explorer prohlédněte příklady v **src/main/java** Složka. Chcete-li spustit příklad, stačí kliknout pravým tlačítkem na soubor a vybrat **Run As** - **Java Application**, bude proveden příklad a výstup bude zobrazen vestavěn v výstupním okně konzole.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

Klikněte na **File** menu a vybrat **Open Project**. Prohlédněte si složku, ze které jste naklonovali nebo stáhli GitHub. Ikona **Examples** Složka ukáže, že jeho Maven projekt. Vyberte příklady a otevřete je.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

Otevře projekt a automaticky stáhne závislosti. Z záložky Projekty, procházet příklady v **zdrojové obaly**. Chcete-li spustit příklad, stačí kliknout pravým tlačítkem na soubor a vybrat **Run File**, bude proveden příklad a výstup bude zobrazen vestavěn v výstupním okně konzole.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)

## Přispějte

Pokud chcete přidat nebo vylepšit příklad, doporučujeme vám přispět k projektu. Všechny příklady a ukázkové projekty v tomto repozitáři jsou otevřeným zdrojem a mohou být volně používány ve vlastních aplikacích.

Chcete-li přispět, můžete vidlička úložiště, editovat zdrojový kód a vytvořit požadavek vytáhnout. Změny přezkoumáme a zahrneme je do úložiště, pokud nám to pomůže.
