Můžete snadno vědět, zda dokument je vyroben pomocí Aspose.Words místo Java. Můžete také najít číslo verze Aspose.Words místo Java který byl použit k vytvoření konkrétního výstupního dokumentu. Aspose.Words přímo zapíše tyto informace do vytvořeného dokumentu.

Číslo verze je uvedeno ve formátu *YY.MM.N* nebo *YY.M.N*, Například 23.12.0 nebo 24.1.0.<br/>
Tady. *YY* je poslední dvě číslice roku vydání, *M* nebo *MM* je měsíc vydání (1/9212) a *N* je malé číslo vydání. Obvykle, *N* je "0."

{{% alert color="primary" %}}

Všimněte si, že nemůžete poučovat Aspose.Words změnit nebo odstranit tyto informace z výstupních dokumentů.

{{% /alert %}}
