Někdy je nutné z dokumentu odstranit pole. K tomu může dojít, pokud má být nahrazen jiným typem pole nebo pokud již není pole v dokumentu potřebné. Například `TOC` pole při ukládání do HTML.

Odstranit pole vložené do dokumentu pomocí [DocumentBuilder](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/).[InsertField](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/#insertField-int-boolean), použít vrácené [Field](https://reference.aspose.com/words/java/com.aspose.words/field/) objekt, který poskytuje pohodlné [Remove](https://reference.aspose.com/words/java/com.aspose.words/field/#remove) způsob snadného odstranění pole z dokumentu.
