Добре дошли в Aspose.Words вместо Java!

Aspose.Words е библиотека от клас, която позволява на Вашето заявление да изпълнява голямо разнообразие от задачи по обработка на документи. Наричаме го програмируем редактор на документи.

Използване Aspose.Words, можете да създавате, модифицирате, конвертирате, правите и печатате документи, без да разчитате на приложения на трети страни, като например Microsoft Word, или Office Automation.

Струва си да се отбележи, че можете да конвертирате документ от всеки поддържан формат на натоварване в поддържан формат за запис, например, Microsoft Word формати за PDF, HTML, или Markdown, HTML към DOCX и така нататък. Прочетете повече за конвертирането на документи в [Конвертиране на документ](/words/bg/java/convert-a-document/) Секция.

Друга популярна опция на нашата библиотека е *Доклади и поща*. Нашите потребители генерират много сложни индивидуално коригирани финансови, медицински и други *Reports*. По този начин генерираният документ изглежда, че сте създали доклада ръчно с помощта на редовен текстов редактор. Друг вариант, *Mail*, осигурява способността много бързо да се генерират хиляди и милиони прости, но все пак лични документи, базирани на шаблони.

В опит да се направи API Колкото е възможно по-ясно и очевидно, решихме да признаем и почетем общото Java практики за разработване на платформи. Така че, Aspose.Words вместо Java следва насоките за кодиране, широко приети от Java разработчици.

## Ползи от използването Aspose.Words

Използване Aspose.Words вместо Java във вашия проект ви дава следните ползи:

* Изпълнение и мащабируемост
* Богат набор функция
* Платформа независимост
* Независимост от молбите на трети страни
* Минимална крива на обучение

### Богат набор от функции

Aspose.Words вместо Java Характеристиките могат да бъдат разделени на четири основни области.

#### Преобразуване

Възможността за лесно и надеждно конвертиране на документи е една от основните характеристики на Aspose.Words.

Цялостно [Зареждане, спестяване и преобразуване](/words/bg/java/loading-saving-and-converting/) с [Поддържани формати на документи](/words/bg/java/supported-document-formats/) позволява на потребителите да конвертират документи от [един популярен формат](https://reference.aspose.com/words/java/com.aspose.words/loadformat/) до [another](https://reference.aspose.com/words/java/com.aspose.words/saveformat/), например от DOCX до PDF, JPEG, или Markdown.

#### Document Object Model (DOM)

Програматичен достъп чрез богат API на всички елементи на документи и форматиране позволява да се създаде, модифицира, извлече, копира, раздели, да се присъединят и замени съдържанието на документи. Разберете повече в [Aspose.Words Document Object Model (DOM)](/words/bg/java/aspose-words-document-object-model/) Секция.

#### Нанасяне

Използваме термина [Нанасяне](/words/bg/java/rendering/) да опише процеса на преобразуване на документ във файлов формат или медиум, въображаем или който има концепцията за страници.

Можете да конвертирате цели документи или конкретни страници в PDF или XPS да генерира документи от страна на сървъра, да ги печата или да прави страници. И всичко това с висока вярност, точно като Microsoft Word щеше да го направи.

#### Докладване

Генериране на документи или доклади от нулата или чрез попълване на шаблони с данни от източници на данни или бизнес обекти. За повече информация относно докладването и извличането на данни от базите данни вижте [Mail Merge и докладване](/words/java/mail-merge-and-reporting/) както и [ЛИНК ДОКАДВАРИТЕЛ](/words/java/linq-reporting-engine/) части.

### Независимост на платформата

Aspose.Words вместо Java обхваща всички Java Околна среда за развитие и платформи за разполагане. Нашата API може да се използва за разработване на приложения за широк спектър от операционни системи, като Windows, Linux, Mac OS, Android, и различни платформи.

Да го кажа просто, ако вашата платформа може да работи стандарт <span notrans="<span notrans=" Java"=""></span>,"> Може да работи. Aspose.Words вместо Java. Ако вашата молба може да се нарече стандарт Java <span notrans="<span notrans=" API"=""></span>,"> Вие няма да имате проблеми да се обадя Aspose.Words вместо Java API от молбата ви.

В момента основната ни линия е JDK1.7 и по-нова. Но ние все още подкрепяме JDK1.6.

Имаме три различни версии, базирани на една и съща кодова база:

* Aspose.Words вместо Java
* Aspose.Words вместо Android през Java
* Aspose.Words вместо JasperReports

Докато Java както и Android за съответната платформа се съставят версии, Aspose.Words вместо JasperReports е просто дебела опаковка около горно Aspose.Words вместо Java. Това е само за удобство. Можете също така ръчно интегриране на вашия JasperReports с Aspace Words год. по конкретен начин, който е по - подходящ за вашите нужди.

### Независимост от други заявления

Aspose.Words не изисква приложения на трети страни, например, Microsoft Офис, който да бъде инсталиран на машината, за да работи. Всички компоненти на базата са напълно независими. Това прави Aspose.Words чудесна алтернатива на автоматизацията по отношение на сигурността, стабилността, мащабируемостта/скоростта, цената и характеристиките за работа с документи и свързани с тях задачи.

### Ефективност и скалабилност

Aspose.Words е проектиран да изпълнява значително както на сървър, така и на клиент. Aspose.Words е единичен Java буркан, който може да се разгърне с всеки Java прилагане чрез просто копиране. Можете също така да добавите `HarfBuzz` плъгин за напреднали Характеристики на шрифта, ако имате нужда от него (зареден в отделен буркан). Стандарт Java ImageIO JAI разширение също може да се добави към вашата платформа за подкрепа на допълнително ImageIO codecs (няма нужда от съвременни Java среди).

С Aspose.Words, можете буквално да генерирате хиляди документи за минути, което включва отваряне на документ, промяна на съдържанието и форматиране или популиране с данни и спасяването му. Дори най-сложните документи се отварят и съхраняват на P4 2.4Ghz 1Gb RAM машина в фракции на секунда.

Aspose.Words е многосфера безопасно, докато само една нишка работи върху документ в даден момент. Това е типичен сценарий да имаме една нишка, работеща върху един документ. Различни нишки могат безопасно да работят върху различни документи в същото време.

Въпреки че има над 150 обществени класа и изброявания в <span notrans="<span notrans=" Aspose.Words"=""></span>,"> кривата на обучение е минимална, тъй като Aspose.Words API е внимателно проектирана със следните цели:

Разработки, които преди това са използвали Microsoft Word Автоматизацията в техните проекти ще намери имената и поведението на много класове, методи и свойства познати.

[Document](https://reference.aspose.com/words/java/com.aspose.words/document/), [Paragraph](https://reference.aspose.com/words/java/com.aspose.words/paragraph/), [Bookmark](https://reference.aspose.com/words/java/com.aspose.words/bookmark/), [Range](https://reference.aspose.com/words/java/com.aspose.words/range/), [Section.PageSetup](https://reference.aspose.com/words/java/com.aspose.words/section/#getPageSetup) както и [Paragraph.ParagraphFormat](https://reference.aspose.com/words/java/com.aspose.words/paragraph/#ParagraphFormat) са някои от Aspose.Words Класове.
