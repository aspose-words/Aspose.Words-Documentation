Aspose.Words اکنون از مدل های پیشرفته AI برای بهبود پردازش و تجزیه و تحلیل اسناد استفاده می کند. با قابلیت مبتنی بر Aspose.Words AI، توسعه دهندگان می توانند به ویژگی هایی دسترسی داشته باشند که از AI برای وظایف مانند خلاصه و تجزیه و تحلیل اسناد، ادغام مدل های زبان مولد از رهبران صنعت استفاده می کنند.

Aspose.Words کتابخانه در حال حاضر از ادغام با OpenAI و مدل های زبان google generative پشتیبانی می کند. این مدل ها قابلیت های قدرتمندی را برای کار با اسناد به ارمغان می آورند. با فعال کردن تعامل بدون درز با این ابزارها AI، Aspose.Words وظایف پیچیده را ساده می کند و بهره وری را برای توسعه دهندگان افزایش می دهد.

شما می توانید یک نمونه مدل با استفاده از روش [Create](https://reference.aspose.com/words/java/com.aspose.words/aimodel/#create-int) کلاس [AiModel](https://reference.aspose.com/words/java/com.aspose.words/aimodel/) ایجاد کنید. لیست کامل انواع مدل های پشتیبانی شده در فهرست [AiModelType](https://reference.aspose.com/words/java/com.aspose.words/aimodeltype/) ذکر شده است.

مثال کد زیر نشان می دهد که چگونه یک نمونه جدید از کلاس **AiModel** ایجاد کنیم.

{{< highlight java>}}
public static AiModel create(int modelType)
{{< /highlight >}}
