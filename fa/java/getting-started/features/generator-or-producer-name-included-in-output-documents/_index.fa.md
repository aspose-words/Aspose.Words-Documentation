شما به راحتی می توانید بدانید که آیا یک سند با استفاده از Aspose.Words برای Java تولید می شود. همچنین می توانید شماره نسخه Aspose.Words را برای Java پیدا کنید که برای ایجاد یک سند خروجی خاص استفاده شده است. Aspose.Words مستقیما این اطلاعات را در سند تولید شده می نویسد.

شماره نسخه در فرمت *YY.MM.N* یا *YY.M.N* مشخص شده است، به عنوان مثال، 23.12.0 یا 24.1.0.<br />
در اینجا *YY* دو رقم آخر سال انتشار، *M* یا *MM* ماه انتشار (1-12) و *N* شماره انتشار جزئی است. معمولا، *N* "0" است.

{{% alert color="primary" %}}

توجه داشته باشید که شما نمی توانید به Aspose.Words دستور دهید که این اطلاعات را از اسناد خروجی تغییر یا حذف کند.

{{% /alert %}}
