Aspose.Words به کاربران اجازه می دهد تا با اسناد در فرمت های مختلف کار کنند و طیف گسترده ای از گزینه ها را برای این کار فراهم می کند. به عنوان مثال، کاربران می توانند یک سند را از ابتدا ایجاد کنند یا یک فایل موجود را در هر [فرمت واردات پشتیبانی شده](/words/java/supported-document-formats/)، برخی تغییرات را در آن سند ایجاد کنید و سپس نتیجه را ذخیره کنید. هنگام وارد کردن یک سند در قالب انتخاب شده، کاربر می تواند گزینه های مختلف بارگذاری را اعمال کند.

{{% alert color="primary" %}}

مهم است که بدانید که بسیاری از گزینه های بارگذاری برای همه فرمت های پشتیبانی شده مشترک هستند و فقط برخی از گزینه ها خاص به فرمت هستند، مانند PDF, HTML, RTF, و دیگران.

{{% /alert %}}
