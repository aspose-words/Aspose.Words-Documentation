## دانلود بازی From GitHub

همه نمونه ها Aspose.Words برای Java میزبانی شده در [Github](https://github.com/aspose-words/Aspose.Words-for-Java)... شما می توانید مخزن را با استفاده از مشتری Github مورد علاقه خود کلون کنید یا فایل ZIP را از طریق فایل ZIP دانلود کنید. [اینجا اینجا](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip)...

محتوای فایل ZIP را به هر پوشه ای بر روی کامپیوتر خود استخراج کنید. تمام نمونه ها در داخل آن قرار دارند. **Examples** پوشه

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

پروژه استفاده می کند Maven ساخت سیستم هر IDE مدرن به راحتی می تواند پروژه و وابستگی های آن را باز یا وارد کند. در زیر ما به شما نشان می دهیم که چگونه از IDE های محبوب برای ساخت و اجرای نمونه ها استفاده کنید.

## IntelliJ IDEA

Click on the **File** منو و انتخاب **Open**... مرور به پوشه پروژه و انتخاب پوشه **pom.xml** فایل

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

این پروژه را باز می کند و وابستگی ها را به طور خودکار دانلود می کند. از برگه پروژه، نمونه ها را در فهرست کنید **src/main/java** پوشه برای اجرای یک مثال، فقط روی فایل کلیک کنید و گزینه "Run." را انتخاب کنید، مثال اجرا خواهد شد و خروجی در پنجره خروجی کنسول ساخته خواهد شد.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

Click on **File** منو و انتخاب **Import**... گزینه Select **Maven** وجود Maven پروژه ها

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

به پوشه ای که از آن استفاده کرده اید یا دانلود کرده اید GitHub انتخاب کنید **pom.xml** فایل

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

این پروژه را باز می کند و وابستگی ها را به طور خودکار دانلود می کند. از زبانه Package Explorer، نمونه ها را در فهرست کنید **src/main/java** پوشه برای اجرای یک مثال، فقط روی فایل کلیک کنید و انتخاب کنید **Run As** - **Java Application**, مثال اجرا خواهد شد و خروجی در پنجره خروجی کنسول ساخته خواهد شد.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

Click on the **File** منو و انتخاب **Open Project**... به پوشه ای که از آن استفاده کرده اید یا دانلود کرده اید GitHub... نماد **Examples** پوشه نشان می دهد که Maven پروژه نمونه ها را انتخاب کنید و آن را باز کنید.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

این پروژه را باز می کند و وابستگی ها را به طور خودکار دانلود می کند. از زبانه پروژه ها، نمونه ها را در فهرست کنید **بسته های منبع**... برای اجرای یک مثال، فقط روی فایل کلیک کنید و انتخاب کنید **Run File**, مثال اجرا خواهد شد و خروجی در پنجره خروجی کنسول ساخته خواهد شد.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)
