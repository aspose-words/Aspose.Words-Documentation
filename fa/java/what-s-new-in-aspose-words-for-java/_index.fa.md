## Aspose.Words برای Java 24.1، 24.3

Aspose.Words 24.1 بهبود تجربه در مورد مدیریت رنگ های سکته، افزایش اشیاء OLE و گزارش LINQ، و همچنین معرفی یک منبع جدید Bibliography عمومی API...

Aspose.Words 24.2 نمودار گسترش یافته API, مدیریت سبک و گزینه های LINQ این نسخه Aspose.Words همچنین قابلیت مشخص کردن SvgSaveاختیاری در هنگام ارائه، بارگیری کنترل انعطاف پذیر تر Markdown فایل ها و کار با متن مرجع برای یادداشت های پا و پایان نامه.

Aspose.Words 24.3 معرفی یک TIFF Reader/Writer و Emulation عملیات دودویی برای متافایل های WMF. Aspose.Words 24.3 همچنین به گسترش نمودارها ادامه می دهد API...

توانایی بارگذاری و تبدیل اسناد PDF به فرمت های صفحه ثابت با وفاداری بالا و عملکرد اجرا شده است.

#### تبدیل به PDF /UA-1 CSSompliant با WCAG 2.0 <sup>22.7</sup>

پشتیبانی از تبدیل یک سند به فرمت PDF /UA-1، مطابق با WCAG 2.0، اضافه شده است.

بنابراین، اگر مشتری یک سند Word قابل دسترس داشته باشد و آن را به PDF/UA-1 از طریق تبدیل کند. Aspose.Words با اشاره به جزئیات تبدیل، خروجی WCAG 2.0 سازگار خواهد بود.

WCAG یا Web Content Accessibility دستورالعمل ها مجموعه ای از دستورالعمل های توسعه یافته توسط W3C با همکاری افراد و سازمان ها در سراسر جهان است. حالا با Aspose.Words شما می توانید سند خود را به فرمت PDF /UA-1 تبدیل کنید که برای ساخت اسناد سازگار با WCAG 2.0 مناسب است.

شایان ذکر است که WCAG 2.0 دارای دو مورد اضافی است که در مشخصات PDF /UA-1 مشخص نشده است:

- الزامات کنتراست
- الزامات برای تگ کردن محتوای چند رسانه ای

اما هر دو الزامات مربوط به مورد ما برای تبدیل از Word به PDF نیست.
