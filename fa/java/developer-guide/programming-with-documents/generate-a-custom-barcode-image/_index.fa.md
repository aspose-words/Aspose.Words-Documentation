{{% alert color="primary" %}}

Aspose.Words دارای یک رابط برای تولید بارکد های سفارشی است که استفاده از آن را بسیار آسان می کند [Aspose.Words](https://products.aspose.com/words/java/) و [بازی Punch BarCode](https://products.aspose.com/barcode/java/) با هم برای ارائه تصاویر بارکد در اسناد خروجی. به عنوان مثال، شما می توانید یک سند DOC، OOXML یا RTF را که حاوی آن است بارگیری کنید. `DISPLAYBARCODE` Field Aspose.Words, پیاده سازی ژنراتور بارکد سفارشی را ارائه دهید و فرمت های صفحه ثابت مانند PDF را ذخیره کنید. XPS و غیره یک معمولی `DISPLAYBARCODE` فیلد دارای نحو زیر است:

{ `DISPLAYBARCODE` "SomeData" QR \h 720 }

{{% /alert %}}
