## عملیات با Cloning Files

عملیات کلونی به شما اجازه می دهد تا فرآیند ساخت اسناد را سریعتر کنید زیرا نیازی به بارگیری و تجزیه یک سند از یک فایل در هر زمان نخواهید داشت.
