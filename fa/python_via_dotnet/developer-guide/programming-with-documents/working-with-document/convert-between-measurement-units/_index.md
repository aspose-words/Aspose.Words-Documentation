بسیاری از ویژگی های شی ارائه شده در Aspose.Words API که نشان دهنده برخی اندازه گیری ها هستند، مانند عرض یا ارتفاع، حاشیه ها و فواصل مختلف، مقادیر را در نقاطی می پذیرند که در آن 1 اینچ برابر با 72 نقطه است. گاهی اوقات این کار راحت نیست و باید امتیازها به واحدهای دیگر تبدیل شوند.

Aspose.Words کلاس [ConvertUtil](https://reference.aspose.com/words/python-net/aspose.words/convertutil/) را ارائه می دهد که توابع کمکی را برای تبدیل بین واحدهای اندازه گیری مختلف فراهم می کند. تبدیل اینچ، پیکسل و میلی متر به نقطه، نقطه به اینچ و پیکسل و تبدیل پیکسل ها از یک وضوح به وضوح دیگر را امکان پذیر می کند. تبدیل پیکسل ها به نقاط و بالعکس را می توان با وضوح 96 dpi (نقطه در اینچ) یا وضوح dpi مشخص انجام داد.

کلاس [ConvertUtil](https://reference.aspose.com/words/python-net/aspose.words/convertutil/) مخصوصاً هنگام تنظیم ویژگی‌های مختلف صفحه مفید است، زیرا، برای مثال، اینچ‌ها واحدهای اندازه‌گیری رایج‌تر از نقاط هستند.
