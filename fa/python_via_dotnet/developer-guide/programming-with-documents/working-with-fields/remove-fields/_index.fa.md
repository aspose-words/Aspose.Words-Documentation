گاهی اوقات لازم است یک فیلد از سند حذف شود. این ممکن است زمانی رخ دهد که قرار است با یک نوع فیلد دیگر جایگزین شود یا زمانی که فیلد دیگر در سند مورد نیاز نیست. به عنوان مثال یک فیلد `TOC` هنگام ذخیره در HTML.

برای حذف یک فیلد درج شده در سند با استفاده از [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/)، از شی [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/) برگشتی استفاده کنید، که روش [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) راحت را برای حذف آسان فیلد از سند ارائه می دهد.
