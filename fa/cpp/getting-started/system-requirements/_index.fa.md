## محیط توسعه

شما می توانید استفاده کنید Aspose.Words برای C++ هنگام توسعه برنامه های کاربردی برای Windows، Linux یا macOS.

Aspose.Words برای C++ می تواند برای توسعه برنامه های کاربردی در هر محیط توسعه ای که از [Microsoft Visual Studio مجموعه ابزار پلتفرم V143](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017)، اما محیط های ذکر شده در جدول زیر به طور صریح پشتیبانی می شوند:

### وابستگی اضافی به Linux

Aspose.Words برای C++ در Linux بستگی دارد [fontconfig](https://www.freedesktop.org/wiki/Software/fontconfig/) باینری هر دو کتابخانه پویا و ابزار. لطفا قبل از استفاده آن را نصب کنید:

1. نصب fontconfig بر روی Ubuntu یا Debian</br>
`sudo apt install libfontconfig fontconfig`
2. نصب fontconfig بر روی Fedora یا CentOs</br>
`sudo yum install fontconfig`

Aspose.Words برای C++ در macOS بستگی دارد [fontconfig](https://www.freedesktop.org/wiki/Software/fontconfig/) باینری هر دو کتابخانه پویا و ابزار. لطفا قبل از استفاده آن را نصب کنید:

`brew install fontconfig`
