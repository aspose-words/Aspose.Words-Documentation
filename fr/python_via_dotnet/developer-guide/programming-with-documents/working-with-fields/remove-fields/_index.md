Il est parfois nécessaire de supprimer un champ du document. Cela peut se produire lorsqu'il doit être remplacé par un type de champ différent ou lorsque le champ n'est plus nécessaire dans le document. Par exemple un champ `TOC` lors de l'enregistrement au format HTML.

Pour supprimer un champ inséré dans un document à l'aide de [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/), utilisez l'objet [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/) renvoyé, qui fournit la méthode [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) pratique pour supprimer facilement le champ du document.
