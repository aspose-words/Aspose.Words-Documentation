Vous pouvez facilement savoir si un document est produit en utilisant Aspose.Words pour Python via .NET. Vous pouvez également trouver le numéro de version de Aspose.Words pour Python via .NET utilisé pour créer un document de sortie particulier. Aspose.Words écrit directement ces informations dans le document généré.

Le numéro de version est spécifié au format *YY.MM.N* ou *YY.M.N*, par exemple 23.12.0 ou 24.1.0.<br />
Ici, *YY* correspond aux deux derniers chiffres de l'année de sortie, *M* ou *MM* correspond au mois de sortie (1 à 12) et *N* est le numéro de version mineure. Habituellement, *N* vaut "0".

{{% alert color="primary" %}}

Notez que vous pouvez demander à Aspose.Words de supprimer ces informations des documents de sortie.

{{% /alert %}}
