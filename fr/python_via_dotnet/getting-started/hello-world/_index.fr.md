Un message "Bonjour le monde!" Le code est souvent le premier exemple simple pour écrire "Aspose.Words pour Python via .NET", et il peut également être utilisé comme test d'intégrité pour garantir que le logiciel destiné à compiler ou à exécuter le code source est correctement installé.

La bibliothèque "Aspose.Words for Python via .NET" donne aux développeurs un accès direct pour créer, modifier, fusionner, convertir, comparer des documents Word et Web. PDF, DOCX, DOC, RTF, ODT, EPUB, HTML et bien d'autres formats de fichiers sont [Formats de documents pris en charge](/words/fr/python-net/supported-document-formats/).

L'extrait Below code suit ces étapes:

1. Créez un nouveau [Document](https://reference.aspose.com/words/python-net/aspose.words/document/) vide
1. Initialisez une classe [DocumentBuilder](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/)
1. Insérez du texte au début du document en utilisant la méthode [write](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/write/) simple
1. Ouvrez un [Document](https://reference.aspose.com/words/python-net/aspose.words/document/) existant à partir d'un fichier. Détecte automatiquement le format de fichier
1. Document [Append](https://reference.aspose.com/words/python-net/aspose.words/document/append_document/) "A" au et du document "B"
1. [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/) la sortie au format PDF
