Vous pouvez facilement savoir si un document est produit en utilisant Aspose.Words pour Java. Vous pouvez également trouver le numéro de version de Aspose.Words pour Java qui a été utilisé pour créer un document de sortie particulier. Aspose.Words écrit directement cette information dans le document généré.

Le numéro de version est spécifié dans le format *YY.MM.N* ou *YY.M.N*, Par exemple, 23.12.0 ou 24.1.0.<br/>
Ici. *YY* est les deux derniers chiffres de l'année de sortie, *M* ou *MM* est le mois de sortie (1–12), et *N* est le numéro de libération mineur. Habituellement, *N* est "0".

{{% alert color="primary" %}}

Notez que vous ne pouvez pas instruire Aspose.Words modifier ou supprimer ces informations des documents de sortie.

{{% /alert %}}
