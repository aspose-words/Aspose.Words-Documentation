{{% alert color="primary" %}}

Aspose.Words a une interface pour générer des codes-barres personnalisés qui le rend très facile à utiliser [Aspose.Words](https://products.aspose.com/words/java/) et [Pose-toi. Code barre](https://products.aspose.com/barcode/java/) ensemble pour rendre les images de code-barres dans les documents de sortie. Par exemple, vous pouvez charger un document DOC, OOXML ou RTF contenant `DISPLAYBARCODE` Champ dans Aspose.Words, fournir votre implémentation de générateur de code-barres personnalisé et enregistrer dans des formats de page fixes tels que PDF, XPS etc. Une caractéristique `DISPLAYBARCODE` champ a la syntaxe suivante:

{ `DISPLAYBARCODE` "SomeData" QR \h 720 }

{{% /alert %}}

Ci-dessous est un générateur de code d'échantillon qui met à profit le `Aspose.BarCode` API. Cet exemple montre comment insérer des images de code-barres à `DISPLAYBARCODE` Position du champ dans le document Word en utilisant Aspose.Words et `Aspose.BarCode` APIPar.
