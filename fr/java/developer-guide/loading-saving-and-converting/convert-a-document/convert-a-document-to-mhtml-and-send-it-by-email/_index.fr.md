Aspose.Words peut convertir n'importe quel document au format MHTML (archive Web). Cela rend pratique l'utilisation conjointe de Aspose.Words et `Aspose.Email`. Vous pouvez charger un document prédéfini dans n'importe quel format pris en charge, tel que DOC, OOXML ou RTF, dans Aspose.Words, le remplir de données, enregistrer le document résultant sous MHTML, puis l'envoyer par e-mail en utilisant `Aspose.Email`.
