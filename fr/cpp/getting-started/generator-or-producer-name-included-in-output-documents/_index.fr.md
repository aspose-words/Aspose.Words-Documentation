Vous pouvez facilement savoir si un document est produit en utilisant Aspose.Words pour C++. Vous pouvez également trouver le numéro de version de Aspose.Words pour C++ cela a été utilisé pour créer un document de sortie particulier. Aspose.Words écrit directement ces informations dans le document généré.

Le numéro de version est spécifié au format *YY.MM.N* ou *YY.M.N*, par exemple 23.12.0 ou 24.1.0.<br />
Ici *YY* est les deux derniers chiffres de l'année de publication, *M* ou *MM* est le mois de publication (1-12) et *N* est le numéro de version mineure. Habituellement, *N* est "0".

{{% alert color="primary" %}}

Notez que vous ne pouvez pas demander à Aspose.Words de modifier ou de supprimer ces informations des documents de sortie.

{{% /alert %}}
