## Environnement de Développement

Vous pouvez utiliser Aspose.Words pour C++ lors du développement d'applications pour Windows, Linux ou macOS.

### Windows

Aspose.Words pour C++ peut être utilisé pour développer des applications dans n'importe quel environnement de développement prenant en charge [Ensemble d'outils de plate-forme Microsoft Visual Studio v143](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017), mais les environnements répertoriés dans le tableau suivant sont explicitement pris en charge:

### Dépendance supplémentaire sur Linux

Aspose.Words pour C++ sur Linux dépend de [fontconfig](https://www.freedesktop.org/wiki/Software/fontconfig/) binaires à la fois bibliothèque dynamique et outil. Veuillez l'installer avant de l'utiliser:

1. Installation de fontconfig sur Ubuntu ou Debian</br>
`sudo apt install libfontconfig fontconfig`
2. Installation de fontconfig sur Fedora ou CentOs</br>
`sudo yum install fontconfig`
