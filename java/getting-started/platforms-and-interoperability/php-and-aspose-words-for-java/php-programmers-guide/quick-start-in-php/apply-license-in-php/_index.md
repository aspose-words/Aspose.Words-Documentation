## Aspose.Words - Apply License

To set license of Aspose.Words Java in php, simply invoke the setLicense() method of License class.

**Php Code**

{{< highlight csharp >}}
 $license = new Java('com.aspose.words.License');
$license->setLicense("Aspose.Words.lic");
{{< /highlight >}}
