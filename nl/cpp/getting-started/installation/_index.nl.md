Zorg ervoor dat uw machine voldoet aan de [systeemvereisten](/words/cpp/system-requirements/) voordat je begint.

In dit artikel wordt uitgelegd hoe u Aspose.Words voor C++ op uw computer installeert.

## Installeer of Update Aspose.Words voor C++ met Beheer NuGet pakketten {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet is de makkelijkste manier om te downloaden en te installeren [Aspose.Words voor C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. Volg hiervoor deze stappen:

1. Open *Microsoft Visual Studio* en *Manage NuGet Packages* vanuit het menu om pakketbeheer te openen
2. Zoek naar "aspose" of "aspose".woorden " om de gewenste Aspose APIte vinden<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. Selecteer de gewenste Aspose API en klik op "Installeren"

De geselecteerde API wordt gedownload en verwezen in uw project.

U kunt Aspose.Words voor C++ ook bijwerken naar de gewenste versie met behulp van *Manage NuGet Packages*.

## Installeer of Update Aspose.Words voor C++ met behulp van Package Manager Console

U kunt ook Aspose.Words installeren of bijwerken voor C++ APIs met behulp van de *Package Manager Console*. Volg hiervoor deze stappen:

1. Open *Microsoft Visual Studio* en *Package Manager Console* vanuit het menu om package manager console te openen
2. Typ de opdracht `Install-Package Aspose.Words.Cpp -Version 21.3.0` en druk op Enter om de nieuwste volledige versie in uw toepassing te installeren
3. Eenmaal gedownload, ziet u bevestigingsberichten<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. Nu kunt u merken dat Aspose.Words is toegevoegd en verwezen in uw toepassing voor u

In *Package Manager Console* kunt u ook het `Update-Package Aspose.Words.Cpp` commando gebruiken om te controleren op updates voor het Aspose.Words pakket en deze indien aanwezig te installeren.

## Aspose.Words voor C++ handmatig installeren

U kunt downloaden [Aspose.Words voor C++ van NuGet Galerij](https://www.nuget.org/packages/Aspose.Words.Cpp/) door te klikken op de" handleiding downloaden " link in de Info sectie. Offline API documentatie is ook te vinden in het gedownloade ZIP pakket.

Als u Aspose.Words wilt gebruiken voor C++ in uw toepassingen, volgt u deze stappen:

1. Download de nieuwste bestanden
2. Unzip de map in een productieomgeving
3. Als u Aspose.Words wilt gebruiken voor C++, raadpleegt u de mappen Include En Lib in uw project
