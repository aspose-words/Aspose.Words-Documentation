Soms is het noodzakelijk om een veld uit het document te verwijderen. Dit kan gebeuren wanneer het wordt vervangen door een ander veldtype of wanneer het veld niet meer nodig is in het document. Bijvoorbeeld a `TOC` veld bij opslaan naar HTML.

Een veld in een document verwijderen met behulp van [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/), gebruik de geretourneerde [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/) object, dat het handige [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) methode om het veld gemakkelijk uit het document te verwijderen.
