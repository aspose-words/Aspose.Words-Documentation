{{% alert color="primary" %}}

Aspose.Words heeft een interface voor het genereren van aangepaste barcodes waardoor het zeer gemakkelijk te gebruiken [Aspose.Words](https://products.aspose.com/words/java/) en [Stel. Barcode](https://products.aspose.com/barcode/java/) samen barcodebeelden in uitvoerdocumenten weergeven. U kunt bijvoorbeeld een DOC, OOXML of RTF document laden dat `DISPLAYBARCODE` Veld naar Aspose.Words, bieden uw implementatie van aangepaste barcode generator en opslaan naar vaste paginaformaten zoals PDF, XPS enz. Een typisch `DISPLAYBARCODE` veld heeft de volgende syntax:

{ `DISPLAYBARCODE` "SomeData" QR \h 720 }

{{% /alert %}}
