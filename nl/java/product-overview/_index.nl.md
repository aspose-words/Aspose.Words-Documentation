Welkom bij Aspose.Words voor Java!

Aspose.Words is een cross-platform klasse bibliotheek die uw applicatie in staat stelt om een breed scala aan documentenverwerking taken uit te voeren. We noemen het de Programmeerbare Documenteditor.

Gebruik Aspose.Words, u kunt documenten maken, wijzigen, converteren, renderen en afdrukken zonder te vertrouwen op toepassingen van derden, zoals Microsoft Word, of Office Automation.

Het is de moeite waard op te merken dat u een document kunt converteren van een ondersteund laadformaat naar een ondersteund opslagformaat, bijvoorbeeld, Microsoft Word formaten naar PDF, HTML, of Markdown, HTML naar DOCX, enzovoort. Lees meer over het omzetten van documenten in de [Een document omzetten](/words/nl/java/convert-a-document/) Section.

Een andere populaire optie van onze bibliotheek is *Reports and Mails*. Onze gebruikers genereren zeer complexe individueel aangepaste financiële, medische en andere *Reports*. Hierbij ziet het gegenereerde document eruit alsof u het rapport handmatig hebt gemaakt met behulp van een reguliere teksteditor. Een andere optie, *Mail*, biedt de mogelijkheid om zeer snel duizenden en miljoenen eenvoudige maar nog steeds persoonlijke template gebaseerde documenten te genereren.

In een poging om de API Zo duidelijk en voor de hand liggend mogelijk, besloten we om de gemeenschappelijke Java platformontwikkelingspraktijken. Dus, Aspose.Words voor Java volgt codering richtlijnen algemeen aanvaard door Java ontwikkelaars.

## Voordelen van het gebruik Aspose.Words

Gebruik Aspose.Words voor Java in uw project geeft u de volgende voordelen:

* Prestaties en schaalbaarheid
* Rich feature set
* Platform onafhankelijkheid
* Onafhankelijkheid van toepassingen van derden
* Minimale leercurve

### Rijke set van functies

Aspose.Words voor Java Kenmerken kunnen worden onderverdeeld in vier hoofdgebieden.

#### Omrekening

Het vermogen om gemakkelijk en betrouwbaar documenten te converteren is een van de belangrijkste kenmerken van Aspose.Words.

Uitgebreide [Laden, opslaan en omzetten](/words/nl/java/loading-saving-and-converting/) met [Onderwijsdocumentformaten](/words/nl/java/supported-document-formats/) stelt gebruikers in staat om documenten van [één populair Formaat](https://reference.aspose.com/words/java/com.aspose.words/loadformat/) tot [another](https://reference.aspose.com/words/java/com.aspose.words/saveformat/), bijvoorbeeld van DOCX naar PDF, JPEG, of Markdown.

#### Document Object Model (DOM)

Programmatische toegang via een rijke API naar alle documentelementen en opmaak maakt het mogelijk om documentinhoud aan te maken, aan te passen, uit te pakken, te kopiëren, te splitsen, samen te voegen en te vervangen. Meer informatie vindt u in de [Aspose.Words Document Object Model (DOM)](/words/nl/java/aspose-words-document-object-model/) Section.

#### Renderen

We gebruiken de term [Renderen](/words/nl/java/rendering/) beschrijven van het proces van het omzetten van een document in een bestandsformaat of een medium, gepagineerd of dat het concept van pagina's heeft.

U kunt hele documenten of bepaalde pagina's converteren naar PDF of XPS om server-side documenten te genereren, af te drukken of pagina's te renderen. En dit alles met een hoge trouw precies zoals Microsoft Word Zou het gedaan hebben.

#### Rapportage

Genereer documenten of rapporten vanaf nul of vul templates met gegevens uit gegevensbronnen of zakelijke objecten. Voor meer informatie over rapportage en het ophalen van gegevens uit databases, zie de [Mail Merge en relatie](/words/java/mail-merge-and-reporting/) en [LINQ Rapportagemotor](/words/java/linq-reporting-engine/) secties.

### Onafhankelijkheid van andere toepassingen

Aspose.Words geen toepassingen van derden vereist, bijvoorbeeld, Microsoft Kantoor, te installeren op de machine om te werken. Alle Aspose componenten zijn volledig onafhankelijk. Dit maakt Aspose.Words een groot alternatief voor automatisering op het gebied van veiligheid, stabiliteit, schaalbaarheid/snelheid, prijs en functies voor het werken met documenten en aanverwante taken.

### Prestaties en schaalbaarheid

Aspose.Words is ontworpen om sterk uit te voeren op zowel een server als client. Aspose.Words is een enkele Java pot die kan worden ingezet met elke Java toepassing door gewoon kopiëren. U kunt ook de `HarfBuzz` plugin voor gevorderden Lettertype functies als je het nodig hebt (geplaatst in een aparte pot). Standaard Java ImageIO Jai-extensie kan ook worden toegevoegd aan uw platform ter ondersteuning van extra ImageIO codecs (niet nodig in moderne Java milieu).

Met Aspose.Words, U kunt letterlijk duizenden documenten in minuten genereren, wat inhoudt dat u een document opent, inhoud wijzigt en formatteert of het bevolkt met gegevens, en het opslaat. Zelfs de meeste complexe documenten worden geopend en opgeslagen op een P4 2.4Ghz 1Gb RAM machine in fracties van een seconde.

Aspose.Words is multithread veilig zolang slechts één draad aan een document tegelijk werkt. Dit is een typisch scenario om één thread aan één document te laten werken. Verschillende draden kunnen veilig werken aan verschillende documenten tegelijkertijd.

### Minimale leercurve

Hoewel er meer dan 150 openbare klassen en opsommingen in <span notrans="<span notrans=" Aspose.Words"=""></span>,"> de leercurve is minimaal omdat de Aspose.Words API zorgvuldig is ontworpen met de volgende doelen in het achterhoofd:

* Leng beste praktijken van andere bekende APIs Microsoft Word Automatisering
* Leen beste praktijken van de Java Richtsnoeren voor het ontwerp
* Zorg voor een evenwicht tussen gebruiksgemak en gedetailleerde manipulatie van documentelementen

Ontwikkelaars die eerder hebben gebruikt Microsoft Word Automatisering in hun projecten vindt de namen en het gedrag van vele klassen, methoden en eigenschappen bekend.

[Document](https://reference.aspose.com/words/java/com.aspose.words/document/), [Paragraph](https://reference.aspose.com/words/java/com.aspose.words/paragraph/), [Bookmark](https://reference.aspose.com/words/java/com.aspose.words/bookmark/), [Range](https://reference.aspose.com/words/java/com.aspose.words/range/), [Section.PageSetup](https://reference.aspose.com/words/java/com.aspose.words/section/#getPageSetup) en [Paragraph.ParagraphFormat](https://reference.aspose.com/words/java/com.aspose.words/paragraph/#ParagraphFormat) zijn sommige van de Aspose.Words lessen. Tegelijkertijd, Aspose.Words is heel anders dan de Microsoft Word Object Model in dat het een document als een boom van objecten vertegenwoordigt, meer als een XML DOM boom. Als u met een XML hebt gewerkt DOM bibliotheek, vindt u het is gemakkelijk te begrijpen en werken met Aspose.Words.
