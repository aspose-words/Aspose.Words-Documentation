## Downloaden van GitHub

Alle voorbeelden van Aspose.Words voor Java worden gehost op [Github](https://github.com/aspose-words/Aspose.Words-for-Java). Je kunt de repository klonen met je favoriete Github client of het ZIP bestand downloaden van [Hier](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

Pak de inhoud van ZIP-bestand naar elke map op uw computer. Alle voorbeelden bevinden zich in de **Examples** map.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

Het project gebruikt Maven bouwsysteem. Elke moderne IDE kan het project en zijn afhankelijkheden eenvoudig openen of importeren. Hieronder laten we je zien hoe je populaire IDE's gebruikt om de voorbeelden te bouwen en uit te voeren.

## IntelliJ IDEA

Klik op de **File** menu en kies **Open**. Blader naar de projectmap en selecteer de **pom.xml** bestand.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

Het zal het project openen en de afhankelijkheden automatisch downloaden. Van het tabblad Project, blader door de voorbeelden in **src/main/java** map. Om een voorbeeld uit te voeren, klikt u met de rechtermuisknop op het bestand en kiest u "Run .." Het voorbeeld wordt uitgevoerd en de uitvoer wordt getoond in het ingebouwde console uitvoervenster.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

Klik op **File** menu en kies **Import**. Selecteren **Maven** - Bestaande Maven Projecten.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

Blader naar de map waarvan u hebt gekloond of gedownload GitHub en selecteer **pom.xml** bestand.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

Het zal het project openen en de afhankelijkheden automatisch downloaden. Vanuit het tabblad Pakket Explorer, blader door de voorbeelden in **src/main/java** map. Om een voorbeeld uit te voeren, klik met de rechtermuisknop op het bestand en kies **Run As** - **Java Application**, het voorbeeld zal uitgevoerd worden en de uitvoer zal getoond worden in het ingebouwde console uitvoervenster.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

Klik op de **File** menu en kies **Open Project**. Blader naar de map waarvan u hebt gekloond of gedownload GitHub. Het pictogram van **Examples** map toont dat het een Maven project. Selecteer Voorbeelden en open het.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

Het zal het project openen en de afhankelijkheden automatisch downloaden. Van het tabblad Projecten, blader door de voorbeelden in **bronpakketten**. Om een voorbeeld uit te voeren, klik met de rechtermuisknop op het bestand en kies **Run File**, het voorbeeld zal uitgevoerd worden en de uitvoer zal getoond worden in het ingebouwde console uitvoervenster.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)

## Bijdragen

Als u een voorbeeld wilt toevoegen of verbeteren, raden wij u aan om bij te dragen aan het project. Alle voorbeelden en showcase projecten in deze repository zijn open source en kunnen vrij gebruikt worden in je eigen applicaties.

Om bij te dragen kun je de repository forken, de broncode bewerken en een pull request aanmaken. We zullen de wijzigingen bekijken en opnemen in de repository indien nuttig.
