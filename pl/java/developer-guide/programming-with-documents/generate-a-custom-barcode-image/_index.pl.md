{{% alert color="primary" %}}

Aspose.Words posiada interfejs do generowania własnych kodów kreskowych, co sprawia, że bardzo łatwy w użyciu [Aspose.Words](https://products.aspose.com/words/java/) oraz [Złote świętowanie. Kod kreskowy](https://products.aspose.com/barcode/java/) razem do renderowania obrazów kodów kreskowych w dokumentach wyjściowych. Na przykład można załadować dokument DOC, OOXML lub RTF zawierający `DISPLAYBARCODE` Pole Aspose.Words, zapewnić wykonanie własnego generatora kodów kreskowych i zapisać do stałych formatów stron, takich jak PDF, XPS itd. Typowe `DISPLAYBARCODE` pole posiada następującą składnię:

{ `DISPLAYBARCODE` "SomeData" QR \h 720 }
