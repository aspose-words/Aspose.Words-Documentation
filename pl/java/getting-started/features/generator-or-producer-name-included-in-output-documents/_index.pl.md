Można łatwo dowiedzieć się, czy dokument jest wytwarzany za pomocą Aspose.Words zamiast Java. Można również znaleźć numer wersji Aspose.Words zamiast Java który został użyty do stworzenia określonego dokumentu wyjściowego. Aspose.Words bezpośrednio zapisuje te informacje do wygenerowanego dokumentu.

Numer wersji jest określony w formacie *YY.MM.N* lub *YY.M.N*, na przykład 23.12.0 lub 24.1.0.<br/>
Proszę. *YY* to dwie ostatnie cyfry roku wydania, *M* lub *MM* jest miesiącem wydania (1- 12), oraz *N* jest to nieznaczny numer wydania. Zazwyczaj, *N* jest "0".

{{% alert color="primary" %}}

Zauważ, że nie możesz uczyć Aspose.Words zmienić lub usunąć te informacje z dokumentów wyjściowych.

{{% /alert %}}
