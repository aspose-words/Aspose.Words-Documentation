## Pobierz z GitHub

Wszystkie przykłady Aspose.Words zamiast Java są hostowane na [Github](https://github.com/aspose-words/Aspose.Words-for-Java). Możesz albo sklonować repozytorium używając swojego ulubionego klienta Github albo pobrać plik ZIP z [Tutaj](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

Wyciągnij zawartość pliku ZIP do dowolnego folderu na komputerze. Wszystkie przykłady znajdują się w **Examples** folder.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

Projekt wykorzystuje Maven zbudować system. Każdy nowoczesny IDE może łatwo otworzyć lub zaimportować projekt i jego zależności. Poniżej pokazujemy jak używać popularnych IDEs do budowania i uruchamiania przykładów.

## IntelliJ IDEA

Kliknij na **File** menu i wybierz **Open**. Przeglądaj folder projektu i wybierz **pom.xml** plik.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

Otwiera projekt i automatycznie pobiera zależności. Z zakładki Projekt, przeglądaj przykłady w **src / main / java** folder. Aby uruchomić przykład, wystarczy kliknąć prawym przyciskiem myszy na plik i wybrać "Uruchom".., przykład zostanie wykonany, a wyjście zostanie pokazane w wbudowanym oknie wyjścia konsoli.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

Kliknij **File** menu i wybierz **Import**. Wybierz **Maven** - Istniejący Maven Projekty.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

Przeglądaj folder, który sklonowałeś lub ściągnąłeś GitHub i wybrać **pom.xml** plik.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

Otwiera projekt i automatycznie pobiera zależności. Z zakładki Pakiet Explorer, przeglądaj przykłady w **src / main / java** folder. Aby uruchomić przykład, wystarczy kliknąć prawym przyciskiem myszy na pliku i wybrać **Run As** - **Java Application**, przykład zostanie wykonany, a wyjście zostanie pokazane w wbudowanym oknie wyjścia konsoli.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

Kliknij na **File** menu i wybierz **Open Project**. Przeglądaj folder, który sklonowałeś lub ściągnąłeś GitHub. Ikona **Examples** folder pokaże, że jego Maven projekt. Wybierz przykłady i otwórz je.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

Otwiera projekt i automatycznie pobiera zależności. Z zakładki Projekty, przeglądaj przykłady w **pakiety źródłowe**. Aby uruchomić przykład, wystarczy kliknąć prawym przyciskiem myszy na pliku i wybrać **Run File**, przykład zostanie wykonany, a wyjście zostanie pokazane w wbudowanym oknie wyjścia konsoli.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)

## Wkład

Jeśli chcesz dodać lub ulepszyć przykład, zachęcamy do udziału w projekcie. Wszystkie przykłady i projekty prezentacyjne w tym repozytorium są otwarte i mogą być swobodnie wykorzystywane we własnych aplikacjach.

Aby wnieść wkład, możesz zawiesić repozytorium, edytować kod źródłowy i utworzyć żądanie ciągnięcia. Przejrzymy zmiany i włączymy je do repozytorium, jeśli okaże się pomocne.
