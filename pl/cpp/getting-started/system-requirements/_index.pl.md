Aspose.Words dla C++ nie wymaga instalacji żadnego produktu innej firmy, takiego jak Microsoft Word. Aspose.Words sam w sobie jest silnikiem do tworzenia, modyfikowania, konwertowania i renderowania dokumentów w różnych formatach, w tym w formatach dokumentów Microsoft Word.

## Środowisko Programistyczne

Możesz użyć Aspose.Words dla C++ podczas tworzenia aplikacji dla Windows, Linux lub macOS.

### Windows

Aspose.Words for C++ może być używany do tworzenia aplikacji w dowolnym środowisku programistycznym, które obsługuje [Microsoft Visual Studio Zestaw narzędzi platformy v143](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017), ale środowiska wymienione w poniższej tabeli są wyraźnie obsługiwane:

### Dodatkowa zależność od Linux

Aspose.Words dla C++ na Linux zależy od [fontconfig](https://www.freedesktop.org/wiki/Software/fontconfig/) pliki binarne zarówno biblioteka dynamiczna, jak i narzędzie. Proszę zainstalować go przed użyciem:

1. Instalowanie fontconfig na Ubuntu LUB Debian</br>
`sudo apt install libfontconfig fontconfig`
2. Instalowanie fontconfig na Fedora lub CentOs</br>
   `sudo yum install fontconfig`
