W tym artykule wyjaśniono, jak zainstalować Aspose.Words dla C++ na komputerze.

## Zainstaluj lub zaktualizuj Aspose.Words dla C++ za pomocą Zarządzaj NuGet pakiety {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet to najprostszy sposób na pobranie i zainstalowanie [Aspose.Words dla C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. Aby to zrobić, wykonaj następujące kroki:

1. Otwórz *Microsoft Visual Studio* i *Manage NuGet Packages* z menu, aby otworzyć menedżera pakietów
2. Wyszukaj " aspose "lub" aspose.słowa " aby znaleźć żądany Aspose API<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. Wybierz żądany Aspose API i kliknij "Zainstaluj"

Wybrany API zostanie pobrany i odwołuje się do twojego projektu.

Możesz także zaktualizować Aspose.Words dla C++ do żądanej wersji za pomocą *Manage NuGet Packages*.

## Zainstaluj lub zaktualizuj Aspose.Words dla C++ za pomocą Package Manager Console

Możesz także zainstalować lub zaktualizować Aspose.Words dla C++ APIs za pomocą *Package Manager Console*. Aby to zrobić, wykonaj następujące kroki:

1. Otwórz *Microsoft Visual Studio* i *Package Manager Console* z menu, aby otworzyć konsolę menedżera pakietów
2. Wpisz polecenie `Install-Package Aspose.Words.Cpp -Version 21.3.0` i naciśnij Enter, aby zainstalować najnowszą pełną wersję w aplikacji
3. Po pobraniu zobaczysz komunikaty potwierdzające<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. Teraz może się okazać, że Aspose.Words został pomyślnie dodany i odwołuje się w aplikacji dla Ciebie

W *Package Manager Console* Możesz również użyć polecenia `Update-Package Aspose.Words.Cpp`, aby sprawdzić dostępność aktualizacji pakietu Aspose.Words i zainstalować je, jeśli są obecne.

## Zainstaluj Aspose.Words dla C++ Ręcznie

Możesz pobrać [Aspose.Words dla C++ z NuGet galerii](https://www.nuget.org/packages/Aspose.Words.Cpp/) klikając link "pobieranie ręczne" w sekcji Informacje. Dokumentacja Offline API znajduje się również w pobranym pakiecie ZIP.

Aby użyć Aspose.Words dla C++ w swoich aplikacjach, wykonaj następujące kroki:

1. Pobierz najnowsze pliki
2. Rozpakuj folder do środowiska produkcyjnego
3. Aby użyć Aspose.Words dla C++, zapoznaj się z folderami Include I Lib w projekcie
