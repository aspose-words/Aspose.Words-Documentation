Pastikan mesin Anda memenuhi persyaratan [persyaratan sistem](/words/cpp/system-requirements/) sebelum Anda mulai.

Artikel ini menjelaskan cara menginstal Aspose.Words untuk C++ di komputermu.

## Instal atau Perbarui Aspose.Words untuk C++ menggunakan Kelola Paket NuGet {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet adalah cara termudah untuk mengunduh dan menginstal [Aspose.Words untuk C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. Untuk melakukannya, ikuti langkah-langkah berikut:

1. Buka *Microsoft Visual Studio* dan *Manage NuGet Packages* dari menu untuk membuka manajer paket
2. Cari "aspose "atau" aspose.kata-kata" untuk menemukan Aspose yang diinginkan API<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. Pilih Aspose API yang diinginkan dan klik "Instal"

API yang dipilih akan diunduh dan dirujuk dalam proyek Anda.

Anda juga dapat memperbarui Aspose.Words untuk C++ ke versi yang diinginkan menggunakan *Manage NuGet Packages*.

## Instal atau Perbarui Aspose.Words untuk C++ menggunakan Package Manager Console

Anda juga dapat menginstal atau memperbarui Aspose.Words untuk C++ APIs menggunakan *Package Manager Console*. Untuk melakukannya, ikuti langkah-langkah berikut:

1. Buka *Microsoft Visual Studio* dan *Package Manager Console* dari menu untuk membuka konsol manajer paket
2. Ketik perintah `Install-Package Aspose.Words.Cpp -Version 21.3.0` dan tekan Enter untuk menginstal rilis lengkap terbaru ke dalam aplikasi Anda
3. Setelah diunduh, Anda akan melihat pesan konfirmasi<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. Sekarang Anda mungkin menemukan bahwa Aspose.Words telah berhasil ditambahkan dan direferensikan dalam aplikasi Anda untuk Anda

Di *Package Manager Console*, Anda juga dapat menggunakan perintah `Update-Package Aspose.Words.Cpp` untuk memeriksa pembaruan pada paket Aspose.Words dan menginstalnya jika ada.

## Instal Aspose.Words untuk C++ Secara manual

Anda dapat mengunduh [Aspose.Words untuk C++ dari Galeri NuGet ](https://www.nuget.org/packages/Aspose.Words.Cpp/) dengan mengklik tautan "Unduh manual" di bagian Info. Dokumentasi API Offline juga dapat ditemukan dalam paket ZIP yang diunduh.

Untuk menggunakan Aspose.Words untuk C++ dalam aplikasi Anda, ikuti langkah-langkah berikut:

1. Unduh berkas terbaru
2. Buka zip folder ke lingkungan produksi
3. Untuk menggunakan Aspose.Words untuk C++, lihat folder Sertakan dan Lib di proyek Anda
