Terkadang perlu untuk menghapus bidang dari dokumen. Hal ini dapat terjadi bila bidang tersebut harus diganti dengan jenis bidang yang berbeda atau bila bidang tersebut tidak lagi diperlukan dalam dokumen. Misalnya kolom `TOC` saat menyimpan ke HTML.

Untuk menghapus bidang yang disisipkan ke dalam dokumen menggunakan [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/), gunakan objek [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/) yang dikembalikan, yang menyediakan metode [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) yang mudah untuk menghapus bidang dari dokumen dengan mudah.
