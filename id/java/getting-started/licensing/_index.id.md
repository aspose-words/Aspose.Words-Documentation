### Pilihan Lisensi

Lisensi dapat diterapkan dari berbagai lokasi:

* Jalan eksplisit
* Sitemap folder yang berisi *Aspose.Words JARLogin
* Sitemap Sumber daya yang tertanam di JAR SitemapAspose.Words JARSitemap
* Sitemap Sebagai Lisensi Meteran - mekanisme lisensi baru

Seringkali cara termudah untuk mengatur lisensi adalah untuk menempatkan file lisensi di folder yang sama *Aspose.Words JAR* dan tentukan hanya nama file tanpa path.

{{% alert color="primary" %}}

Gunakan [SetLicense](https://reference.aspose.com/words/java/com.aspose.words/license/#setLicense-java.io.InputStream) metode untuk lisensi komponen.

Login **SetLicense** beberapa kali tidak berbahaya, itu hanya membuang waktu prosesor.

Login [SetMeteredKey](https://reference.aspose.com/words/java/com.aspose.words/metered/#setMeteredKey-java.lang.String-java.lang.String) beberapa kali tidak berbahaya baik, tetapi hanya waktu prosesor limbah dan dapat mengumpulkan konsumsi dengan tidak benar.
