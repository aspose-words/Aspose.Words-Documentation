## Unduh dari GitHub

Semua contoh Aspose.Words untuk Java dihosting di [Github](https://github.com/aspose-words/Aspose.Words-for-Java). Anda dapat mengkloning repositori menggunakan klien Github favorit Anda atau mengunduh file ZIP dari [di sini](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

Ekstrak isi file ZIP ke folder mana pun di komputer Anda. Semua contoh berada di folder **Examples**.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

Proyek ini menggunakan sistem build Maven. Setiap IDE modern dapat dengan mudah membuka atau mengimpor proyek dan dependensinya. Di bawah ini kami tunjukkan cara menggunakan popular IDEs untuk membuat dan menjalankan contoh.

## IntelliJ IDEA

Klik menu **File** dan pilih **Open**. Jelajahi ke folder proyek dan pilih file **pom.xml**.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

Ini akan membuka proyek dan mengunduh dependensi secara otomatis. Dari tab Project, telusuri contoh di folder **src/main/java**. Untuk menjalankan contoh, cukup klik kanan pada file tersebut dan pilih " Run ..", contoh akan dieksekusi dan output akan ditampilkan di jendela output konsol bawaan.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

Klik menu **File** dan pilih **Import**. Pilih **Maven** - Proyek Maven yang ada.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

Telusuri ke folder yang Anda kloning atau unduh dari GitHub dan pilih file **pom.xml**.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

Ini akan membuka proyek dan mengunduh dependensi secara otomatis. Dari tab Package Explorer, telusuri contoh di folder **src/main/java**. Untuk menjalankan contoh, cukup klik kanan pada file dan pilih **Run As** - **Java Application**, contoh akan dijalankan dan hasilnya akan ditampilkan di jendela keluaran konsol bawaan.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

Klik menu **File** dan pilih **Open Project**. Telusuri ke folder yang Anda kloning atau unduh dari GitHub. Ikon folder **Examples** akan menunjukkan bahwa itu adalah proyek Maven. Pilih Contoh dan buka.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

Ini akan membuka proyek dan mengunduh dependensi secara otomatis. Dari tab Proyek, telusuri contoh dalam **source packages**. Untuk menjalankan contoh, cukup klik kanan pada file dan pilih **Run File**, contoh akan dijalankan dan hasilnya akan ditampilkan di jendela keluaran konsol bawaan.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)

## Berkontribusi

Jika Anda ingin menambahkan atau meningkatkan contoh, kami mendorong Anda untuk berkontribusi pada proyek tersebut. Semua contoh dan proyek showcase di repositori ini adalah open source dan dapat digunakan secara bebas di aplikasi Anda sendiri.

Untuk berkontribusi, Anda dapat melakukan fork repositori, mengedit kode sumber, dan membuat permintaan tarik. Kami akan meninjau perubahan dan memasukkannya ke dalam repositori jika dirasa bermanfaat.
