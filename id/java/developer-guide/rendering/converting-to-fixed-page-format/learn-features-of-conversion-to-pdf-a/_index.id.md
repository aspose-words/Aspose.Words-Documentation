PDF adalah format halaman tetap yang sangat populer di kalangan pengguna dan didukung secara luas oleh berbagai aplikasi, sebagai dokumen PDF terlihat sama pada perangkat apa pun. Untuk alasan ini, konversi ke PDF adalah fitur penting dari Aspose.WordsSitemap

PDF adalah format yang kompleks dengan sendirinya, karena memiliki struktur file tertentu, model grafis, font embedding, dan beberapa fungsi output yang kompleks seperti tag struktur dokumen, enkripsi, tanda tangan digital, dan bentuk yang dapat diedit. Selain itu, mengkonversi dokumen ke PDF memerlukan beberapa tahap perhitungan, yang kompleks dan memakan waktu.

Di bagian ini, kami akan mempertimbangkan masalah utama yang mungkin timbul ketika bekerja dengan dokumen dalam berbagai standar PDF dan menjelaskan opsi untuk memecahkannya.

## Login Login Aspose.Words Sitemap

Aspose.Words Sekarang memungkinkan pengguna untuk bekerja dengan format PDF / A-1, PDF / A-2 dan PDF / A-4, serta PDF / U-1:

* PDF / A-1 memiliki batasan serius seperti transparansi dan beberapa opsi kompresi dilarang
* PDF / A-2 menghilangkan beberapa keterbatasan PDF / A-1, seperti dukungan transparansi dan efek lapisan atau embedding font OpenType
* PDF / A-4 mengasumsikan tingkat kesesuaian yang direvisi: kesesuaian PDF / A-4 biasa setara dengan kesesuaian U versi sebelumnya, dan tingkat Kesesuaian dihapus
* Konten PDF / U-1 harus ditandai dan distandarisasi sesuai dengan ISO 32000-1: 2008

PDF/A adalah ISO-standarisasi versi PDF dimaksudkan untuk digunakan dalam lengkungan dan penyimpanan dokumen elektronik jangka panjang. Pada saat yang sama, PDF/UA adalah yang lain ISO versi standar PDF yang dirancang untuk memastikan aksesibilitas bagi orang dengan cacat yang menggunakan teknologi bantu. Untuk menentukan tingkat kepatuhan dengan standar PDF, gunakan [Compliance](https://reference.aspose.com/words/java/com.aspose.words/pdfsaveoptions/#getCompliance) Login Karena kondisi penyimpanan, dokumen PDF/A harus timbul semua font dan enkripsi yang dapat dinonaktifkan, sementara PDF/UA hanya boleh timbul semua font.

Di bagian ini, kita akan melihat lebih dekat bekerja dengan dokumen PDF/A atau PDF/UA-1.
