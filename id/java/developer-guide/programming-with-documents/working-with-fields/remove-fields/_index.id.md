Terkadang perlu menghapus bidang dokumen. Ini mungkin terjadi ketika harus diganti dengan jenis lapangan yang berbeda atau ketika bidang tidak diperlukan lagi dalam dokumen. Contohnya `TOC` field saat menyimpan ke HTML.

Untuk menghapus kolom yang dimasukkan ke dalam dokumen menggunakan [DocumentBuilder](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/)Sitemap[InsertField](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/#insertField-int-boolean), gunakan kembali [Field](https://reference.aspose.com/words/java/com.aspose.words/field/) objek, yang menyediakan [Remove](https://reference.aspose.com/words/java/com.aspose.words/field/#remove) metode untuk menghapus kolom dengan mudah dari dokumen.
