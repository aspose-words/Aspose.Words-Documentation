Microsoft Word dokumen dapat berisi berbagai karakter yang memiliki arti khusus. Biasanya mereka digunakan untuk memformat tujuan dan tidak ditarik dalam mode normal. Anda dapat membuat mereka terlihat jika Anda mengklik tombol Show/Hide Formatting Marks yang terletak di bilah alat Standar.

Kadang-kadang Anda mungkin perlu menambahkan atau menghapus karakter ke/dari teks. Misalnya, ketika memperoleh program teks secara tematik dari dokumen, Aspose.Words menjaga sebagian besar karakter kontrol, jadi jika Anda perlu bekerja dengan teks ini Anda mungkin harus menghapus atau mengganti karakter.

Login [ControlChar](https://reference.aspose.com/words/java/com.aspose.words/controlchar/) kelas adalah repositori untuk konstanta yang mewakili karakter kontrol sering ditemui dalam dokumen. Ini menyediakan versi char dan string dari konstanta yang sama. Misalnya, string [LineBreak](https://reference.aspose.com/words/java/com.aspose.words/controlchar/#LINE-BREAK) dan char [LineBreakChar](https://reference.aspose.com/words/java/com.aspose.words/controlchar/#LINE-BREAK_CHAR) memiliki nilai yang sama.
