Visual Basic untuk Aplikasi (VBA) untuk Microsoft Word adalah bahasa pemrograman sederhana namun kuat yang dapat digunakan untuk memperpanjang fungsi. Aspose.Words API menyediakan tiga kelas untuk mendapatkan akses ke kode sumber proyek VBA:

- Sitemap [VBAProject](https://reference.aspose.com/words/java/com.aspose.words/vbaproject/) kelas menyediakan akses ke informasi proyek VBA
- Sitemap [VBAModulesCollection](https://reference.aspose.com/words/java/com.aspose.words/vbamodulecollection/) kelas mengembalikan koleksi modul proyek VBA
- Sitemap [VbaModule](https://reference.aspose.com/words/java/com.aspose.words/vbamodule/) kelas menyediakan akses ke modul proyek VBA
