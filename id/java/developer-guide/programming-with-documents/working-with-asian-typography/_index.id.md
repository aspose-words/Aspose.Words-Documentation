Typografi Asia adalah satu set pilihan untuk paragraf teks dalam dokumen yang ditulis dalam bahasa Asia.

Aspose.Words mendukung Typografi Asia menggunakan [ParagraphFormat](https://reference.aspose.com/words/net/aspose.words/paragraphformat/) kelas dan beberapa sifatnya.

## Secara otomatis Menyesuaikan Ruang antara Asia dan Latin Teks atau Nomor

Jika Anda merancang template dengan teks Asia Timur dan Latin dan ingin meningkatkan tampilan template formulir Anda dengan mengendalikan ruang antara kedua jenis teks, Anda dapat mengkonfigurasi template formulir Anda untuk secara otomatis menyesuaikan ruang antara dua jenis teks ini. Untuk mencapai ini, Anda dapat menggunakan [AddSpaceBetweenFarEastAndAlpha](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getAddSpaceBetweenFarEastAndAlpha) Login [AddSpaceBetweenFarEastAndDigit](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getAddSpaceBetweenFarEastAndDigit) sifat `ParagraphFormat` Sitemap

## Set Line Break Pilihan

Tab Typografi Asia dari kotak dialog properti paragraf di Microsoft Word memiliki kelompok istirahat garis. Pilihan kelompok ini dapat diatur menggunakan [FarEastLineBreakControl](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getFarEastLineBreakControl), [WordWrap](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getWordWrap), [HangingPunctuation](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getHangingPunctuation) sifat **ParagraphFormat** Sitemap
