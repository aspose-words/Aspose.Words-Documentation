Sebuah font adalah set karakter dengan ukuran tertentu, warna, dan desain. Aspose.Words memungkinkan Anda untuk bekerja dengan font menggunakan berbagai kelas terkait font, termasuk [Font](https://reference.aspose.com/words/java/com.aspose.words/font/) Sitemap

## Sitemap

Format font saat ini diwakili oleh **Font** objek yang dikembalikan oleh [Font](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/#getFont) Login Login **Font** kelas mengandung berbagai macam properti font mungkin Microsoft WordSitemap

Contoh kode berikut menunjukkan cara mengatur format font:

Isi properti sekarang juga tersedia untuk font untuk mengatur format teks. Ini memberikan kemampuan untuk mengubah, misalnya, warna latar belakang atau transparansi isi teks.

## Mendapatkan Font Line Spacing

Jarak garis font adalah jarak vertikal antara dasar dua garis berturut-turut teks. Dengan demikian, jarak garis termasuk ruang kosong antara garis bersama dengan ketinggian karakter itu sendiri.

Login [LineSpacing](https://reference.aspose.com/words/java/com.aspose.words/font/#getLineSpacing) properti diperkenalkan di properti **Font** kelas untuk mendapatkan nilai ini seperti yang ditunjukkan dalam contoh yang diberikan di bawah ini:
