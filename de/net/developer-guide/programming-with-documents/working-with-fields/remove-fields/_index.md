Manchmal ist es notwendig, ein Feld aus einem Dokument zu entfernen. Dies kann auftreten, wenn es durch einen anderen Feldtyp ersetzt werden muss oder wenn das Feld im Dokument nicht mehr benötigt wird. Zum Beispiel das `TOC`-Feld beim Speichern in HTML.

Um ein mit [DocumentBuilder](https://reference.aspose.com/words/net/aspose.words/documentbuilder/).[InsertField](https://reference.aspose.com/words/net/aspose.words/documentbuilder/insertfield/) in ein Dokument eingefügtes Feld zu entfernen, verwenden Sie das zurückgegebene [Field](https://reference.aspose.com/words/net/aspose.words.fields/field/)-Objekt, das eine praktische [Remove](https://reference.aspose.com/words/net/aspose.words.fields/field/remove/)-Methode zum einfachen Entfernen des Felds aus dem Dokument bietet.
