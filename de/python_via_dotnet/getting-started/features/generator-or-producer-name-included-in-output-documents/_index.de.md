Mithilfe von Aspose.Words für Python via .NET können Sie leicht erkennen, ob ein Dokument erstellt wurde. Sie können auch die Versionsnummer von Aspose.Words für Python via .NET finden, die zum Erstellen eines bestimmten Ausgabedokuments verwendet wurde. Aspose.Words schreibt diese Informationen direkt in das generierte Dokument.

Die Versionsnummer wird im Format *YY.MM.N* oder *YY.M.N* angegeben, zum Beispiel 23.12.0 oder 24.1.0.<br />
Hier sind *YY* die letzten beiden Ziffern des Veröffentlichungsjahres, *M* oder *MM* der Veröffentlichungsmonat (1–12) und *N* die Nebenversionsnummer. Normalerweise ist *N* "0".

{{% alert color="primary" %}}

Beachten Sie, dass Sie Aspose.Words anweisen können, diese Informationen aus Ausgabedokumenten zu entfernen.

{{% /alert %}}
