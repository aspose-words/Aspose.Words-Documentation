Manchmal ist es notwendig, ein Feld aus dem Dokument zu entfernen. Dies kann auftreten, wenn das Feld durch einen anderen Feldtyp ersetzt werden soll oder wenn das Feld im Dokument nicht mehr benötigt wird. Zum Beispiel ein `TOC`-Feld beim Speichern in HTML.

Um ein mit [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/) in ein Dokument eingefügtes Feld zu entfernen, verwenden Sie das zurückgegebene [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/)-Objekt, das die praktische [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default)-Methode zum einfachen Entfernen des Felds aus dem Dokument bietet.
