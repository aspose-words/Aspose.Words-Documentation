Manchmal ist es notwendig, ein Feld aus einem Dokument zu entfernen. Dies kann auftreten, wenn es durch einen anderen Feldtyp ersetzt werden muss oder wenn das Feld im Dokument nicht mehr benötigt wird. Zum Beispiel das Feld `TOC` beim Speichern in HTML.

Um ein Feld zu entfernen, das mit [DocumentBuilder](https://reference.aspose.com/words/cpp/aspose.words/documentbuilder/) in ein Dokument eingefügt wurde.[InsertField](https://reference.aspose.com/words/cpp/class/aspose.words/documentbuilder/insertfield/), verwenden Sie das zurückgegebene [Field](https://reference.aspose.com/words/cpp/class/aspose.words.fields/field/) -Objekt, das eine praktische [Remove](https://reference.aspose.com/words/cpp/aspose.words.fields/field/remove/) -Methode zum einfachen Entfernen des Felds aus dem Dokument bietet.
