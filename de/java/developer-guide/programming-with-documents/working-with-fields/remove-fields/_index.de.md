Manchmal ist es notwendig, ein Feld aus dem Dokument zu entfernen. Dies kann auftreten, wenn es durch einen anderen Feldtyp ersetzt werden soll oder wenn das Feld im Dokument nicht mehr benötigt wird. Zum Beispiel ein `TOC` -Feld beim Speichern in HTML.

Um ein Feld zu entfernen, das mit [DocumentBuilder](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/) in ein Dokument eingefügt wurde.[InsertField](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/#insertField-int-boolean), verwenden Sie das zurückgegebene [Field](https://reference.aspose.com/words/java/com.aspose.words/field/) -Objekt, das eine praktische [Remove](https://reference.aspose.com/words/java/com.aspose.words/field/#remove) -Methode zum einfachen Entfernen des Felds aus dem Dokument bietet.
