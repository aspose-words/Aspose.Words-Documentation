Manchmal ist es notwendig, ein Feld aus dem Dokument zu entfernen. Dies kann auftreten, wenn es durch einen anderen Feldtyp ersetzt werden soll oder wenn das Feld nicht mehr im Dokument benötigt wird. Beispiel a `TOC` Feld beim Speichern in HTML.

Um ein in ein Dokument eingefügtes Feld zu entfernen [DocumentBuilder](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/).[InsertField](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/#insertField-int-boolean), zurückgegeben [Field](https://reference.aspose.com/words/java/com.aspose.words/field/) Objekt, das eine bequeme [Remove](https://reference.aspose.com/words/java/com.aspose.words/field/#remove) Verfahren zum einfachen Entfernen des Feldes aus dem Dokument.
