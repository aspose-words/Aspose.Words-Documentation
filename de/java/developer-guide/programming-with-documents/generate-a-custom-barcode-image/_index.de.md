{{% alert color="primary" %}}

Aspose.Words hat eine Schnittstelle zur Erstellung von benutzerdefinierten Barcodes, die es sehr einfach zu bedienen [Aspose.Words](https://products.aspose.com/words/java/) und [Aspose. Barcode](https://products.aspose.com/barcode/java/) zusammen, um Barcode-Bilder in Ausgabedokumenten zu machen. Sie können beispielsweise ein DOC-, OOXML- oder RTF-Dokument mit `DISPLAYBARCODE` Feld in Aspose.Words, bieten Sie Ihre Implementierung von benutzerdefinierten Barcode-Generator und speichern Sie auf festen Seitenformaten wie PDF, XPS usw. Ein typischer `DISPLAYBARCODE` Feld hat die folgende Syntax:

{ `DISPLAYBARCODE` "SomeData" QR \h 720 }

{{% /alert %}}
