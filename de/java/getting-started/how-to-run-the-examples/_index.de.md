## Download von: GitHub

Alle Beispiele Aspose.Words für Java werden gehostet [Github](https://github.com/aspose-words/Aspose.Words-for-Java). Sie können entweder das Repository mit Ihrem Lieblings-Github-Client klonen oder die ZIP-Datei herunterladen [Hier](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

Extrahieren Sie den Inhalt der ZIP-Datei in jeden Ordner auf Ihrem Computer. Alle Beispiele befinden sich in der **Examples** Ordner.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

Das Projekt nutzt Maven Bausystem. Jede moderne IDE kann das Projekt und seine Abhängigkeiten leicht öffnen oder importieren. Im Folgenden zeigen wir Ihnen, wie Sie beliebte IDEs verwenden, um die Beispiele zu erstellen und auszuführen.

## IntelliJ IDEA

Klicken Sie auf die **File** Menü und Auswahl **Open**. Suchen Sie in den Projektordner und wählen Sie die **Pom.xml** Datei.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

Es öffnet das Projekt und lädt die Abhängigkeiten automatisch herunter. Aus dem Projekt Tab, durchsuchen Sie die Beispiele in **src/main/java** Ordner. Um ein Beispiel auszuführen, klicken Sie mit der rechten Maustaste auf die Datei und wählen Sie "Run ..", das Beispiel wird ausgeführt und die Ausgabe wird im eingebauten Konsolenausgangsfenster angezeigt.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

Klicken Sie auf **File** Menü und Auswahl **Import**. Wählen **Maven** - Vorhandene Maven Projekte.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

Durchsuchen Sie den Ordner, den Sie kloniert oder heruntergeladen haben GitHub und wählen **Pom.xml** Datei.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

Es öffnet das Projekt und lädt die Abhängigkeiten automatisch herunter. Aus der Registerkarte Paket Explorer, durchsuchen Sie die Beispiele in **src/main/java** Ordner. Um ein Beispiel auszuführen, klicken Sie mit der rechten Maustaste auf die Datei und wählen **Run As** - Ja. **Java Application**, das Beispiel wird ausgeführt und die Ausgabe wird im eingebauten Konsolenausgangsfenster angezeigt.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

Klicken Sie auf die **File** Menü und Auswahl **Open Project**. Durchsuchen Sie den Ordner, den Sie kloniert oder heruntergeladen haben GitHub. Die Ikone **Examples** Ordner zeigt, dass es ein Maven Projekt. Wählen Sie Beispiele aus und öffnen Sie sie.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

Es öffnet das Projekt und lädt die Abhängigkeiten automatisch herunter. Aus der Registerkarte Projekte, durchsuchen Sie die Beispiele in **Quellpakete**. Um ein Beispiel auszuführen, klicken Sie mit der rechten Maustaste auf die Datei und wählen **Run File**, das Beispiel wird ausgeführt und die Ausgabe wird im eingebauten Konsolenausgangsfenster angezeigt.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)

## Beitrag

Wenn Sie ein Beispiel hinzufügen oder verbessern möchten, empfehlen wir Ihnen, zum Projekt beizutragen. Alle Beispiele und Präsentationsprojekte in diesem Projektarchiv sind Open Source und können in eigenen Anwendungen frei genutzt werden.

Um beizutragen, können Sie das Repository verlassen, den Quellcode bearbeiten und eine Pull-Anfrage erstellen. Wir werden die Änderungen überprüfen und in das Repository einfügen, wenn hilfreich gefunden.
