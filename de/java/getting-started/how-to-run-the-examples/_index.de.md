## Herunterladen von GitHub

Alle Beispiele von Aspose.Words für Java werden gehostet auf [Github](https://github.com/aspose-words/Aspose.Words-for-Java). Sie können das Repository entweder mit Ihrem bevorzugten Github -Client klonen oder die ZIP-Datei von herunterladen [hier](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

Extrahieren Sie den Inhalt der Datei ZIP in einen beliebigen Ordner auf Ihrem Computer. Alle Beispiele befinden sich im Ordner **Examples**.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

Das Projekt verwendet das Build-System Maven. Jeder moderne IDE kann das Projekt und seine Abhängigkeiten einfach öffnen oder importieren. Im Folgenden zeigen wir Ihnen, wie Sie popular IDEs zum Erstellen und Ausführen der Beispiele verwenden.

## IntelliJ IDEA

Klicken Sie auf das Menü **File** und wählen Sie **Open**. Navigieren Sie zum Projektordner und wählen Sie die **pom.xml**-Datei aus.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

Es öffnet das Projekt und lädt die Abhängigkeiten automatisch herunter. Durchsuchen Sie auf der Registerkarte Projekt den Ordner Beispiele in **src/main/java**. Um ein Beispiel auszuführen, klicken Sie einfach mit der rechten Maustaste auf die Datei und wählen Sie "Ausführen ..", wird das Beispiel ausgeführt und die Ausgabe im eingebauten Konsolenausgabefenster angezeigt.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

Klicken Sie auf das Menü **File** und wählen Sie **Import**. Wählen Sie **Maven** - Vorhandene Maven Projekte.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

Navigieren Sie zu dem Ordner, den Sie geklont oder von GitHub heruntergeladen haben, und wählen Sie **pom.xml**-Datei aus.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

Es öffnet das Projekt und lädt die Abhängigkeiten automatisch herunter. Durchsuchen Sie auf der Registerkarte Paket-Explorer die Beispiele im Ordner **src/main/java**. Um ein Beispiel auszuführen, klicken Sie einfach mit der rechten Maustaste auf die Datei und wählen Sie **Run As** - **Java Application**. Das Beispiel wird ausgeführt und die Ausgabe wird im eingebauten Konsolenausgabefenster angezeigt.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

Klicken Sie auf das Menü **File** und wählen Sie **Open Project**. Navigieren Sie zu dem Ordner, den Sie geklont oder von GitHub heruntergeladen haben. Das Symbol des Ordners **Examples** zeigt an, dass es sich um ein Maven -Projekt handelt. Wählen Sie Beispiele aus und öffnen Sie sie.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

Es öffnet das Projekt und lädt die Abhängigkeiten automatisch herunter. Durchsuchen Sie auf der Registerkarte Projekte die Beispiele in **source packages**. Um ein Beispiel auszuführen, klicken Sie einfach mit der rechten Maustaste auf die Datei und wählen Sie **Run File**. Das Beispiel wird ausgeführt und die Ausgabe wird im eingebauten Konsolenausgabefenster angezeigt.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)

## Beitragen

Wenn Sie ein Beispiel hinzufügen oder verbessern möchten, ermutigen wir Sie, zum Projekt beizutragen. Alle Beispiele und Vorzeigeprojekte in diesem Repository sind Open Source und können in Ihren eigenen Anwendungen frei verwendet werden.

Um einen Beitrag zu leisten, können Sie das Repository verzweigen, den Quellcode bearbeiten und einen Pull-Request erstellen. Wir werden die Änderungen überprüfen und sie in das Repository aufnehmen, wenn sie hilfreich sind.
