Sie können leicht wissen, ob ein Dokument erstellt wird, indem Sie Aspose.Words für Java. Sie finden auch die Versionsnummer Aspose.Words für Java die zur Erstellung eines bestimmten Ausgabedokuments verwendet wurde. Aspose.Words schreibt diese Informationen direkt in das generierte Dokument.

Die Versionsnummer ist im Format angegeben *YY.MM.N* oder *YY.M.N*, beispielsweise 23.12.0 oder 24.1.0.<br/>
Hier *YY* ist die letzten zwei Ziffern des Veröffentlichungsjahres, *M* oder *MM* ist der Freigabemonat (1–12), und *N* ist die kleinere Release-Nummer. Normalerweise, *N* ist "0".

{{% alert color="primary" %}}

Beachten Sie, dass Sie nicht anweisen können Aspose.Words diese Informationen aus Ausgabedokumenten zu ändern oder zu entfernen.

{{% /alert %}}
