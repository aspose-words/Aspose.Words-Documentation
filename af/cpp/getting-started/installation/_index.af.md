Maak seker dat jou masjien voldoen aan die [stelselvereistes](/words/cpp/system-requirements/) voordat jy begin.

Hierdie artikel verduidelik hoe om te installeer Aspose.Words vir C++ op jou rekenaar.

## Installeer of Werk Op Aspose.Words vir C++ Gebruik Bestuur NuGet Pakkette {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet is die maklikste manier om af te laai en te installeer [Aspose.Words vir C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. Om dit te doen, volg hierdie stappe:

1. Open *Microsoft Visual Studio* en *Manage NuGet Packages* vanaf die kieslys om pakketbestuurder oop te maak
2. Soek vir "aspose" of " aspose.woorde " om die gewenste Aspose APIte vind<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. Kies Die gewenste Aspose API en klik op"Installeer"

Die geselekteerde API sal afgelaai en verwys word in jou projek.

Jy kan ook opdateer Aspose.Words vir C++ na die gewenste weergawe met behulp van *Manage NuGet Packages*.

## Installeer of Werk Op Aspose.Words vir C++ gebruik Package Manager Console

U kan ook installeer of opdateer Aspose.Words vir C++ APIs gebruik die *Package Manager Console*. Om dit te doen, volg hierdie stappe:

1. Open *Microsoft Visual Studio* en *Package Manager Console* vanaf die kieslys om pakketbestuurderkonsole oop te maak
2. Tik die opdrag `Install-Package Aspose.Words.Cpp -Version 21.3.0` en druk Enter om die nuutste volledige weergawe in u toepassing te installeer
3. Sodra dit afgelaai is, sal jy bevestigingsboodskappe sien<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. Nou kan jy vind dat Aspose.Words suksesvol bygevoeg en verwys is in jou aansoek vir jou

In *Package Manager Console*, kan jy ook die `Update-Package Aspose.Words.Cpp` opdrag gebruik om te kyk vir updates om die Aspose.Words pakket en installeer hulle indien teenwoordig.

## Installeer Aspose.Words vir C++ Handmatig

Jy kan aflaai [Aspose.Words vir C++ Van NuGet Galery](https://www.nuget.org/packages/Aspose.Words.Cpp/) deur op die skakel "Handleiding aflaai" in die Inligtingafdeling te klik. Offline API Dokumentasie kan ook gevind word in die afgelaai ZIP pakket.

Om te gebruik Aspose.Words vir C++ volg hierdie stappe in u toepassings:

1. Laai die nuutste lêers af
2. Pak die gids oop in'n produksie-omgewing
3. Om te gebruik Aspose.Words vir C++, verwys Na Die Include en Lib dopgehou in jou projek
