### Bykomende Afhanklikheid van Linux

Aspose.Words vir C++ op Linux hang af van [fontconfig](https://www.freedesktop.org/wiki/Software/fontconfig/) binaries beide dinamiese biblioteek en instrument. Installeer dit voordat u dit gebruik:

1. Installeer fontconfig op Ubuntu of Debian</br>
`sudo apt install libfontconfig fontconfig`
2. Installeer fontconfig op Fedora of CentOs</br>
`sudo yum install fontconfig`
