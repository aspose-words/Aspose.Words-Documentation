Visual Basic vir Toepassings (VBA) vir Microsoft Word is'n eenvoudige maar kragtige programmeertaal wat gebruik kan word om die funksionaliteit uit te brei. Aspose.Words API bied drie klasse om toegang tot die VBA projek bronkode te kry:

- Die [VBAProject](https://reference.aspose.com/words/java/com.aspose.words/vbaproject/) klas bied toegang tot die VBA projek inligting
- Die [VBAModulesCollection](https://reference.aspose.com/words/java/com.aspose.words/vbamodulecollection/) klas gee terug die versameling van VBA projek modules
- Die [VbaModule](https://reference.aspose.com/words/java/com.aspose.words/vbamodule/) klas bied toegang tot die VBA projek module
