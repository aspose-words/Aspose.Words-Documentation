Jy kan maklik weet of'n dokument geproduseer word deur die gebruik van Aspose.Words vir Java. Jy kan ook die weergawe nommer van Aspose.Words vir Java wat gebruik is om'n spesifieke uitset dokument te skep. Aspose.Words skryf hierdie inligting direk in die gegenereerde dokument.

Die weergawe nommer word gespesifiseer in die formaat *YY.MM.N* of *YY.M.N*, byvoorbeeld, 23.12.0 of 24.1.0.<br />
Hier *YY* is die laaste twee syfers van die vrystelling jaar, *M* of *MM* is die vrystelling maand (1–12), en *N* is die klein vrystelling nommer. Gewoonlik is *N* "0".

{{% alert color="primary" %}}

Let daarop dat u nie Aspose.Words kan opdrag gee om hierdie inligting uit uitvoerdokumente te verander of te verwyder nie.

{{% /alert %}}
