Un "Bună, Lume!"codul este adesea primul exemplu simplu pentru a scrie uisng "Aspose.Words pentru Python via .NET", și poate fi, de asemenea, folosit ca un test de sănătate mintală pentru a se asigura că software-ul destinat compilării sau rulării codului sursă este instalat corect.

"Aspose.Words pentru Python via .NET"Biblioteca oferă dezvoltatorilor acces direct la crearea, modificarea, îmbinarea, conversia, compararea documentelor Word și Web. PDF, DOCX, DOC, RTF, ODT, EPUB, HTML și multe alte formate de fișiere sunt [suportat](/words/python-net/supported-document-formats/).

Below code fragment urmează acești pași:

1. Creați un nou gol [Document](https://reference.aspose.com/words/python-net/aspose.words/document/)
1. Inițializează o clasă [DocumentBuilder](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/)
1. Introduceți text în document începeți să utilizați metoda simplă [write](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/write/)
1. Deschideți un [Document](https://reference.aspose.com/words/python-net/aspose.words/document/) existent dintr-un fișier. Detectează automat formatul de fișier
1. [Append](https://reference.aspose.com/words/python-net/aspose.words/document/append_document/) document "A" la și a documentului "B"
1. [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/) Rezultatul ca PDF
