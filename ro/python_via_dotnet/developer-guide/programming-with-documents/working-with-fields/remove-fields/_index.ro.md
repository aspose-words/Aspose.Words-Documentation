Uneori este necesar să eliminați un câmp din document. Acest lucru se poate întâmpla atunci când trebuie înlocuit cu un alt tip de câmp sau când câmpul nu mai este necesar în document. De exemplu, un câmp `TOC` când salvați în HTML.

Pentru a elimina un câmp inserat într-un document folosind [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/), utilizați obiectul returnat [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/), care oferă metoda convenabilă [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) pentru a elimina cu ușurință câmpul din document.
