Uneori este necesar să eliminați un câmp dintr-un document. Acest lucru poate apărea atunci când trebuie înlocuit cu un alt tip de câmp sau când câmpul nu mai este necesar în document. De exemplu, câmpul `TOC` când salvați în HTML.

Pentru a elimina un câmp inserat într-un document folosind [DocumentBuilder](https://reference.aspose.com/words/cpp/aspose.words/documentbuilder/).[InsertField](https://reference.aspose.com/words/cpp/class/aspose.words/documentbuilder/insertfield/), utilizați obiectul returnat [Field](https://reference.aspose.com/words/cpp/class/aspose.words.fields/field/), care oferă o metodă convenabilă [Remove](https://reference.aspose.com/words/cpp/aspose.words.fields/field/remove/) pentru a elimina cu ușurință câmpul din document.
