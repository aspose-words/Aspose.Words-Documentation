Aspose.Words pentru C++ nu necesită instalarea unui produs terță parte, cum ar fi Microsoft Word. Aspose.Words în sine este un motor pentru crearea, modificarea, conversia și redarea documentelor în diferite formate, inclusiv Microsoft Word formate de documente.

## Mediul De Dezvoltare

Puteți utiliza Aspose.Words pentru C++ la dezvoltarea aplicațiilor pentru Windows, Linux sau macOS.

### Windows

Aspose.Words pentru C++ poate fi folosit pentru a dezvolta aplicații în orice mediu de dezvoltare care acceptă [Microsoft Visual Studio set de instrumente de platformă v143](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017), dar mediile enumerate în tabelul de mai jos sunt suportate în mod explicit:

		</tr>
  <tr>

			</tr>
</table>

### Dependență suplimentară de Linux

Aspose.Words pentru C++ pe Linux depinde de [fontconfig](https://www.freedesktop.org/wiki/Software/fontconfig/) binare atât bibliotecă dinamică și instrument. Vă rugăm să-l instalați înainte de a utiliza:

1. Instalarea fontconfig pe Ubuntu sau Debian</br>
`sudo apt install libfontconfig fontconfig`
2. Instalarea fontconfig pe Fedora sau CentOs</br>
`sudo yum install fontconfig`
