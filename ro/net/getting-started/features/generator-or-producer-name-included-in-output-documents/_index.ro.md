Poţi cu uşurinţă să-ţi dai seama dacă un document este produs utilizând Aspose.Words pentru .NET. De asemenea, puteți găsi numărul de versiune al Aspose.Words pentru .NET care a fost utilizat pentru a crea un anumit document de ieșire. Aspose.Words scrie direct această informație în documentul generat.

Numărul versiunii este specificat în formatul *YY.MM.N* sau *YY.M.N*, de exemplu, 23.12.0 sau 24.1.0.<br />
Aici *YY* sunt ultimele două cifre ale anului de lansare, *M* sau *MM* este luna lansării (1-12) și *N* este numărul minor al lansării. De obicei, *N* este "0".

{{% alert color="primary" %}}

Rețineți că îl puteți instrucționa pe Aspose.Words să elimine această informație din documentele de ieșire.

{{% /alert %}}
