## Important Note:

Starting from release **1.9.0** there are several versions of JAR library (necessary for better support of older versions of JasperServer and JasperReports):

|For<br>JasperServer|For<br>JasperReports|Use the following JAR library|
| :- | :- | :- |
|7.1.0 ... 7.5.0|6.5.0 ... 6.11.0|**aspose.words.jasperreports-22.2.0-7.1.0-7.5.0.jar**|
|6.4.2 ... 6.4.3|6.4.2 ... 6.4.21|**aspose.words.jasperreports-22.2.0-6.4.2-6.4.3.jar**|
|6.3.0 ... 6.4.0|6.3.0 ... 6.4.1|**aspose.words.jasperreports-22.2.0-6.3.0-6.4.0.jar**|
|6.0.0 ... 6.2.1|6.0.0 ... 6.2.2|**aspose.words.jasperreports-22.2.0-6.0.0-6.2.1.jar**|
Note that there is no more support of JasperReports 5.6.1 and lower versions.

Please copy **ONLY ONE** jar library according to the version of installed JasperReports.
