#### Завантаження файлу ліцензії

У Aspose.Words для Android Зареєструватися Java, Ліцензування може бути [Ліцензування та підписка](/words/uk/java/licensing/), або завантажено з потоку:

1,1 км Введіть номер мобільного, який Ви вказали при укладаннi договору з банком - **/mnt/sdcard/**й
1,1 км Створіть потік, який посилається файл.
1,1 км Передача потоку (збереження файлу ліцензії) в `SetLicense` метод.

**Java**

{{< highlight java >}}
String dataDir = Environment.getExternalStorageDirectory().getPath() + "/";

// Create a stream object containing the license file
FileInputStream fstream = new FileInputStream(dataDir + "Aspose.Email.Android.Java.lic");

// Instantiate the `License` class
License license = new License();
//Set the license through the stream object
license.setLicense(fstream);
{{< /highlight >}}

Застосування ліцензії з вбудованого ресурсу. Щоб отримати доступ до ліцензії як ресурс за назвою Android файл пакета:

1,1 км Введіть номер мобільного, який Ви вказали при укладаннi договору з нами **Резюме** папка.
   Введіть номер мобільного, який Ви вказали при укладаннi договору з банком - **Резюме** папка.
1,1 км Доступ/завантаження ліцензії з ресурсу з наступним зразкам коду.

**Java**

{{< highlight java >}}
License license = new License();

license.setLicense(inputStream);
{{< /highlight >}}

#### Застосувати Ліцензію

Aspose.Words дозволяє розробникам застосувати вимірювальний ключ. Це новий механізм ліцензування. Новий механізм ліцензування буде використовуватися разом з існуючим методом ліцензування. Ті клієнти, які хочуть бути враховані на основі використання API Можливість використання вимірювальної ліцензії.

**Java**

{{< highlight java >}}
Metered metered = new Metered();
try
{
	// Access the setMeteredKey property and pass public and private keys as parameters
    metered.setMeteredKey(publicKey, privateKey);
	Log.i("Metered License", "Ok. Metered License has been applied successfully.");
}
catch (Exception ex)
{
    ex.printStackTrace();
    Log.e("Metered License", "Setting metered key has failed: " + ex.getMessage());
}
{{< /highlight >}}

Ім'я файлу ліцензії не має бутиAspose.WordsйAndroidйJava.lic'. Ви можете перейменувати його на будь-який смак і використовувати цю назву при викликі `License.SetLicense`й

### Виняток - Не знайти логін

Коли ви завантажте ліцензію, яку ви придбали, названий файл **Aspose.WordsйAndroidйJavaЛогін** за замовчуванням. Файл ліцензії завантажується через веб-переглядач і деякі браузери розпізнають файл ліцензії як XML і додають розширення .xml до імені. Завантажений файл стає **Aspose.WordsйAndroidйJava.lic.XML**й

Коли Microsoft Windows налаштовано приховати розширення відомих типів файлів (надалі це за замовчуванням в більшості випадків Windows інсталяції), вказаний у вигляді ліцензії **Aspose.WordsйAndroidйJavaЛогін** в Windows Провідник. Як виглядає очікувана назва файлу. Зателефонуйте `License.SetLicense` і пропуск 'Aspose.WordsйAndroidйJava.lic', ви побачите і виняток тому, що немає такого файлу

Для вирішення проблеми перейменування файлу для видалення невидимого розширення .xml. Ми рекомендуємо вам вимкнути опцію "приховати розширення" в Microsoft Windowsй

## Використання декількох продуктів Aspose

Якщо ви використовуєте кілька продуктів Aspose в додатку, наприклад, Aspose.Words і `Aspose.Cells`, ось кілька корисних порад.

- Встановити ліцензію на кожен продукт Aspose окремо.
  Навіть якщо у вас є один файл ліцензії для всіх компонентів, наприклад "Аспоза". Всього.AndroidйJava.lic', вам все ще потрібно викликати `License.SetLicense` спосіб окремо для кожного продукту Aspose.
- Використовуйте повну кваліфіковану назву класу ліцензії.
  Що Продукція Aspose `License` клас у своєму просторі імен. Наприклад, Aspose.Words має com.aspose.words. Ліцензія та `Aspose.Cells` має com.aspose.cells. Ліцензійний клас. Використання повнорозмірного імені класу дозволяє уникнути будь-яких труднощів, про які наноситься ліцензія на який товар.
