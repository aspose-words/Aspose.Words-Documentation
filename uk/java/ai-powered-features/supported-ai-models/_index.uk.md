Aspose.Words тепер використовуються розширені моделі AI для покращення обробки та аналізу документів. Завдяки функціональності, заснованій на Aspose.Words AI, розробники можуть отримати доступ до функцій, які використовують AI для таких завдань, як узагальнення та аналіз документів, інтегруючи моделі генеративної мови від лідерів галузі.

Aspose.Words в даний час бібліотека підтримує інтеграцію з моделями OpenAI та Google generative language. Ці моделі надають потужні можливості для роботи з документами. Забезпечуючи безперебійну взаємодію з цими інструментами AI, Aspose.Words спрощує складні завдання і підвищує продуктивність для розробників.

Ви можете створити екземпляр моделі, використовуючи метод [Create](https://reference.aspose.com/words/java/com.aspose.words/aimodel/#create-int) класу [AiModel](https://reference.aspose.com/words/java/com.aspose.words/aimodel/). Повний список підтримуваних типів моделей наведено в перерахуванні [AiModelType](https://reference.aspose.com/words/java/com.aspose.words/aimodeltype/).

Наступний приклад коду показує, як створити новий екземпляр класу **AiModel**.

{{< highlight java>}}
public static AiModel create(int modelType)
{{< /highlight >}}
