{{% alert color="primary" %}}

Aspose.Words має інтерфейс для створення користувальницьких штрих-кодів, що робить його дуже простим у використанні [Aspose.Words](https://products.aspose.com/words/java/) і [Про нас Кошик](https://products.aspose.com/barcode/java/) разом із зображенням штрих-коду у вихідних документах. Наприклад, ви можете завантажити DOC, OOXML або RTF документ, що містить `DISPLAYBARCODE` Сфера в Aspose.Words, забезпечення вашої реалізації користувацького генератора штрих-кодів та збереження для основних форматів сторінок, таких як PDF, XPS і т.д. типовий `DISPLAYBARCODE` поле має наступний синтаксис:

{ `DISPLAYBARCODE` "SomeData" QR \h 720 }

{{% /alert %}}
