Іноді необхідно видалити поле з документа. Це може відбуватися, коли його потрібно замінити різними типами поля або коли поле більше не потрібно в документі. Наприклад, `TOC` поле при збереженні HTML.

Щоб видалити поле в документ за допомогою [DocumentBuilder](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/)й[InsertField](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/#insertField-int-boolean), використовувати повернуті [Field](https://reference.aspose.com/words/java/com.aspose.words/field/) об'єкт, який забезпечує зручне [Remove](https://reference.aspose.com/words/java/com.aspose.words/field/#remove) спосіб легко видалити поле з документа.
