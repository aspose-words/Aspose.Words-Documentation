Привітання Aspose.Words для Java!

Aspose.Words Бібліотека крос-платформного класу, що дозволяє вам виконати широкий спектр завдань обробки документів. Зателефонуйте редактору Програми.

Використання Aspose.Words, Ви можете створювати, змінювати, конвертувати, передавати та друкувати документи без релігування на сторонніх додатках, таких як Microsoft Word, або Office Automationй

Варто відзначити, що ви можете конвертувати документ з будь-якого формату завантаження на будь-який підтримуваний формат збереження, наприклад, Microsoft Word формати в PDF, HTML, або Markdown, HTML в DOCX і так далі. Детальніше про конвертацію документів в [Конвертувати документ](/words/uk/java/convert-a-document/) секція.

Ще один популярний варіант нашої бібліотеки *Reports і Mails*. Наші користувачі генерують висококомплексні індивідуально налаштовані фінансові, медичні та інші *Reports*й Таким чином, створений документ виглядає так, як ви створили звіт вручну, використовуючи звичайний текстовий редактор. Ще один варіант, *Mail*, надає можливість дуже швидко генерувати тисячі і мільйони простих, але ще і особистих шаблонних документів.

Увімкніть API Ми вирішили визнати і віддати перевагу поширенню Java досвід розробки платформ. Так, Aspose.Words для Java наступні принципи кодування, які широко приймаються Java розробники.

### Багатий набір функцій

Aspose.Words для Java Особливості можна розділити на чотири основні ділянки.

#### Переклад

Уміння легко і надійно конвертувати документи є однією з основних особливостей Aspose.Wordsй

Комплексний [Завантаження, збереження та перетворення](/words/uk/java/loading-saving-and-converting/) з [Формати документів](/words/uk/java/supported-document-formats/) дозволяє користувачам конвертувати документи з [один популярний формат](https://reference.aspose.com/words/java/com.aspose.words/loadformat/) до [another](https://reference.aspose.com/words/java/com.aspose.words/saveformat/), Наприклад, від DOCX до PDF, JPEG, або Markdownй

#### Document Object Model (DOM)

Програмний доступ через багату API до всіх елементів документа і форматування дозволяє створювати, змінювати, витягувати, копіювати, розбити, приєднатися і замінити вміст документа. Дізнатися більше [Aspose.Words Document Object Model (DOM)](/words/uk/java/aspose-words-document-object-model/) секція.

#### Рендеринг

Ми використовуємо термін [Рендеринг](/words/uk/java/rendering/) описати процес перетворення документа в формат файлу або середню, вагінальну або яка має поняття сторінок.

Ви можете конвертувати всі документи або конкретні сторінки в PDF або XPS генерувати резервні документи, друкувати їх або надавати сторінки. І все це з високою якістю – точно так Microsoft Word зробивши це.

#### Звітність

Створення документів або звітів з нуля або заповнення шаблонів з даними джерел даних або об’єктів бізнесу. Для отримання більш детальної інформації про звітування та отримання даних з баз даних див. інформацію [Mail Merge та звітність](/words/java/mail-merge-and-reporting/) і [ЛІНК Звітний двигун](/words/java/linq-reporting-engine/) секція.

### Майдан Незалежності

Aspose.Words для Java обкладинки всіх Java розробка середовища та платформи розгортання. Наші партнери API може бути використаний для розробки додатків для широкого спектру операційних систем, таких як Windows, Linux, Мак ОС, Android, і різні платформи.

Щоб покласти його просто, якщо ваша платформа може запустити стандарт Java, > він може запустити Aspose.Words для Javaй Якщо Ваш додаток може викликати стандарт Java API, > Ви не маєте жодних проблем Aspose.Words для Java API від вашого додатка.

На даний момент наша базова лінія - JDK1.7 і новачок. Але ми ще підтримуємо JDK1.6.

Ми маємо три різні релізи на основі тієї ж бази коду:

* Ім'я * Aspose.Words для Java
* Ім'я * Aspose.Words для Android Зареєструватися Java
* Ім'я * Aspose.Words для JasperReports

Під час Java і Android випуски скомпільовані для відповідної платформи, Aspose.Words для JasperReports – це просто товста обгортка навколо "звичайного" Aspose.Words для Javaй Просто для зручності. Ви також можете вручну інтегрувати JasperReports з Aspose Words - в конкретному випадку, який краще підходить для ваших потреб.

### Незалежність від інших додатків

Aspose.Words не потрібно сторонніх додатків, наприклад, Microsoft Офіс, який встановлюється на машині для роботи. Всі компоненти Aspose повністю незалежні. Це робить Aspose.Words відмінна альтернатива автоматизації з точки зору безпеки, стабільності, масштабності / швидкості, ціни та особливостей роботи з документами та суміжними завданнями.

### Продуктивність і масштабованість

Aspose.Words призначений для виконання на сервері або клієнту. Aspose.Words є одним Java баночка, яка може бути розгорнута будь-яким Java додаток просто копіюючи його. Ви також можете додати `HarfBuzz` плагін для розширеного Особливості шрифту, якщо це потрібно (подається в окремій банці). Стандарти Java ImageIO Розширення JAI також може бути додана до вашої платформи для підтримки додаткового ImageIO кодеки (не потрібно в сучасному Java середовища.

З Aspose.Words, Ви можете буквально генерувати тисячі документів за хвилину, що передбачає відкриття документа, внесення змін до вмісту та форматування даних, а також збереження його. Навіть найскладніші документи відкриті і зберігаються на P4 2.4Ghz 1Gb оперативної пам'яті машини в дробах другого.

Aspose.Words є багатопрочитаним сейфом, доки тільки одна нитка працює на документі одночасно. Це типовий сценарій, щоб мати одну нитку, яка працює на одному документі. Різні нитки можуть безпечно працювати на різних документах одночасно.
