Іноді деякі ряди в таблиці вимагають заголовка або великі блоки тексту, які займають повну ширину таблиці. Для належного оформлення таблиці користувач може об'єднати кілька клітин таблиці в одну. Aspose.Words підтримує об'єднані клітинки при роботі з усіма форматами введення, включаючи імпортування контенту HTML.

## Як захопити настільні клітини

У Aspose.Words, Об'єднані клітинки представлені такими властивостями [CellFormat](https://reference.aspose.com/words/python-net/aspose.words.tables/cellformat/) клас:

- до [HorizontalMerge](https://reference.aspose.com/words/python-net/aspose.words.tables/cellformat/horizontal_merge/) що описує, якщо клітина є частиною горизонтального зливу клітин
- до [VerticalMerge](https://reference.aspose.com/words/python-net/aspose.words.tables/cellformat/vertical_merge/) що описує, якщо клітина є частиною вертикальної зливи клітин

Значення цих властивостей визначають концентрацію поведінки клітин:

- до Перша клітинка в послідовності об'єднаних клітин буде мати [CellMerge.First](https://reference.aspose.com/words/python-net/aspose.words.tables/cellmerge/#first)
- до Будь-який згодом об'єднані клітинки будуть мати [CellMerge.Previous](https://reference.aspose.com/words/python-net/aspose.words.tables/cellmerge/#previous)
- Клітка, яка не зливається, буде мати [CellMerge.None](https://reference.aspose.com/words/python-net/aspose.words.tables/cellmerge/#none)

{{% alert color="primary" %}}

Іноді при завантаженні наявних елементів документа в таблиці з'являються об'єднання. Тим не менш, це може бути одна довга клітинка - іноді Microsoft Word експорт зливних клітин таким чином. При спробі працювати з окремими клітинами, але не здавалося б, бути певним шаблоном, коли це відбувається.

{{% /alert %}}
