Іноді необхідно видалити поле з документа. Це може відбуватися, коли потрібно замінити різним типом поля або коли поле більше не потрібно в документі. Наприклад, `TOC` поле при збереженні HTML.

Щоб видалити поле в документ за допомогою [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/), використовувати повернуті [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/) об'єкт, який забезпечує зручне [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) спосіб легко видалити поле з документа.
