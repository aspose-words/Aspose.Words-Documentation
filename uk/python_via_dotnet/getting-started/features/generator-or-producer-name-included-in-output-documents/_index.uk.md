Ви можете легко знати, якщо документ виробляється за допомогою Aspose.Words для Python via .NETй Ви також можете знайти номер версії Aspose.Words для Python via .NET щоб створити конкретний документ виведення. Aspose.Words безпосередньо напишіть цю інформацію в сформований документ.

Номер версії вказаний у форматі *YY.MM.N* або *YY.M.N*, Наприклад, 23.12.0 або 24.1.0.<br/>
Про нас *YY* - останні дві цифри релізу, *M* або *MM* місяць випуску (1–12), і *N* – неповний номер релізу. Як правило, *N* - "0".

{{% alert color="primary" %}}

Зверніть увагу, що ви можете інструктувати Aspose.Words видалити цю інформацію з вихідних документів.

{{% /alert %}}
