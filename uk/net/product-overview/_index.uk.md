Привітання Aspose.Words для .NET!

Aspose.Words Бібліотека кросплатформного класу, що дозволяє вашим додаткам виконувати широкий спектр завдань обробки документів.

Використання Aspose.Words, Ви можете конвертувати документ з будь-якого формату завантаження до будь-якого формату збереження, наприклад, формати Word у PDF, HTML або Markdown, PDF в DOCX і так далі. Детальніше про конвертацію документів в [Конвертувати документ](/words/uk/net/convert-a-document/) секція.

Варто відзначити, що з Aspose.Words Ви можете створювати, змінювати, конвертувати, передавати та друкувати документи без релігування на сторонніх додатках, таких як Microsoft Word, або Office Automationй

### Багатий набір функцій

Aspose.Words для .NET Особливості можна розділити на чотири основні ділянки.

#### Переклад

Уміння легко і надійно конвертувати документи є однією з основних особливостей Aspose.Wordsй

Комплексний [Завантаження, збереження та перетворення](/words/uk/net/loading-saving-and-converting/) з [Підтримувані формати документів](/words/uk/net/supported-document-formats/) дозволяє користувачам конвертувати документи з [один популярний формат](https://reference.aspose.com/words/net/aspose.words/loadformat/) до [another](https://reference.aspose.com/words/net/aspose.words/saveformat/), Наприклад, від DOCX до PDF, JPEG, або Markdown, або з PDF в різні формати Word.

#### Document Object Model (DOM)

Програмний доступ через багату API до всіх елементів документа і форматування дозволяє створювати, модифікувати, витягувати, копіювати, розбити, приєднатися і замінити вміст документа. Дізнатися більше [Aspose.Words Document Object Model (DOM)](/words/uk/net/aspose-words-document-object-model/) секція.

#### Рендеринг

Ми використовуємо термін [Рендеринг](/words/uk/net/rendering/) описати процес перетворення документа в формат файлу або середню, вагінальну або яка має поняття сторінок.

Ви можете конвертувати всі документи або конкретні сторінки в PDF або XPS генерувати резервні документи, друкувати їх або рендерувати сторінки, використовуючи .NET Графіка. І все це з високою якістю – точно так Microsoft Word зробивши це.

#### Звітність

Генерувати документи або звіти з нуля або шляхом заповнення шаблонів з даними джерел даних або об'єктів бізнесу. Для отримання більш детальної інформації про звітність та отримання даних з баз даних див. інформацію [Mail Merge та звітність](/words/net/mail-merge-and-reporting/) і [ЛІНК Звітний двигун](/words/net/linq-reporting-engine/) секція.

### Майдан Незалежності

Aspose.Words для .NET охоплює найбільш популярні середовища розробки та платформи розгортання. Своїм API може бути використаний для розробки додатків для широкого спектру операційних систем, таких як Windows, Linux, і Mac OS, і різні платформи.

Ви можете використовувати Aspose.Words для .NET для побудови будь-якого типу 32-bit або 64-bit .NET додаток, включаючи ASP.NET, WCF, WinForms та ін. Крім того, ви можете використовувати Aspose.Words для .NET Зареєструватися COM Interop з ASP, Perl, PHP, і Pythonй

Ви також можете використовувати Aspose.Words для .NET для побудови додатків з Mono, а також на Windows Azure, Microsoft SharePoint, Microsoft Silverlight, XamarinйAndroid, XamarinйiOS, і `Xamarin.Mac`й

### Незалежність від інших додатків

Aspose.Words не потрібно сторонніх додатків, наприклад, Microsoft Офіс, який встановлюється на машині для роботи. Всі компоненти Aspose повністю незалежні. Це робить Aspose.Words відмінна альтернатива автоматизації з точки зору безпеки, стабільності, масштабності / швидкості, ціни та можливостей для роботи з документами та суміжними завданнями.

### Продуктивність і масштабованість

Aspose.Words призначений для виконання на сервері або клієнту. Aspose.Words є одним .NET збірка, яка може бути розгорнута будь-яким .NET додаток просто копіюючи його. Вам не потрібно турбуватися про інші послуги або модулі.

З Aspose.Words, Ви можете буквально генерувати тисячі документів за хвилину, що передбачає відкриття документа, внесення змін до вмісту та форматування даних та збереження його. Навіть найскладніші документи відкриті і зберігаються на P4 2.4Ghz 1Gb оперативної пам'яті машини в дробах другого.

Aspose.Words є багатопрочитаним сейфом, доки тільки одна нитка працює на документі одночасно. Це типовий сценарій, щоб мати одну нитку, яка працює на одному документі. Різні нитки можуть безпечно працювати на різних документах одночасно.
