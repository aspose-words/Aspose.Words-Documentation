Перед початком роботи переконайтеся, що ваш комп'ютер відповідає вимогам до системи [системні вимоги ](/words/cpp/system-requirements/).

У цій статті пояснюється, як встановити Aspose.Words для C++ на ваш комп'ютер.

## Встановіть або оновіть Aspose.Words для C++ за допомогою керування NuGet пакетами {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet це найпростіший спосіб завантаження та встановлення [Aspose.Words для C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. Для цього виконайте наступні дії:

1. Відкрийте *Microsoft Visual Studio* та *Manage NuGet Packages* у меню, щоб відкрити менеджер пакетів
2. Шукайте "aspose" або " aspose.words", щоб знайти потрібний Aspose API<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. Виберіть потрібне налаштування API і натисніть "Встановити"

Вибраний файл API буде завантажений і на нього буде зроблено посилання у вашому проекті.

Ви також можете оновити Aspose.Words для C++ до потрібної версії, використовуючи *Manage NuGet Packages*.

## Встановіть або оновіть Aspose.Words для C++ за допомогою Package Manager Console

Ви також можете встановити або оновити Aspose.Words для C++ APIs, Використовуючи *Package Manager Console*. Для цього виконайте наступні дії:

1. Відкрийте *Microsoft Visual Studio* та *Package Manager Console* у меню, щоб відкрити консоль менеджера пакетів
2. Введіть команду `Install-Package Aspose.Words.Cpp -Version 21.3.0` і натисніть Enter, щоб встановити останню повну версію у вашому додатку
3. Після завантаження ви побачите підтверджуючі повідомлення<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. Тепер ви можете виявити, що Aspose.Words було успішно додано і на нього є посилання у вашому додатку

У *Package Manager Console* ви також можете скористатися командою `Update-Package Aspose.Words.Cpp`, щоб перевірити наявність оновлень пакета Aspose.Words та встановити їх, якщо вони є.

## Встановіть Aspose.Words для C++ вручну

Ви можете завантажити [Aspose.Words для C++ з NuGet галереї](https://www.nuget.org/packages/Aspose.Words.Cpp/), натиснувши на посилання " ручне завантаження "в розділі"Інформація". Документацію в автономному режимі API також можна знайти в завантаженому пакеті ZIP.

Щоб використовувати Aspose.Words замість C++ у своїх додатках, виконайте такі дії:

1. Завантажте останні версії файлів
2. Розпакуйте папку в робоче середовище
3. Щоб використовувати Aspose.Words замість C++, зверніться до папок Include та Lib у вашому проекті
