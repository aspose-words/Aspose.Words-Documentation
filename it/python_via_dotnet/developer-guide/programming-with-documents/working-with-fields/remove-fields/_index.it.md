A volte è necessario rimuovere un campo dal documento. Ciò può verificarsi quando deve essere sostituito con un tipo di campo diverso o quando il campo non è più necessario nel documento. Ad esempio un campo `TOC` durante il salvataggio in HTML.

Per rimuovere un campo inserito in un documento utilizzando [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/), utilizzare l'oggetto [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/) restituito, che fornisce il comodo metodo [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) per rimuovere facilmente il campo dal documento.
