Puoi facilmente sapere se un documento è prodotto utilizzando Aspose.Words per Python via .NET. È inoltre possibile trovare il numero di versione di Aspose.Words per Python via .NET utilizzato per creare un particolare documento di output. Aspose.Words scrive direttamente queste informazioni nel documento generato.

Il numero di versione è specificato nel formato *YY.MM.N* o *YY.M.N*, ad esempio 23.12.0 o 24.1.0.<br />
Qui *YY* rappresenta le ultime due cifre dell'anno di rilascio, *M* o *MM* è il mese di rilascio (1–12) e *N* è il numero di rilascio minore. Di solito, *N* è "0".

{{% alert color="primary" %}}

Tieni presente che puoi indicare a Aspose.Words di rimuovere queste informazioni dai documenti di output.

{{% /alert %}}
