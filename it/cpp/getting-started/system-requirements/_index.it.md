Aspose.Words per C++ non richiede l'installazione di alcun prodotto di terze parti come Microsoft Word. Aspose.Words stesso è un motore per la creazione, la modifica, la conversione e il rendering di documenti in vari formati, inclusi i formati di documenti Microsoft Word.

## Ambiente di sviluppo

È possibile utilizzare Aspose.Words per C++ quando si sviluppano applicazioni per Windows, Linux o macOS.

### Windows

Aspose.Words per C++ può essere utilizzato per sviluppare applicazioni in qualsiasi ambiente di sviluppo che supporta [Set di strumenti della piattaforma Microsoft Visual Studio v143](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017), ma gli ambienti elencati nella tabella seguente sono esplicitamente supportati:

<table>
	<tr>
			<td style="font-weight: bold; width:800px">Ambienti di sviluppo</td>
		</tr>
  <tr>
			<td><ul><li>Microsoft Visual Studio 2022</li></ul></td>
			</tr>
</table>

### Linux

Aspose.Words per C++ può essere utilizzato per sviluppare applicazioni nell'ambiente di sviluppo che supportano [Standard di linguaggio C++14](https://en.cppreference.com/w/cpp/14#Compiler_support), ma i compilatori seguenti sono esplicitamente supportati:

<table>
	<tr>
			<td style="font-weight: bold; width:800px">Compilatore</td>
		</tr>
  <tr>
			<td><ul><li>Clang 3.9.1 o successivo</li></ul></td>
			</tr>
</table>

### Dipendenza aggiuntiva da Linux

Aspose.Words per C++ su Linux dipende da [fontconfig](https://www.freedesktop.org/wiki/Software/fontconfig/) binari sia libreria dinamica e strumento. Si prega di installare prima di utilizzare:

1. Installazione di fontconfig su Ubuntu o Debian</br>
`sudo apt install libfontconfig fontconfig`
2. Installazione di fontconfig su Fedorao CentOs</br>
`sudo yum install fontconfig`

### Dipendenza aggiuntiva da macOS

Aspose.Words per C++ su macOS dipende da [fontconfig](https://www.freedesktop.org/wiki/Software/fontconfig/) binari sia libreria dinamica e strumento. Si prega di installare prima di utilizzare:

Installazione di fontconfig su macOS con [homebrew](https://brew.sh)</br>
`brew install fontconfig`
