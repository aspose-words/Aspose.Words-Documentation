Si può facilmente sapere se un documento viene prodotto utilizzando Aspose.Words per C++. È inoltre possibile trovare il numero di versione di Aspose.Words per C++ che è stato utilizzato per creare un particolare documento di output. Aspose.Words scrive direttamente queste informazioni nel documento generato.

Il numero di versione è specificato nel formato *YY.MM.N* o *YY.M.N*, ad esempio 23.12.0 o 24.1.0.<br />
Qui *YY* sono le ultime due cifre dell'anno di rilascio, *M* o *MM* è il mese di rilascio (1-12) e *N* è il numero di rilascio minore. Di solito, *N* è "0".

{{% alert color="primary" %}}

Si noti che non è possibile istruire Aspose.Words a modificare o rimuovere queste informazioni dai documenti di output.

{{% /alert %}}
