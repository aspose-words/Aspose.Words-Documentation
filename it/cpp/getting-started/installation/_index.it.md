Assicurati che la tua macchina soddisfi il [requisiti di sistema](/words/cpp/system-requirements/) prima di iniziare.

Questo articolo spiega come installare Aspose.Words per C++ sul tuo computer.

## Installare o aggiornare Aspose.Words per C++ utilizzo di Manage NuGet Packages {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet è il modo più semplice per scaricare e installare [Aspose.Words per C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. Per fare ciò, segui questi passaggi:

1. Aprire *Microsoft Visual Studio* e *Manage NuGet Packages* dal menu per aprire gestore pacchetti
2. Cerca " aspose "o" aspose.parole " per trovare l'API Aspose desiderata<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. Seleziona l'API Aspose desiderata e fai clic su"Installa"

L'API selezionata verrà scaricata e referenziata nel progetto.

È inoltre possibile aggiornare Aspose.Words per C++ alla versione desiderata usando *Manage NuGet Packages*.

## Installare o aggiornare Aspose.Words per C++ utilizzo di Package Manager Console

È inoltre possibile installare o aggiornare Aspose.Words per C++ APIs usando il *Package Manager Console*. Per fare ciò, segui questi passaggi:

1. Aprire *Microsoft Visual Studio* e *Package Manager Console* dal menu per aprire la console di gestione pacchetti
2. Digitare il comando `Install-Package Aspose.Words.Cpp -Version 21.3.0` e premere Invio per installare l'ultima versione completa nell'applicazione
3. Una volta scaricato, vedrai messaggi di conferma<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. Ora potresti scoprire che Aspose.Words è stato aggiunto e referenziato con successo nella tua applicazione per te

In *Package Manager Console*, è anche possibile utilizzare il comando `Update-Package Aspose.Words.Cpp` per verificare la presenza di aggiornamenti al pacchetto Aspose.Words e installarli se presenti.

## Installare Aspose.Words per C++ Manualmente

È possibile scaricare [Aspose.Words per C++ dalla Galleria NuGet ](https://www.nuget.org/packages/Aspose.Words.Cpp/) cliccando sul link "Download manuale" nella sezione Info. La documentazione API offline può anche essere trovata nel pacchetto ZIP scaricato.

Utilizzare Aspose.Words per C++ nelle applicazioni, attenersi alla seguente procedura:

1. Scarica i file più recenti
2. Decomprimere la cartella in un ambiente di produzione
3. Utilizzare Aspose.Words per C++, faccia riferimento alle cartelle di inclusione e Lib nel Suo progetto
