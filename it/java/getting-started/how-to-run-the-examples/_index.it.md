## Scarica da GitHub

Tutti gli esempi di Aspose.Words per Java sono ospitati su [Github](https://github.com/aspose-words/Aspose.Words-for-Java). È possibile clonare il repository utilizzando il client Github preferito o scaricare il file ZIP da [qui](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

Estrarre il contenuto del file ZIP in qualsiasi cartella del computer. Tutti gli esempi si trovano nella cartella **Examples**.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

Il progetto utilizza Maven sistema di compilazione. Qualsiasi IDE moderno può facilmente aprire o importare il progetto e le sue dipendenze. Di seguito ti mostriamo come utilizzare popular IDEs per creare ed eseguire gli esempi.

## IntelliJ IDEA

Fare clic sul menu **File** e scegliere **Open**. Passare alla cartella del progetto e selezionare il file **pom.xml**.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

Aprirà il progetto e scaricherà automaticamente le dipendenze. Dalla scheda Progetto, sfogliare gli esempi nella cartella **src/main/java**. Per eseguire un esempio, basta fare clic destro sul file e scegliere "Esegui ..", l'esempio verrà eseguito e l'output verrà mostrato nella finestra di output della console integrata.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

Fare clic sul menu **File** e scegliere **Import**. Seleziona **Maven** - Progetti esistenti Maven.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

Passare alla cartella clonata o scaricata da GitHub e selezionare **pom.xml** file.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

Aprirà il progetto e scaricherà automaticamente le dipendenze. Dalla scheda Esplora pacchetti, sfogliare gli esempi nella cartella **src/main/java**. Per eseguire un esempio, basta fare clic destro sul file e scegliere **Run As** - **Java Application**, l'esempio verrà eseguito e l'output verrà mostrato nella finestra di output della console integrata.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

Fare clic sul menu **File** e scegliere **Open Project**. Passare alla cartella da cui è stato clonato o scaricato GitHub. L'icona della cartella **Examples** mostrerà che è un progetto Maven. Selezionare Esempi e aprirlo.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

Aprirà il progetto e scaricherà automaticamente le dipendenze. Nella scheda Progetti, sfogliare gli esempi in **source packages**. Per eseguire un esempio, basta fare clic destro sul file e scegliere **Run File**, l'esempio verrà eseguito e l'output verrà mostrato nella finestra di output della console integrata.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)

## Contribuire

Per contribuire, è possibile fork il repository, modificare il codice sorgente e creare una richiesta pull. Esamineremo le modifiche e le includeremo nel repository se trovate utili.
