## Scarica da GitHub

Tutti gli esempi Aspose.Words per Java sono ospitati su [Github](https://github.com/aspose-words/Aspose.Words-for-Java). È possibile clone il repository utilizzando il client Github preferito o scaricare il file ZIP da [Qui](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

Estrarre il contenuto del file ZIP in qualsiasi cartella sul computer. Tutti gli esempi si trovano nel **Examples** cartella.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

Il progetto utilizza Maven costruire il sistema. Qualsiasi IDE moderno può facilmente aprire o importare il progetto e le sue dipendenze. Qui di seguito vi mostriamo come utilizzare IDE popolari per costruire ed eseguire gli esempi.

## IntelliJ IDEA

Fare clic su **File** menu e scegliere **Open**. Sfoglia la cartella del progetto e seleziona la **pom.xml** file.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

Apre il progetto e scarica automaticamente le dipendenze. Dalla scheda Progetto, sfogliare gli esempi in **src/main/java** cartella. Per eseguire un esempio, basta fare clic con il tasto destro del mouse sul file e scegliere "Run."., l'esempio verrà eseguito e l'output verrà mostrato nella finestra di uscita della console costruita.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

Clicca su **File** menu e scegliere **Import**. Seleziona **Maven** - Esiste Maven Progetti.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

Sfoglia la cartella da cui hai clonato o scaricato GitHub e selezionare **pom.xml** file.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

Apre il progetto e scarica automaticamente le dipendenze. Dalla scheda Package Explorer, sfoglia gli esempi in **src/main/java** cartella. Per eseguire un esempio, basta cliccare a destra sul file e scegliere **Run As** - No. **Java Application**, l'esempio verrà eseguito e l'output verrà mostrato nella finestra di uscita della console costruita.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

Fare clic su **File** menu e scegliere **Open Project**. Sfoglia la cartella da cui hai clonato o scaricato GitHub. L'icona di **Examples** cartella mostrerà che la sua Maven progetto. Selezionare Esempi e aprirlo.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

Apre il progetto e scarica automaticamente le dipendenze. Dalla scheda Progetti, sfogliare gli esempi in **pacchetti sorgente**. Per eseguire un esempio, basta cliccare a destra sul file e scegliere **Run File**, l'esempio verrà eseguito e l'output verrà mostrato nella finestra di uscita della console costruita.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)
