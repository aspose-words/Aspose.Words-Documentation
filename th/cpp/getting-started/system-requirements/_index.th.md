### การพึ่งพาเพิ่มเติมในLinux

Aspose.WordsสำหรับC++ บนLinuxขึ้นอยู่กับ [การกำหนดค่าแบบอักษร](https://www.freedesktop.org/wiki/Software/fontconfig/) ไบนารีทั้งไลบรารีแบบไดนามิกและเครื่องมือ กรุณาติดตั้งก่อนใช้:

1. การติดตั้งการกำหนดค่าแบบอักษรบนUbuntuหรือDebian</br>
`sudo apt install libfontconfig fontconfig`
2. การติดตั้งการกำหนดค่าแบบอักษรบนFedoraหรือCentOs</br>
`sudo yum install fontconfig`

### การพึ่งพาเพิ่มเติมเกี่ยวกับมาโก้

Aspose.WordsสำหรับC++ บนมาคอสขึ้นอยู่กับ [การกำหนดค่าแบบอักษร](https://www.freedesktop.org/wiki/Software/fontconfig/) ไบนารีทั้งไลบรารีแบบไดนามิกและเครื่องมือ กรุณาติดตั้งก่อนใช้:

การติดตั้งการกำหนดค่าแบบอักษรบนมาโก้ด้วย [homebrew](https://brew.sh)</br>
`brew install fontconfig`
