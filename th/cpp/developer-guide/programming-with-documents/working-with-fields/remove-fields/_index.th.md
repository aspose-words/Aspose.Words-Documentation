บางครั้งก็เป็นสิ่งจำเป็นที่จะเอาเขตข้อมูลจากเอกสาร นต้องถูกแทนที่ด้วยชนิดฟิลด์อื่นหรือเมื่อฟิลด์ไม่จำเป็นในเอกสารอีกต่อไป ตัวอย่างเช่นฟิลด์`TOC`เมื่อบันทึกเป็นHTML.

เมื่อต้องการเอาเขตข้อมูลที่แทรกลงในเอกสารโดยใช้[DocumentBuilder](https://reference.aspose.com/words/cpp/aspose.words/documentbuilder/)[InsertField](https://reference.aspose.com/words/cpp/class/aspose.words/documentbuilder/insertfield/)ใช้วัตถุที่ส่งคืน[Field](https://reference.aspose.com/words/cpp/class/aspose.words.fields/field/)ซึ่งมีวิธีการที่สะดวก[Remove](https://reference.aspose.com/words/cpp/aspose.words.fields/field/remove/)สำหรับการลบฟิลด์ออกจากเอกสาร.
