คุณสมบัติของออบเจกต์ส่วนใหญ่ที่ให้มา Aspose.Words API นั่น แสดง ถึง การ วัด บาง อย่าง เช่น ความ กว้าง หรือ ความ สูง, ระยะ ห่าง ต่าง ๆ, ค่า ใน จุด, โดย 1 นิ้ว เท่ากับ 72 จุด. บางครั้งมันไม่สะดวก และจุดต้องแปลงมาเป็นหน่วยอื่น

Aspose.Words ให้ [ConvertUtil](https://reference.aspose.com/words/java/com.aspose.words/convertutil/) คลาสที่มีฟังก์ชันผู้ช่วยในการแปลงหน่วยวัดต่าง ๆ มันสามารถเปลี่ยนนิ้ว พิกเซลและมิลลิเมตร เป็นจุดชี้เป็นนิ้วและพิกเซล และแปลงพิกเซลจากหนึ่งความละเอียดเป็นอีก การแปลงพิกเซลเป็นจุด และในทางกลับกัน สามารถดําเนินการได้ที่ 96 จุดต่อนิ้ว (dots ต่อนิ้ว) มติหรือความละเอียดที่ระบุไว้

เดอะ **ConvertUtil** คลาสนี้มีประโยชน์อย่างยิ่งเมื่อตั้งค่าคุณสมบัติหลาย ๆ หน้า ตัวอย่างเช่น นิ้วนั้นใช้วัดได้ทั่วไปมากกว่าจุด
