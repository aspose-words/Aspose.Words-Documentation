{{% alert color="primary" %}}

Aspose.Words มีส่วนติดต่อในการสร้างบาร์โค้ดที่กําหนดเอง ซึ่งทําให้มันใช้งานได้ง่าย [Aspose.Words](https://products.aspose.com/words/java/) ถึง [ข้อเสนอ แถบสี](https://products.aspose.com/barcode/java/) ร่วมกันทําการแสดงผลแฟ้ม barcode ในเอกสารผลลัพธ์ ตัวอย่างเช่น คุณสามารถโหลดเอกสาร DOC, OOXML หรือ RTF ที่มีเอกสาร `DISPLAYBARCODE` ช่องข้อมูลใน Aspose.Wordsให้โปรแกรมสร้างบาร์โค้ดและบันทึกในรูปแบบเพจที่คงที่ เช่น PDF XPS ฯลฯ แบบทั่วไป `DISPLAYBARCODE` สนามมีไวยากรณ์ต่อไปนี้:

{ `DISPLAYBARCODE` "SomeData" QR \h 720 }

{{% /alert %}}

ข้างล่างเป็นเครื่องสร้างรหัสตัวอย่างที่ยกระดับ `Aspose.BarCode` API. ตัวอย่างนี้แสดงวิธีแทรกภาพบาร์โค้ดที่ `DISPLAYBARCODE` ตําแหน่งของช่องข้อมูลในเอกสารคําที่ใช้ Aspose.Words ถึง `Aspose.BarCode` APIเอส.
