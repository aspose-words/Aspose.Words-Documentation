บาง ครั้ง จําเป็น ต้อง เอา ทุ่ง นา ออก จาก เอกสาร. อาจ เกิด เหตุ การณ์ เช่น นี้ เมื่อ จะ มี การ แทน ที่ สนาม ต่าง ๆ หรือ เมื่อ ไม่ จําเป็น ต้อง ใช้ สนาม ใน เอกสาร อีก ต่อ ไป. ตัวอย่างเช่น `TOC` ช่องข้อมูลเมื่อบันทึกเป็น HTML

เพื่อลบช่องข้อมูลที่แทรกเข้าไปในเอกสารโดยใช้ [DocumentBuilder](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/).[InsertField](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/#insertField-int-boolean), ใช้การตอบกลับ [Field](https://reference.aspose.com/words/java/com.aspose.words/field/) วัตถุที่ช่วยให้สะดวก [Remove](https://reference.aspose.com/words/java/com.aspose.words/field/#remove) วิธีการในการเอาช่องข้อมูลออกจากเอกสารได้อย่างง่ายดาย
