บางครั้งก็เป็นสิ่งจำเป็นที่จะเอาเขตข้อมูลจากเอกสาร นี้อาจเกิดขึ้นเมื่อมันจะถูกแทนที่ด้วยชนิดฟิลด์อื่นหรือเมื่อฟิลด์ไม่จำเป็นในเอกสาร ตัวอย่างเช่นฟิลด์`TOC`เมื่อบันทึกเป็นHTML.

เมื่อต้องการเอาเขตข้อมูลที่แทรกลงในเอกสารโดยใช้[DocumentBuilder](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/)[InsertField](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/#insertField-int-boolean)ใช้วัตถุที่ส่งคืน[Field](https://reference.aspose.com/words/java/com.aspose.words/field/)ซึ่งมีวิธีการที่สะดวก[Remove](https://reference.aspose.com/words/java/com.aspose.words/field/#remove)สำหรับการลบฟิลด์ออกจากเอกสาร.
