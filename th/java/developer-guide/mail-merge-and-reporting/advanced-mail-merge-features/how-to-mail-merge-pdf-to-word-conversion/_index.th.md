บทความนี้แสดงให้เห็นถึงตัวอย่างง่ายๆของการแสดงวิธีการMail Mergeบนเอกสารคำที่แปลงจากPDFแล้วบันทึกPDF โดยใช้Aspose.Wordsดำเนินการอย่างง่ายMail MergeกระบวนการบนPDFเป็นคำที่แปลงไฟล์ไม่ทำงานในบางกรณี ปัญหาเกิดขึ้นเนื่องจาก`Aspose.PDF`ไม่ได้เขียนจริงMERGEFIELDsระหว่างการแปลงPDFเป็นDOCX(เอกสารคำ) แต่มันสามารถทำได้โดยการแปลงข้อความแบบคงที่เหล่านั้นเป็นจริงMERGEFIELDsแล้วดำเนินการMail Mergeการดำเนินการ โปรดดูวิธีแก้ปัญหาต่อไปนี้.
