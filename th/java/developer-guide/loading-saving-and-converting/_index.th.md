ส่วนนี้จะให้ข้อมูลเกี่ยวกับคุณลักษณะเมื่อโหลดและบันทึกเอกสารเช่นเดียวกับข้อมูลเกี่ยวกับความเป็นไปได้เมื่อแปลงเอกสารจากรูปแบบหนึ่งไปยังอีก.

หากต้องการเรียนรู้เพิ่มเติมเกี่ยวกับคุณลักษณะAspose.Wordsเมื่อโหลดบันทึกหรือแปลงเอกสารให้เลือก สำหรับคำอธิบายโดยละเอียดของชั้นเรียนเมธอดและคุณสมบัติโปรดดูที่Aspose.WordsสำหรับJavaAPI Referenceส่วน.
