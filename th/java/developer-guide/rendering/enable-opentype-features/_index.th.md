แบบ Open Planguage เป็นรูปแบบแบบอักษรที่ถูกนํามาใช้เพื่อรองรับภาษาต่าง ๆ และระบบการเขียนระหว่างประเทศได้ดีขึ้น เมื่อเทียบกับภาษาโพสต์สคริปต์และรูปแบบภาษาจริง ลักษณะ การ วาง ผัง ของ การ แสดง แบบ เปิด เป็น ที่ รู้ จัก กัน ทั่ว ไป ว่า ลักษณะ แบบ เปิด รูป แบบ. Aspose.WordsสําลักHarfBuzz แพกเกจให้การสนับสนุนคุณสมบัติ OpenTranslate ใน Aspose.Words ใช้ `HarfBuzz` เครื่องยนต์หล่อหลอมข้อความ

Aspose.Words สามารถใช้วัตถุที่มีรูปทรงต่าง ๆ ภายนอกได้ ตัวแปลงข้อความแทนแบบอักษร และคํานวณการแปลงข้อมูลสําหรับข้อความ เอกสารมักอ้างถึงแบบอักษรหลายรูปแบบ ดังนั้น จึงจําเป็นสําหรับโรงงานแปลงข้อความ แพกเกจนี้บรรจุคู่มือของโรงงานแปลงเนื้อข้อความที่ถูกใช้โดย Aspose.Wordsสืบค้นเมื่อ 7 July 2015.

{{% alert color="primary" %}}

รูปแบบการแปลงข้อความ จะกระทําเมื่อมีการส่งออกไปยัง PDF หรือ XPS รูปแบบ

{{% /alert %}}

ในโปรแกรมทั่วไป ตัวอย่างของโรงงานแปลงข้อความ มีการใช้ร่วมกับเอกสารทั้งหมด เมื่อไรก็ตามที่ตัวแปลงข้อความถูกสร้างให้ใช้แฟ้มแบบอักษร การวิเคราะห์แฟ้มแบบอักษรเป็นปฏิบัติการที่แพง จึงขอแนะนําให้ใช้ canging Aspose.Words อุปกรณ์ ข้อความพื้นฐานShaper คลาสแคชซึ่งตัดส่วนตัวแปลงข้อความของโรงงานและตัวแปลงข้อความ ที่กลับมาโดยโรงงานที่ห่อไว้

ตัว อย่าง รหัส ต่อ ไป นี้ แสดง ให้ เห็น วิธี ที่ คุณ จะ เปิด การ สนับสนุน ลักษณะ แบบ เปิด เผย.
