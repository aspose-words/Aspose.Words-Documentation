## แพลตฟอร์มที่รองรับเมฆ

Aspose.Words สําหรับ Java รองรับแพลตฟอร์มเมฆที่สามารถทํางานได้ Java ใบสมัคร

## รับการรองรับ Java รุ่น

Aspose.Words สําหรับ Java สนับสนุน Java 7 และใหม่กว่า ถ้าคุณถูกบังคับให้ใช้ Java 6 เรายังคงปล่อยแยก JAR สําหรับ Java 6


## ความขึ้นกับตัวเลือก

อย่างที่เรากล่าวไปแล้ว Aspose.Words สําหรับ Java ต้องการ *only* คือ Java สภาพแวดล้อมการทํางาน และคุณไม่จําเป็นต้องติดตั้งไลบรารีเพิ่มเติมใด ๆ

อย่างไรก็ตาม บางครั้งมีสถานการณ์ที่คุณอาจจะ ต้องเพิ่มห้องสมุดสามส่วน:

- JogAmp JOGL ไลบรารีสําหรับฮาร์ดแวร์ที่เร่งการแสดงผล OpenGL ไลบรารีนี้รวมอยู่ในคําค้น POM เป็นไฟล์ที่เชื่อมโยงภายนอก คุณสามารถแก้ไข POM ไฟล์ที่จะเก็บรักษาโถ JO GL ที่ต้องการสําหรับแพลตฟอร์มของคุณเท่านั้น หรือคุณสามารถลบสิ่งเชื่อมโยงได้ ถ้าคุณไม่จําเป็นต้องใช้ฮาร์ดแวร์ที่เร่งการแสดงผล OpenGL
- HarfBuzz เครื่องยนต์แบบอักษร มี การ รวม คํา นี้ ไว้ แล้ว ใน ข้อ 9 และ ฉบับ ใหม่. ถ้าคุณใช้ JRE 8 หรือมากกว่านี้ คุณสามารถใช้การแทรกแซงแบบ Horfbuzz ใน POM แฟ้ม มิฉะนั้น คุณสามารถกําจัดสิ่งเชื่อมโยงได้
- Java Advanced Imaging (JAI) รองรับแฟ้มภาพประเภท TIFF ไม่รองรับแฟ้มภาพประเภท TIFF Java 8 และแก่กว่า สําหรับรองรับ TIFF แบบเก่า Java สิ่งแวดล้อม Aspose.Words สําหรับ Java ขึ้นอยู่กับ [Java Advanced Imaging (JAI) แพกเกจ](https://www.oracle.com/java/technologies/java-archive-downloads-java-client-downloads.html) จาก Oracle. เราจะบรรยายวิธีการติดตั้ง JI ด้านล่าง

### วิธี ติด ตั้ง JI Linux

ทําตามขั้นตอนเหล่านี้เพื่อติดตั้ง JI พื้นเมืองและ ImageIO บน Linux

1 ดาวน์โหลด **JI 1.3** เลือกสถาปัตยกรรมที่เหมาะสม
   1 [I586](https://download.java.net/media/jai/builds/release/1_1_3/jai-1_1_3-lib-linux-i586.tar.gz) สําหรับระบบ 32 บิต
   1 [amd64](https://download.java.net/media/jai/builds/release/1_1_3/jai-1_1_3-lib-linux-amd64.tar.gz) สําหรับหน่วยประมวลผล 64 บิต (แม้จะใช้หน่วยประมวลผลข้อมูล)
1 คลายแฟ้มออกจากไดเรกทอรีชั่วคราว
1 ย้าย JAR แฟ้มไปยัง JDK/JRE **lib/ext** โฟลเดอร์
1 ย้ายแฟ้ม SO ไปยัง JDK/JRE **lib/amd64** โฟลเดอร์<br/>

   ยกตัวอย่างเช่น บนเส้น 64 บิต Ubuntu ระบบ ขั้นที่ 1-4 จะเป็นแบบนี้<br/>
   {{< highlight csharp >}}
   $ cd /tmp
   $ wget https://download.java.net/media/jai/builds/release/1_1_3/jai-1_1_3-lib-linux-amd64.tar.gz && \
   gunzip -c jai-1_1_3-lib-linux-amd64.tar.gz | tar xf - && \
   mv /tmp/jai-1_1_3/lib/*.jar $JAVA_HOME/jre/lib/ext/ && \
   mv /tmp/jai-1_1_3/lib/*.so $JAVA_HOME/jre/lib/amd64/ && \
   rm /tmp/jai-1_1_3-lib-linux-amd64.tar.gz && \
   rm -r /tmp/jai-1_1_3
   {{< /highlight >}}
1 ดาวน์โหลด **ภาพ I/O 1**, เลือกสถาปัตยกรรมที่เหมาะสม
   1 [I586](https://download.java.net/media/jai-imageio/builds/release/1.1/jai_imageio-1_1-lib-linux-i586.tar.gz) สําหรับระบบ 32 บิต

## โฟลเดอร์สําหรับแฟ้มชั่วคราว

Aspose.Words สําหรับ Java ใช้ไดเรกทอรีชั่วคราวบนคอมพิวเตอร์ของคุณ ซึ่งระบุผ่านทาง `java.io.tmpdir` ทรัพย์สินระบบ Aspose.Words สําหรับ Java เขียนแฟ้มชั่วคราวแล้วลบแฟ้มเหล่านี้ทันทีที่การแปลเสร็จ

`java.io.tmpdir` เป็นมาตรฐาน Java คุณสมบัติของระบบที่ถูกใช้โดยนโยบายการจัดเก็บแบบดิสก์ มันกําหนดว่า JVM เขียนแฟ้มชั่วคราวที่ใด
