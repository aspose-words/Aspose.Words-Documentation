Aspose.Wordsให้ผู้ใช้ที่มีความหลากหลายของคุณสมบัติ. ผู้ใช้สามารถดำเนินการเป็นจำนวนมากของงานที่เกี่ยวข้องกับเอกสาร-จากเพียงแค่การแ.

เอกสารหรือคุณลักษณะการประมวลผลเอกสารอื่นๆในโครงการของคุณโซลูชันในทางปฏิบัติ แต่การใช้ส่วนประกอบของบุคคลที่สามมักจะมีความเสี่ยงบางอย่าง หนึ่งในความเสี่ยงที่ไม่ซ้ำกันในการประมวลผลเอกสารเป็นคำถามเกี่ยวกับวิธีการอย่างส.

แอสโพสมุ่งมั่นที่จะส่งมอบรูปแบบและมาตรฐานของเอกสารที่สมบูรณ์และถูกต้องที่สุด ทีมAspose.Wordsแสดงให้เห็นถึงความมุ่งมั่นของพวกเขาในการทำงานร่วมกันโดยการให้หมายเหตุกา.

ตารางด้านล่างสรุปAspose.WordsสำหรับคุณลักษณะJavaและให้ลิงก์ไปยังคำอธิบายคุณลักษณะโดยละเอียด.

{{% alert color="primary" %}}

ภาพหน้าจอส่วนใหญ่ในส่วนนี้แสดงให้เห็นถึงเอกสารที่สร้างโดยAspose.Wordsแล้วเปิดเพื่อดูในMicrosoft Wordและแอปพลิเคชันอื่นๆ.

{{% /alert %}}

## คุณสมบัติการจัดรูปแบบเอกสาร

| งคุณ |
| :- |
| เข้าถึงและปรับเปลี่ยนการจัดรูปแบบอักขระทั้งหมดรวมทั้งแบบอักษรสีเอฟเฟกต์เส้นขอบและแรเงา. |
| เข้าถึงและปรับเปลี่ยนการจัดรูปแบบย่อหน้าทั้งหมดรวมทั้งเยื้อง,ระยะห่าง,เส้นขอบ,และการแรเงา,แท็บหยุด. |
| เข้าถึงและปรับเปลี่ยนการจัดรูปแบบรายการสัญลักษณ์และหมายเลขทั้งหมดรวมทั้งหมายเลขและระดับ. |
| การเข้าถึงและปรับเปลี่ยนการจัดรูปแบบตารางรวมทั้งการจัดตำแหน่งการวางแนวเส้นขอบและการแรเงา. |
| ง่างามและใส่ใจในรายละเอียดต่างๆที่สร้างสรรค์อย่างประณีตด้วยทักษะความชำนาญและ. |
| เข้าถึงสร้างและปรับเปลี่ยนรูปแบบเอกสาร. |
| การเข้าถึงและปรับเปลี่ยนวัตถุวาดภาพรวมทั้งตำแหน่ง,ขนาด,เส้นและเติมคุณสมบัติไบต์ภาพ. |

{{% alert color="primary" %}}

| :- |

| รันMail Mergeสำหรับเอกสารที่มีแม้แต่ฟิลด์และสูตรที่ซับซ้อนและซับซ้อนที่สุด`IF`. |

| ใช้การควบคุมที่แม่นยำเช่นการจัดรูปแบบตัวเลขวิธีการผสานข้อมูลของคุณโดยใช้เหตุการณ์. |
| แทรกภาพระหว่างmail merge. |
