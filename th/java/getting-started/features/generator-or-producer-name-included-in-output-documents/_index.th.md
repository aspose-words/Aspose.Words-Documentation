คุณสามารถทราบได้อย่างง่ายดายว่าเอกสารถูกผลิตโดยใช้Aspose.WordsสำหรับJava นอกจากนี้คุณยังสามารถค้นหาหมายเลขรุ่นของAspose.WordsสำหรับJavaที่ใช้ในการสร้างเอกสารส่งออกเฉ Aspose.Wordsเขียนข้อมูลนี้ลงในเอกสารที่สร้างขึ้นโดยตรง.

หมายเลขรุ่นระบุไว้ในรูปแบบ*YY.MM.N*หรือ*YY.M.N*ตัวอย่างเช่น23.12.0หรือ24.1.0<br />
ที่นี่*YY*เป็นตัวเลขสองหลักสุดท้ายของปีปล่อย*M*หรือ*MM*เป็นเดือนปล่อย(1-12)และ*N*เป็นหมายเลขปล่อย โดยปกติ*N*คือ"0".

{{% alert color="primary" %}}

โปรดทราบว่าคุณไม่สามารถสั่งให้Aspose.Wordsเปลี่ยนแปลงหรือลบข้อมูลนี้ออกจากเอกสารที่ส่งออก.

{{% /alert %}}
