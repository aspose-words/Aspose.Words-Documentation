#### กําลังโหลดแฟ้มสัญญาอนุญาตสิทธิ์

ใน Aspose.Words สําหรับ Android ผ่าน Java, ใบอนุญาตสามารถ [การ บอก รับ และ การ บอก รับ](/words/th/java/licensing/), ขอสาบานต่อลมที่พัดออกมาติดต่อกัน

1 ใส่แฟ้มใบอนุญาตที่ตําแหน่งใด ๆ บน **/ int/sdcard /**.
1 สร้างพาธที่ต้องการอ้างอิง
1 ส่งสายข้อมูล (การบรรจุแฟ้มใบอนุญาต) ไปยัง `SetLicense` วิธี

**Java**

{{< highlight java >}}
String dataDir = Environment.getExternalStorageDirectory().getPath() + "/";

// Create a stream object containing the license file
FileInputStream fstream = new FileInputStream(dataDir + "Aspose.Email.Android.Java.lic");

// Instantiate the `License` class
License license = new License();
//Set the license through the stream object
license.setLicense(fstream);
{{< /highlight >}}

กําลังปรับใช้ใบรับรองจากทรัพยากรที่ฝังตัวอยู่ เพื่อเข้าใช้งานสัญญาอนุญาตสิทธิ์โดยใช้ชื่อ Android แฟ้มแพกเกจ:

1 เพิ่มแฟ้มใบรับรองเป็นทรัพยากรของโปรแกรมของคุณ **ทําซ้ํา/ สแกน** โฟลเดอร์
   ควรจะเห็นแฟ้มใบอนุญาตใน **ทําซ้ํา/ สแกน** โฟลเดอร์
1 การเข้าใช้งาน/ การโหลดใบรับรองจากทรัพยากร โดยมีตัวอย่างโค้ดต่อไปนี้

**Java**

{{< highlight java >}}
License license = new License();
InputStream inputStream = getResources().openRawResource(R.raw.license);
license.setLicense(inputStream);
{{< /highlight >}}

#### ปรับใช้สัญญาอนุญาตสิทธิ์

Aspose.Words อนุญาตให้นักพัฒนาใช้ปุ่มพิมพ์มิเตอร์ได้ มันเป็นกลไกการเริมแบบใหม่ กลไกการเหาใหม่จะถูกนําไปใช้ร่วมกับวิธีการเหาที่มีอยู่แล้ว ลูกค้าเหล่านั้นที่ต้องการที่จะเรียกเก็บเงินตามการใช้งานของ API ลักษณะต่าง ๆ สามารถใช้ตัวเหาวัดได้

**Java**

{{< highlight java >}}
Metered metered = new Metered();
try
{
	// Access the setMeteredKey property and pass public and private keys as parameters
    metered.setMeteredKey(publicKey, privateKey);
	Log.i("Metered License", "Ok. Metered License has been applied successfully.");
}
catch (Exception ex)
{
    ex.printStackTrace();
    Log.e("Metered License", "Setting metered key has failed: " + ex.getMessage());
}
{{< /highlight >}}

### เปลี่ยนชื่อแฟ้มสัญญาอนุญาตสิทธิ์

ชื่อแฟ้มใบอนุญาตไม่จําเป็นต้องเป็น 'Aspose.Words.Android.Javaลิก คุณสามารถเปลี่ยนชื่อมันเป็นอะไรก็ได้ที่คุณชอบ และใช้ชื่อนั้นเมื่อเรียก `License.SetLicense`.

### การยกเว้น - ไม่พบชื่อแฟ้มใบรับรอง

เมื่อคุณดาวน์โหลดใบอนุญาตที่คุณซื้อ ไฟล์ใบอนุญาตชื่อ **Aspose.Words.Android.Java.lic** โดยปริยายแล้ว แฟ้มใบรับรองถูกดาวน์โหลดผ่านเบราว์เซอร์ของคุณ และเบราว์เซอร์บางตัวรู้จักแฟ้มใบอนุญาตเป็น XML และเพิ่มนามสกุล .xml ไปยังชื่อนี้ แฟ้มที่ถูกดาวน์โหลดมาจะกลายเป็น **Aspose.Words.Android.Java.lic.XML**.

เมื่อ Microsoft Windows ได้ตั้งค่าให้ซ่อนนามสกุลของแฟ้มที่รู้จัก (ค่าปริยายแล้ว นี่เป็นค่าปริยายมากที่สุด Windows การติดตั้ง), แฟ้มสัญญาอนุญาตจะถูกแสดงเป็น **Aspose.Words.Android.Java.lic** ใน Windows นักสํารวจ มันดูเหมือนชื่อแฟ้มที่คาดหวัง ถ้าคุณโทรมา `License.SetLicense` และผ่านAspose.Words.Android.Java.lic'คุณจะเห็นและยกเว้นเพราะไม่มีไฟล์ดังกล่าว

เพื่อแก้ปัญหา ให้เปลี่ยนชื่อแฟ้มเพื่อเอาส่วนขยาย .xml ที่มองไม่เห็นออกไป เราแนะนําให้คุณปิดตัวเลือก "การขยายแบบ Hide" ใน Microsoft Windows.

## การ ใช้ ข้อ เสนอ หลาย อย่าง

ถ้าคุณใช้ผลิตภัณฑ์ที่เป็น Asposite ในโปรแกรม Aspose.Words ถึง `Aspose.Cells`, นี่คือเคล็ดลับที่มีประโยชน์

- กําหนดใบอนุญาตสําหรับผลิตภัณฑ์ Asposite แต่ละแยก
  ต่อให้คุณมีเอกสารใบอนุญาตเดียว สําหรับส่วนประกอบทั้งหมด ตัวอย่างเช่น 'คุณสมบัติ' ยอดเลยAndroid.Javaลิก คุณยังต้องโทรหา `License.SetLicense` วิธีการแยกแต่ละผลิตภัณฑ์ Aspost
- ใช้ชื่อคลาสสัญญาบัตร
  แต่ละ ผลผลิตเชิงบวกมี `License` คลาสในชื่อที่ว่าง ยกตัวอย่างเช่น Aspose.Words มี com.asposite. words. สัญญาอนุญาตสิทธิ์และ `Aspose.Cells` มี com.asposect.เซลล์. ชั้นเรียนใบอนุญาติ การ ใช้ ชื่อ ชั้น เรียน ที่ มี คุณสมบัติ ครบ ถ้วน ทํา ให้ คุณ สามารถ หลีก เลี่ยง ความ สับสน ใด ๆ เกี่ยว กับ การ ใช้ ใบ อนุญาต สําหรับ ผลิตภัณฑ์ ใด.
