## ดาวน์โหลดจากGitHub

ตัวอย่างทั้งหมดของAspose.WordsสำหรับJavaถูกโฮสต์บน [Github](https://github.com/aspose-words/Aspose.Words-for-Java). คุณสามารถโคลนพื้นที่เก็บข้อมูลโดยใช้ไคลเอ็นต์Githubที่คุณชื่นชอบหรือดาวน์โหลดไฟล์ZIPจาก [ที่นี่](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

แยกเนื้อหาของZIPไฟล์ไปยังโฟลเดอร์ใดๆในคอมพิวเตอร์ของคุณ ตัวอย่างทั้งหมดจะอยู่ในโฟลเดอร์**Examples**.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

โครงการใช้Mavenสร้างระบบ ใดๆที่ทันสมัยIDEสามารถเปิดหรือนำเข้าโครงการและการอ้างอิงของมัน ด้านล่างนี้เราจะแสดงวิธีใช้ยอดนิยมIDEsเพื่อสร้างและเรียกใช้ตัวอย่าง.

## IntelliJ IDEA

คลิกเมนู**File**และเลือก**Open** เรียกดูโฟลเดอร์โครงการและเลือกไฟล์**pom.xml**.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

มันจะเปิดโครงการและดาวน์โหลดอ้างอิงโดยอัตโนมัติ จากแท็บโปรเจ็กต์เรียกดูตัวอย่างใน**src/main/java**โฟลเดอร์ ในการเรียกใช้ตัวอย่างเพียงแค่คลิกขวาที่ไฟล์และเลือก"เรียกใช้."ตัวอย่างจะถูกดำเนินการและการส่งออกจะแสดงในหน้าต่างออกคอนโซลในตัว.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

คลิกที่เมนู**File**และเลือก**Import** เลือก**Maven**-ที่มีอยู่Mavenโครงการ.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

เรียกดูโฟลเดอร์ที่คุณโคลนหรือดาวน์โหลดจากGitHubแล้วเลือก**pom.xml**ไฟล์.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

มันจะเปิดโครงการและดาวน์โหลดอ้างอิงโดยอัตโนมัติ จากแท็บสำรวจแพคเกจเรียกดูตัวอย่างใน**src/main/java**โฟลเดอร์ ในการเรียกใช้ตัวอย่างเพียงแค่คลิกขวาบนไฟล์และเลือก**Run As**-**Java Application**ตัวอย่างจะถูกดำเนินการและผลลัพธ์จะแสดงในหน้าต่างออกคอนโซลในตัว.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

คลิกเมนู**File**และเลือก**Open Project** เรียกดูโฟลเดอร์ที่คุณโคลนหรือดาวน์โหลดจากGitHub ไอคอนของ**Examples**โฟลเดอร์จะแสดงว่าโครงการMaven เลือกตัวอย่างและเปิด.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

มันจะเปิดโครงการและดาวน์โหลดอ้างอิงโดยอัตโนมัติ จากแท็บโปรเจ็กต์เรียกดูตัวอย่างใน**source packages** ในการเรียกใช้ตัวอย่างเพียงแค่คลิกขวาบนไฟล์และเลือก**Run File**ตัวอย่างจะดำเนินการและผลลัพธ์จะแสดงในหน้าต่างออกคอนโซลในตัว.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)

## มีส่วนร่วม

หากคุณต้องการที่จะเพิ่มหรือปรับปรุงตัวอย่างเราขอแนะนำให้คุณมีส่วนร่วมในโครงการ ตัวอย่างทั้งหมดและโครงการแสดงในพื้นที่เก็บข้อมูลนี้เป็นโอเพนซอร์สและสามารถนำมาใ.

ในการมีส่วนร่วมคุณสามารถแยกที่เก็บแก้ไขรหัสแหล่งที่มาและสร้างการร้องขอการดึง เราจะตรวจสอบการเปลี่ยนแปลงและรวมไว้ในที่เก็บหากพบว่าเป็นประโยชน์.
