## ดาวน์โหลดจาก GitHub

ตัวอย่างทั้งหมดของ Aspose.Words สําหรับ Java จะเป็นเจ้าภาพ [กิทูบ](https://github.com/aspose-words/Aspose.Words-for-Java). คุณสามารถโคลนคลังแพกเกจโดยใช้ไคลเอนต์ Github ของคุณ หรือดาวน์โหลดแฟ้ม ZIP จาก [ที่นี่](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

คลายเนื้อหาออกจากแฟ้ม ZIP ไปยังโฟลเดอร์ใด ๆ บนคอมพิวเตอร์ของคุณ ตัวอย่างทั้งหมดอยู่ใน **Examples** โฟลเดอร์

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

โครงการนี้ใช้ Maven ระบบสร้าง ไอดีอีสมัยใหม่ สามารถเปิดหรือนําเข้าโครงการและความขึ้นต่อกันได้ ข้างล่างเราแสดงให้คุณเห็นวิธีการใช้ IDEs ยอดนิยมในการสร้างและดําเนินการตัวอย่าง

## IntelliJ IDEA

คลิกบน **File** เมนูและเลือก **Open**. เรียกดูโฟลเดอร์โครงการและเลือก **pom.xml** แฟ้ม

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

มันจะเปิดโครงการและดาวน์โหลดการขึ้นต่อกันโดยอัตโนมัติ จากแท็บโครงการ ให้เรียกดูตัวอย่าง **irc/ หลัก/ java** โฟลเดอร์ เพื่อประมวลผลตัวอย่าง คลิกเมาส์ปุ่มขวาบนแฟ้ม และเลือก "ทํางาน" ตัวอย่างจะถูกประมวลผล และการแสดงผลจะถูกแสดงในหน้าต่างการแสดงผลของคอนโซล

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

คลิกบน **File** เมนูและเลือก **Import**. เลือก **Maven** - มีอยู่ Maven โครงการ

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

เรียกดูโฟลเดอร์ที่คุณคัดลอกหรือดาวน์โหลดจาก GitHub เลือก **pom.xml** แฟ้ม

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

มันจะเปิดโครงการและดาวน์โหลดการขึ้นต่อกันโดยอัตโนมัติ จากแท็บของแพกเกจสํารวจ, การเรียกดูตัวอย่างใน **irc/ หลัก/ java** โฟลเดอร์ เพื่อเรียกตัวอย่าง ให้คลิกขวาบนแฟ้มและเลือก **Run As** - **Java Application**, จะมีการประมวลผลตัวอย่าง และการแสดงผลจะถูกแสดงในหน้าต่างที่แสดงผลมาจากคอนโซล

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

คลิกบน **File** เมนูและเลือก **Open Project**. เรียกดูโฟลเดอร์ที่คุณคัดลอกหรือดาวน์โหลดจาก GitHub. ไอคอนของ **Examples** โฟลเดอร์จะแสดงให้เห็นว่า Maven โครงการ เลือกตัวอย่างและเปิดมัน

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

มันจะเปิดโครงการและดาวน์โหลดการขึ้นต่อกันโดยอัตโนมัติ จากแท็บโครงการ ให้เรียกดูตัวอย่าง **แพกเกจต้นทาง**. เพื่อเรียกตัวอย่าง ให้คลิกขวาบนแฟ้มและเลือก **Run File**, จะมีการประมวลผลตัวอย่าง และการแสดงผลจะถูกแสดงในหน้าต่างที่แสดงผลมาจากคอนโซล

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)

## ผู้มีส่วนร่วม

ถ้า คุณ อยาก เพิ่ม หรือ ปรับ ปรุง ตัว อย่าง เรา สนับสนุน คุณ ให้ มี ส่วน ร่วม ใน โครงการ นี้. ตัวอย่างและแสดงโครงการทั้งหมดในคลังแพกเกจนี้ เป็นแหล่งเปิดและสามารถใช้ได้อย่างอิสระกับโปรแกรมของคุณเอง

เพื่อให้ความช่วยเหลือ คุณสามารถแยกคลังข้อมูล แก้ไขรหัสต้นทาง และสร้างการร้องขอดึง เรา จะ ทบทวน การ เปลี่ยน แปลง และ รวม เอา การ เปลี่ยน แปลง นี้ เข้า ไว้ ใน ห้อง เก็บ ของ หาก พบ ว่า เป็น ประโยชน์.
