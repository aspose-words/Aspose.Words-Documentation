คุณสมบัติวัตถุส่วนใหญ่ที่ให้ไว้ใน Aspose.Words API ที่แสดงการวัดบางอย่าง เช่น ความกว้างหรือความสูง ระยะขอบ และระยะทางต่างๆ ยอมรับค่าเป็นจุด โดยที่ 1 นิ้วเท่ากับ 72 จุด บางครั้งไม่สะดวกและต้องแปลงคะแนนเป็นหน่วยอื่น

Aspose.Words จัดเตรียมคลาส [ConvertUtil](https://reference.aspose.com/words/python-net/aspose.words/convertutil/) ที่มีฟังก์ชันตัวช่วยในการแปลงระหว่างหน่วยการวัดต่างๆ ช่วยให้สามารถแปลงนิ้ว พิกเซล และมิลลิเมตรเป็นจุด ชี้เป็นนิ้วและพิกเซล และแปลงพิกเซลจากความละเอียดหนึ่งไปอีกความละเอียดหนึ่ง การแปลงพิกเซลเป็นจุดและในทางกลับกันสามารถทำได้ที่ความละเอียด 96 dpi (จุดต่อนิ้ว) หรือความละเอียด dpi ที่ระบุ

คลาส [ConvertUtil](https://reference.aspose.com/words/python-net/aspose.words/convertutil/) มีประโยชน์อย่างยิ่งเมื่อตั้งค่าคุณสมบัติของหน้าต่างๆ เนื่องจาก ตัวอย่างเช่น นิ้วเป็นหน่วยการวัดที่ใช้กันทั่วไปมากกว่าจุด
