Aspose.Words สำหรับ .NET เป็นไลบรารีคลาสที่ช่วยให้แอปพลิเคชันของคุณสามารถดำเนินการประมวลผลเอกสารได้อย่างหลากหลาย ด้วย Aspose.Words คุณสามารถสร้าง แก้ไข แปลง แสดงผล และพิมพ์เอกสารโดยไม่ต้องใช้ Microsoft Word กล่าวคือ ไม่จำเป็นต้องใช้ Microsoft Word เพื่อใช้ Aspose.Words

Aspose.Words สำหรับ .NET เขียนขึ้นอย่างสมบูรณ์ใน C# เป็นไปตามข้อกำหนด CLS และมีเฉพาะโค้ดที่ได้รับการจัดการที่ปลอดภัยเท่านั้น

API Reference ล่าสุดสามารถพบได้ที่ [Aspose.Words สำหรับ .NET API Reference](https://reference.aspose.com/words/net/)
