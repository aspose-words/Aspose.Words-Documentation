Asegúrese de que su máquina cumpla con los [requisitos del sistema](/words/cpp/system-requirements/) antes de que empieces.

Este artículo explica cómo instalar Aspose.Words para C++ en tu computadora.

## Instalar o Actualizar Aspose.Words para C++ uso de Administrar NuGet paquetes {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet es la forma más fácil de descargar e instalar [Aspose.Words para C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. Para ello, sigue estos pasos:

1. Abra *Microsoft Visual Studio* y *Manage NuGet Packages* en el menú para abrir el administrador de paquetes
2. Busque "aspose" o " aspose.palabras " para encontrar la API Aspose deseada<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. Seleccione la API Aspose deseada y haga clic en "Instalar"

La API seleccionada se descargará y se hará referencia a ella en su proyecto.

También puedes actualizar Aspose.Words para C++ a la versión deseada usando *Manage NuGet Packages*.

## Instalar o Actualizar Aspose.Words para C++ usando Package Manager Console

También puede instalar o actualizar Aspose.Words para C++ APIs usando el *Package Manager Console*. Para ello, sigue estos pasos:

1. Abra *Microsoft Visual Studio* y *Package Manager Console* en el menú para abrir la consola del administrador de paquetes
2. Escriba el comando `Install-Package Aspose.Words.Cpp -Version 21.3.0` y presione Entrar para instalar la última versión completa en su aplicación
3. Una vez descargado, verá mensajes de confirmación. <br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. Ahora puede descubrir que Aspose.Words se ha agregado y referenciado correctamente en su solicitud por usted

En *Package Manager Console*, también puede usar el comando `Update-Package Aspose.Words.Cpp` para buscar actualizaciones del paquete Aspose.Words e instalarlas si están presentes.

## Instalar Aspose.Words para C++ Manualmente

Puedes descargar [Aspose.Words para C++ de la galería NuGet ](https://www.nuget.org/packages/Aspose.Words.Cpp/) haciendo clic en el enlace" Descarga manual " en la sección de Información. La documentación de la API sin conexión también se puede encontrar en el paquete ZIP descargado.

Para usar Aspose.Words para C++ en sus aplicaciones, siga estos pasos:

1. Descargue los archivos más recientes
2. Descomprima la carpeta en un entorno de producción
3. Para usar Aspose.Words para C++, consulte las carpetas Include y Lib en su proyecto
