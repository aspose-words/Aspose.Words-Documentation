Puede saber fácilmente si un documento se produce utilizando Aspose.Words para C++. También puede encontrar el número de versión de Aspose.Words para C++ eso se usó para crear un documento de salida en particular. Aspose.Words escribe directamente esta información en el documento generado.

El número de versión se especifica en el formato *YY.MM.N* o *YY.M.N*, por ejemplo, 23.12.0 o 24.1.0.<br />
Aquí *YY* son los dos últimos dígitos del año de lanzamiento, *M* o *MM* es el mes de lanzamiento (1-12) y *N* es el número de lanzamiento menor. Por lo general, *N* es "0".

{{% alert color="primary" %}}

Tenga en cuenta que no puede indicar a Aspose.Words que cambie o elimine esta información de los documentos de salida.

{{% /alert %}}
