Aspose.Words para C++ no requiere la instalación de ningún producto de terceros, como Microsoft Word. Aspose.Words en sí mismo es un motor para crear, modificar, convertir y renderizar documentos en varios formatos, incluidos los formatos de documentos de Microsoft Word.

## Entorno de Desarrollo

Puedes usar Aspose.Words para C++ al desarrollar aplicaciones para Windows, Linux o macOS.

Aspose.Words para C++ se puede utilizar para desarrollar aplicaciones en cualquier entorno de desarrollo que admita [Conjunto de herramientas de la plataforma Microsoft Visual Studio v143](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017), pero los entornos enumerados en la siguiente tabla son explícitamente compatibles:

### Dependencia adicional de Linux

Aspose.Words para C++ en Linux depende de [fontconfig](https://www.freedesktop.org/wiki/Software/fontconfig/) binarios tanto de biblioteca dinámica como de herramienta. Instálelo antes de usarlo:

1. Instalación de fontconfig en Ubuntu o Debian</br>
`sudo apt install libfontconfig fontconfig`
2. Instalación de fontconfig en Fedora o CentOs</br>
`sudo yum install fontconfig`
