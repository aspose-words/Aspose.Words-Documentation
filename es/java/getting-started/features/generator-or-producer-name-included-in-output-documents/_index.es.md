Usted puede saber fácilmente si un documento se produce utilizando Aspose.Words para Java. También puede encontrar el número de versión Aspose.Words para Java que se utilizó para crear un documento de salida particular. Aspose.Words escribe directamente esta información en el documento generado.

El número de versión se especifica en el formato *YY.MM.N* o *YY.M.N*, por ejemplo, 23.12.0 o 24.1.0.<br/>
Aquí. *YY* son los dos últimos dígitos del año de liberación, *M* o *MM* es el mes de liberación (1–12), y *N* es el número de liberación menor. Por lo general, *N* es "0".

{{% alert color="primary" %}}

Tenga en cuenta que no puede instruir Aspose.Words cambiar o eliminar esta información de los documentos de salida.

{{% /alert %}}
