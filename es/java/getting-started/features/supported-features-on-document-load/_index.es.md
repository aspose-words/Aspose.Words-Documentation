Aspose.Words permite a los usuarios trabajar con documentos en varios formatos y ofrece una amplia gama de opciones para esto. Por ejemplo, los usuarios pueden crear un documento desde cero o cargar un archivo existente en cualquier [Formatos de documentos compatibles](/words/es/java/supported-document-formats/), hacer algunos cambios en ese documento y luego guardar el resultado. Al importar un documento en el formato seleccionado, el usuario puede aplicar varias opciones de carga.

{{% alert color="primary" %}}

Es importante saber que muchas opciones de carga son comunes a todos los formatos soportados, y sólo algunas de las opciones son específicas para formato, como PDF, HTML, RTF y otros.

{{% /alert %}}

Para una lista completa de funciones, vea la API referencia:

- [opciones de carga para todos los formatos](https://reference.aspose.com/words/java/com.aspose.words/loadoptions/)
- [Opciones de carga CHM](https://reference.aspose.com/words/java/com.aspose.words/chmloadoptions/) para formato CHM
- [Opciones de carga HTML](https://reference.aspose.com/words/java/com.aspose.words/htmlloadoptions/) para formato HTML
- [Opciones de carga PDF](https://reference.aspose.com/words/java/com.aspose.words/pdfloadoptions/) para formato PDF
- [Opciones de carga RTF](https://reference.aspose.com/words/java/com.aspose.words/rtfloadoptions/) para formato RTF
- [Opciones de carga TXT](https://reference.aspose.com/words/java/com.aspose.words/txtloadoptions/) para formato TXT
