{{% alert color="primary" %}}

Aspose.Words tiene una interfaz para generar códigos de barras personalizados que lo hace muy fácil de usar [Aspose.Words](https://products.aspose.com/words/java/) y [Supongamos. BarCode](https://products.aspose.com/barcode/java/) para hacer imágenes de código de barras en documentos de salida. Por ejemplo, puede cargar un documento DOC, OOXML o RTF que contenga `DISPLAYBARCODE` Field into Aspose.Words, proporcionar su implementación de generador de códigos de barras personalizados y guardar a formatos de página fijos como PDF, XPS etc. Un típico `DISPLAYBARCODE` campo tiene la siguiente sintaxis:

{ `DISPLAYBARCODE` "SomeData" QR \h 720 }

{{% /alert %}}
