Aspose.Words ahora aprovecha los modelos avanzados de AI para mejorar el procesamiento y análisis de documentos. Con el Aspose.Words.En el espacio de nombres AI, los desarrolladores pueden acceder a funciones que utilizan AI para tareas como resumen y análisis de documentos, integrando modelos de lenguaje generativo de líderes de la industria.

La biblioteca Aspose.Words actualmente admite la integración con OpenAI y los modelos de lenguaje generativo de Google. Estos modelos brindan capacidades poderosas para trabajar con documentos. Al permitir una interacción fluida con estas AI herramientas, Aspose.Words simplifica tareas complejas y mejora la productividad de los desarrolladores.

Puede crear una instancia de modelo utilizando el método [create](https://reference.aspose.com/words/python-net/aspose.words.ai/aimodel/create/) de la clase [AiModel](https://reference.aspose.com/words/python-net/aspose.words.ai/aimodel/). La lista completa de tipos de modelos admitidos se enumera en la enumeración [AiModelType](https://reference.aspose.com/words/python-net/aspose.words.ai/aimodeltype/).

El siguiente ejemplo de código muestra cómo crear una nueva instancia de la clase **AiModel**.

{{< highlight python >}}
def create(self, model_type: aspose.words.ai.AiModelType):
    ...
{{< /highlight >}}
