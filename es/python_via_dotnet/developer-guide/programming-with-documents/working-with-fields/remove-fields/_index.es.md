A veces es necesario eliminar un campo del documento. Esto puede ocurrir cuando se va a reemplazar con un tipo de campo diferente o cuando el campo ya no es necesario en el documento. Por ejemplo, un campo `TOC` al guardar en HTML.

Para eliminar un campo insertado en un documento usando [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/), use el objeto [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/) devuelto, que proporciona el método [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) conveniente para eliminar fácilmente el campo del documento.
