Puede saber fácilmente si un documento se produce utilizando Aspose.Words para Python via .NET. También puede encontrar el número de versión de Aspose.Words para Python via .NET que se utilizó para crear un documento de salida en particular. Aspose.Words escribe directamente esta información en el documento generado.

El número de versión se especifica en el formato *YY.MM.N* o *YY.M.N*, por ejemplo, 23.12.0 o 24.1.0.<br />
Aquí *YY* son los dos últimos dígitos del año de lanzamiento, *M* o *MM* son el mes de lanzamiento (1 a 12) y *N* es el número de lanzamiento menor. Normalmente, *N* es "0".

{{% alert color="primary" %}}

Tenga en cuenta que puede indicarle a Aspose.Words que elimine esta información de los documentos de salida.

{{% /alert %}}
