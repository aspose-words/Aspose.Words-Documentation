A veces es necesario eliminar un campo de un documento. Esto puede ocurrir cuando es necesario reemplazarlo con un tipo de campo diferente o cuando el campo ya no es necesario en el documento. Por ejemplo, el campo `TOC` al guardar en HTML.

Para eliminar un campo insertado en un documento usando [DocumentBuilder](https://reference.aspose.com/words/es/net/aspose.words/documentbuilder/).[InsertField](https://reference.aspose.com/words/es/net/aspose.words/documentbuilder/insertfield/), use el objeto [Field](https://reference.aspose.com/words/es/net/aspose.words.fields/field/) devuelto, que proporciona un método [Remove](https://reference.aspose.com/words/es/net/aspose.words.fields/field/remove/) conveniente para eliminar fácilmente el campo del documento.
