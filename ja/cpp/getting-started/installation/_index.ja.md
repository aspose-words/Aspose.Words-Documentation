## インストールまたは更新 C++の場合Aspose.Words 管理NuGetパッケージ{#install-or-update-aspose-words-for-cpp-using-nuget}の使用

NuGetはダウンロードしてインストールする最も簡単な方法です [C++の場合Aspose.Words](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. これを行うには、次の手順に従います:

1. メニューから*Microsoft Visual Studio*と*Manage NuGet Packages*を開き、パッケージマネージャを開きます
2. 「Aspose」または「aspose」を検索します。目的のAspose APIを見つけるための「words」<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. 目的のAspose APIを選択し、「インストール」をクリックします。

選択したAPIがダウンロードされ、プロジェクト内で参照されます。

更新することもできます C++の場合Aspose.Words *Manage NuGet Packages*を使用して目的のバージョンに移動します。

## インストールまたは更新 C++の場合Aspose.Words Package Manager Consoleを使用する

また、インストールまたは更新することもできます C++の場合Aspose.Words APIsは*Package Manager Console*を使用します。 これを行うには、次の手順に従います:

1. メニューから*Microsoft Visual Studio*と*Package Manager Console*を開き、パッケージマネージャーコンソールを開きます
2. コマンド`Install-Package Aspose.Words.Cpp -Version 21.3.0`を入力してEnterキーを押し、最新のフルリリースをアプリケーションにインストールします
3. ダウンロードすると、確認メッセージが表示されます<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. これで、Aspose.Wordsがアプリケーションで正常に追加され、参照されたことがわかります

*Package Manager Console*では、`Update-Package Aspose.Words.Cpp`コマンドを使用して、Aspose.Wordsパッケージの更新を確認し、存在する場合はインストールすることもできます。

## インストール C++の場合Aspose.Words 手動で

あなたはダウンロードすることができます [C++の場合Aspose.Words NuGetギャラリーから](https://www.nuget.org/packages/Aspose.Words.Cpp/) 情報セクションの"手動ダウンロード"リンクをクリックしてください。 オフラインAPIのドキュメントは、ダウンロードしたZIPパッケージにもあります。

使用するには C++の場合Aspose.Words アプリケーションでは、次の手順に従います:

1. 最新のファイルをダウンロードする
2. このフォルダを本番環境に解凍します
3. 使用するには C++の場合Aspose.Words、プロジェクトのIncludeフォルダーとLibフォルダーを参照してください
