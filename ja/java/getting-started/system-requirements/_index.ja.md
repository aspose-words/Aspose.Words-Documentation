### JAIをインストールする方法 Linux

   1。 [野田64](https://download.java.net/media/jai/builds/release/1_1_3/jai-1_1_3-lib-linux-amd64.tar.gz) 64ビットの場合(Intelプロセッサを使用する場合でも)
1。 ファイルを一時ディレクトリに抽出します。
1。 交通アクセス JAR JDK/JREへのファイル **lib/ext ライブラリ** フォルダ。
1。 SOファイルをJDK/JREに移動 **lib/amd64 ライブラリ** フォルダ。<br/>

   例えば、64ビット Ubuntu システム、手順1-4はこのようになります。<br/>
   {{< highlight csharp >}}
   $ cd /tmp
   $ wget https://download.java.net/media/jai/builds/release/1_1_3/jai-1_1_3-lib-linux-amd64.tar.gz && \
   gunzip -c jai-1_1_3-lib-linux-amd64.tar.gz | tar xf - && \
   mv /tmp/jai-1_1_3/lib/*.jar $JAVA_HOME/jre/lib/ext/ && \
   mv /tmp/jai-1_1_3/lib/*.so $JAVA_HOME/jre/lib/amd64/ && \
   rm /tmp/jai-1_1_3-lib-linux-amd64.tar.gz && \
   rm -r /tmp/jai-1_1_3
   {{< /highlight >}}
1. ダウンロード **JAIイメージI/O 1.1**, 適切なアーキテクチャを選択する:
   1。 [i586の](https://download.java.net/media/jai-imageio/builds/release/1.1/jai_imageio-1_1-lib-linux-i586.tar.gz) 32ビットシステムの場合

## 一時的なファイルのためのフォルダ

Aspose.Words お問い合わせ Java コンピューターに一時ディレクトリを使用する。 `java.io.tmpdir` システムプロパティ。 Aspose.Words お問い合わせ Java 一時的なファイルを書き、レンダリングが完了するとすぐにこれらのファイルを削除します。

`java.io.tmpdir` スタンダード Java ディスクベースのストレージポリシーでも使用されるシステムプロパティ。 JVM が一時ファイルを書き込む場所を決定します。
