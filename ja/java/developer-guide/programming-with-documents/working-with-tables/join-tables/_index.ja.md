Aspose.Wordsドキュメントオブジェクトモデルで表されるテーブルは、独立した行とセルで構成されているため、テーブルを簡単に結合できます。

テーブルを操作して別のテーブルと結合するには、2番目のテーブルから最初のテーブルの最後に行を移動し、2番目のテーブルのコンテナを削除するだけ
