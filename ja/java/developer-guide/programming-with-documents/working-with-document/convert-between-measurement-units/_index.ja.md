提供されているオブジェクトプロパティのほとんど Aspose.Words API 幅や高さ、マージン、さまざまな距離などの測定を表現し、1インチが72ポイントに等しい点で値を受け入れます。 時々これは便利ではないし、ポイントは他の単位に変える必要があります。

Aspose.Words 提供して下さい [ConvertUtil](https://reference.aspose.com/words/java/com.aspose.words/convertutil/) さまざまな測定単位間で変換するヘルパー関数を提供するクラス。 インチ、ピクセル、ミリグラムをポイント、ポイントをインチ、ピクセルに変換し、1つの解像度から別のピクセルに変換できます。 ピクセルをポイントに変換し、その逆を96 dpi(ドット/インチ)解像度または指定されたdpi解像度で実行できます。

ザ・オブ・ザ・ **ConvertUtil** たとえば、インチはポイントよりも一般的な測定単位が多いため、さまざまなページプロパティを設定する際には、特にクラスが便利です。
