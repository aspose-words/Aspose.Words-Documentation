Microsoft Word 特別な意味を持つさまざまな文字を含むことができます。 通常、それらはフォーマットの目的のために使用され、通常のモードで描画されません。 標準のツールバーにあるShow/Hide Formatting Marksボタンをクリックすると、それらを表示できます。

テキストから文字を追加または削除する必要がある場合があります。 例えば、ドキュメントからテキストをプログラム的に取得する場合、 Aspose.Words ほとんどの制御文字を保存しますので、このテキストで動作する必要がある場合は、おそらく文字を削除または置換する必要があります。

ザ・オブ・ザ・ [ControlChar](https://reference.aspose.com/words/java/com.aspose.words/controlchar/) クラスは、多くの場合、文書で遭遇する制御文字を表す定数のリポジトリです。 同じ定数の文字と文字列の両方のバージョンを提供します。 例えば、文字列 [LineBreak](https://reference.aspose.com/words/java/com.aspose.words/controlchar/#LINE-BREAK) そして木 [LineBreakChar](https://reference.aspose.com/words/java/com.aspose.words/controlchar/#LINE-BREAK_CHAR) 同じ値を持っています。
