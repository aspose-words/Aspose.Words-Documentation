Às vezes é necessário remover um campo do documento. Isso pode ocorrer quando ele for substituído por um tipo de campo diferente ou quando o campo não for mais necessário no documento. Por exemplo, um campo `TOC` ao salvar em HTML.

Para remover um campo inserido em um documento usando [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/), use o objeto [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/) retornado, que fornece o método [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) conveniente para remover facilmente o campo do documento.
