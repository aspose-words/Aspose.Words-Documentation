Aspose.Words agora utiliza modelos avançados AI para melhorar o processamento e a análise de documentos. Com o Aspose.Words.AI namespace, os desenvolvedores podem acessar recursos que utilizam AI Para Tarefas como sumarização e análise de documentos, integrando modelos de linguagem generativa de líderes do setor.

Aspose.Words biblioteca atualmente suporta integração com OpenAI e modelos de linguagem generativa do Google. Esses modelos trazem recursos poderosos para trabalhar com documentos. Ao permitir uma interação perfeita com essas ferramentas AI, Aspose.Words simplifica tarefas complexas e aumenta a produtividade dos desenvolvedores.

Você pode criar uma instância de modelo usando o método [create](https://reference.aspose.com/words/python-net/aspose.words.ai/aimodel/create/) da classe [AiModel](https://reference.aspose.com/words/python-net/aspose.words.ai/aimodel/). A lista completa dos tipos de modelo suportados está listada na enumeração [AiModelType](https://reference.aspose.com/words/python-net/aspose.words.ai/aimodeltype/).

O exemplo de código a seguir mostra como criar uma nova instância da classe **AiModel**.

{{< highlight python >}}
def create(self, model_type: aspose.words.ai.AiModelType):
    ...
{{< /highlight >}}
