Um "Olá, mundo!" O código geralmente é o primeiro exemplo simples para escrever "Aspose.Words para Python via .NET" e também pode ser usado como um teste de integridade para garantir que o software destinado a compilar ou executar o código-fonte esteja instalado corretamente.

A biblioteca "Aspose.Words for Python via .NET" oferece aos desenvolvedores acesso direto para criar, modificar, mesclar, converter e comparar documentos do Word e da Web. PDF, DOCX, DOC, RTF, ODT, EPUB, HTML e muitos outros formatos de arquivo são [Formatos de documentos suportados](/words/pt/python-net/supported-document-formats/).

O snippet Below code segue estas etapas:

1. Crie um novo [Document](https://reference.aspose.com/words/python-net/aspose.words/document/) vazio
1. Inicialize uma classe [DocumentBuilder](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/)
1. Insira texto no documento, comece usando o método [write](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/write/) simples
1. Abra um [Document](https://reference.aspose.com/words/python-net/aspose.words/document/) existente de um arquivo. Detecta automaticamente o formato do arquivo
1. Documento [Append](https://reference.aspose.com/words/python-net/aspose.words/document/append_document/) "A" ao e do documento "B"
1. [Save](https://reference.aspose.com/words/python-net/aspose.words/document/save/) a saída como PDF
