Um "Olá, mundo!" O código geralmente é o primeiro exemplo simples para escrever "Aspose.Words para .NET" e também pode ser usado como um teste de integridade para garantir que o software destinado a compilar ou executar o código-fonte esteja instalado corretamente.

A biblioteca C# "Aspose.Words para .NET" oferece aos desenvolvedores acesso direto para criar, modificar, mesclar, converter e comparar documentos do Word e da Web. PDF, DOCX, DOC, RTF, ODT, EPUB, HTML e muitos outros formatos de arquivo são [Formatos de documentos suportados](/words/pt/net/supported-document-formats/).

O snippet Below code segue estas etapas:

1. Crie um novo [Document](https://reference.aspose.com/words/net/aspose.words/document) vazio
1. Inicialize uma classe [DocumentBuilder](https://reference.aspose.com/words/net/aspose.words/documentbuilder/)
1. Insira texto no documento, comece usando o método [Write](https://reference.aspose.com/words/net/aspose.words/documentbuilder/write/) simples
1. Abra um [Document](https://reference.aspose.com/words/net/aspose.words/document/document/) existente de um arquivo. Detecta automaticamente o formato do arquivo
1. Documento [Append](https://reference.aspose.com/words/net/aspose.words/document/appenddocument/) "A" ao e do documento "B"
1. [Save](https://reference.aspose.com/words/net/aspose.words/document/save/) a saída como PDF
