Aspose.Words fornece configurações de fallback de fonte predefinidas para Google Noto fontes. Estas são fontes gratuitas licenciadas sob SIL Open Font License, que podem ser descarregadas a partir de Google Noto Fonts. A classe **FontFallbackSettings** fornece um método [LoadNotoFallbackSettings](https://reference.aspose.com/words/cpp/aspose.words.fonts/fontfallbacksettings/loadnotofallbacksettings/).
