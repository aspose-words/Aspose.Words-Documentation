Aspose.Words para C++ não requer a instalação de qualquer produto de terceiros, como Microsoft Word. Aspose.Words em si é um mecanismo para criar, modificar, converter e renderizar documentos em vários formatos, incluindo Microsoft Word formatos de documentos.

## Ambiente De Desenvolvimento

Você pode usar Aspose.Words Para C++ ao desenvolver aplicações para Windows, Linux ou macOS.

Aspose.Words Para C++ pode ser usado para desenvolver aplicações em qualquer ambiente de desenvolvimento que suporte [Microsoft Visual Studio Conjunto de ferramentas da plataforma v143](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017), mas os ambientes listados na tabela a seguir são explicitamente suportados:

### Dependência adicional de Linux

Aspose.Words para C++ em Linux depende de [fontconfig](https://www.freedesktop.org/wiki/Software/fontconfig/) binários biblioteca dinâmica e ferramenta. Instale-o antes de usar:

1. Instalando fontconfig em Ubuntu ou Debian</br>
`sudo apt install libfontconfig fontconfig`
2. Instalando fontconfig em Fedora ou CentOs</br>
`sudo yum install fontconfig`
