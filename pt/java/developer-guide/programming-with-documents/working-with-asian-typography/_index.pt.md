Asian Typography é um conjunto de opções para parágrafos de texto em documentos escritos em línguas asiáticas.

Aspose.Words suporta tipografia asiática usando o [ParagraphFormat](https://reference.aspose.com/words/net/aspose.words/paragraphformat/) classe e algumas de suas propriedades.

## Ajuste automaticamente o espaço entre texto asiático e latim ou números

Se você estiver projetando um modelo com texto oriental asiático e latino e deseja aprimorar a aparência de seu modelo de formulário controlando os espaços entre ambos os tipos de texto, você pode configurar seu modelo de formulário para ajustar automaticamente os espaços entre esses dois tipos de texto. Para conseguir isso, você pode usar [AddSpaceBetweenFarEastAndAlpha](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getAddSpaceBetweenFarEastAndAlpha) e [AddSpaceBetweenFarEastAndDigit](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getAddSpaceBetweenFarEastAndDigit) propriedades da `ParagraphFormat` classe.

## Definir opções de quebra de linha

A guia de tipografia asiática das propriedades de parágrafo caixa de diálogo em Microsoft Word tem grupo de quebra de linha. As opções deste grupo podem ser definidas usando o [FarEastLineBreakControl](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getFarEastLineBreakControl), [WordWrap](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getWordWrap), [HangingPunctuation](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getHangingPunctuation) propriedades da **ParagraphFormat** classe.
