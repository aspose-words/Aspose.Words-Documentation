PDF é um formato de página fixa que é muito popular entre os usuários e é amplamente suportado por várias aplicações, como um documento PDF parece o mesmo em qualquer dispositivo. Por esta razão, converter para PDF é uma característica importante de Aspose.Words.

PDF é um formato complexo por si só, pois tem uma estrutura de arquivo específica, modelo gráfico, incorporação de fontes e alguma funcionalidade de saída complexa, como tags de estrutura de documentos, criptografia, assinaturas digitais e formas editáveis. Além disso, converter um documento para PDF requer várias etapas de cálculo, que são complexas e demoradas.

Nesta seção, vamos considerar os principais problemas que podem surgir ao trabalhar com documentos em vários padrões PDF e descrever opções para resolvê-los.

## Qual PDF Padrão Aspose.Words Suportes

Aspose.Words agora permite que os usuários trabalhem com formatos PDF/A-1, PDF/A-2 e PDF/A-4, bem como PDF/UA-1:

* PDF/A-1 tem limitações graves, como transparência e algumas opções de compressão são proibidas
* PDF/A-2 elimina algumas das limitações de PDF/A-1, tais como o suporte de transparência e efeitos de camada ou incorporação de fontes OpenType
* PDF/A-4 assume níveis de conformidade revistos: conformidade regular PDF/A-4 é equivalente à conformidade U das versões anteriores, e a conformidade nível A é removida
* O conteúdo PDF/UA-1 deve ser etiquetado e padronizado de acordo com ISO 32000-1: 2008

PDF/A é um ISO- versão padronizada do PDF destinado a ser utilizado no arquivamento e armazenamento a longo prazo de documentos electrónicos. Ao mesmo tempo, PDF/UA é outro ISO versão padronizada de PDF projetado para garantir a acessibilidade para pessoas com deficiência que usam tecnologia assistiva. Para especificar o nível de conformidade com os padrões PDF, use o [Compliance](https://reference.aspose.com/words/java/com.aspose.words/pdfsaveoptions/#getCompliance) propriedade. Devido às condições de armazenamento, o documento PDF/A deve incorporar todas as fontes e desativar a criptografia, enquanto o PDF/UA só deve incorporar todas as fontes.

Nesta seção, vamos dar uma olhada em trabalhar com documentos PDF / A ou PDF / UA-1.

## Relevante ISO para PDF Normas

Para saber mais sobre os diferentes padrões PDF, verifique o seguinte ISOS:

- PDF 1,7 = ISO-32000-1: 2008
- PDF 2.0 = ISO-32000-2: 2020
- PDF/A-1 = ISO-19005-1: 2005
- PDF/A-2 = ISO-19005-2: 2011
- PDF/A-4 = ISO-19005-4: 2020
- PDF/UA-1 = ISO-14289: 2014
