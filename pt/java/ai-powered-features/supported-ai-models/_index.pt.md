Aspose.Words agora utiliza modelos avançados AI para melhorar o processamento e a análise de documentos. Com a funcionalidade baseada em Aspose.Words AI, os desenvolvedores podem acessar recursos que utilizam AI para Tarefas como sumarização e análise de documentos, integrando modelos de linguagem generativa de líderes do setor.

Aspose.Words biblioteca atualmente suporta integração com OpenAI e modelos de linguagem generativa do Google. Esses modelos trazem recursos poderosos para trabalhar com documentos. Ao permitir uma interação perfeita com essas ferramentas AI, Aspose.Words simplifica tarefas complexas e aumenta a produtividade dos desenvolvedores.

Você pode criar uma instância de modelo usando o método [Create](https://reference.aspose.com/words/java/com.aspose.words/aimodel/#create-int) da classe [AiModel](https://reference.aspose.com/words/java/com.aspose.words/aimodel/). A lista completa de tipos de modelos suportados está listada na enumeração [AiModelType](https://reference.aspose.com/words/java/com.aspose.words/aimodeltype/).

O exemplo de código a seguir mostra como criar uma nova instância da classe **AiModel**.

{{< highlight java>}}
