Você pode facilmente saber se um documento é produzido usando Aspose.Words para Java. Você também pode encontrar o número de versão de Aspose.Words para Java que foi usado para criar um documento de saída particular. Aspose.Words escreve diretamente esta informação no documento gerado.

O número da versão é especificado no formato *YY.MM.N* ou *YY.M.N*, por exemplo, 23.12.0 ou 24.1.0.<br/>
Aqui. *YY* são os últimos dois dígitos do ano de lançamento, *M* ou *MM* é o mês de lançamento (1–12), e *N* é o menor número de lançamento. Normalmente, *N* é "0".

{{% alert color="primary" %}}

Note que você não pode instruir Aspose.Words para alterar ou remover essas informações de documentos de saída.

{{% /alert %}}
