OpenType — это формат шрифтов, представленный для обеспечения лучшей поддержки международных языков и систем письма по сравнению с PostScript и TrueType. Возможности макета OpenType широко известны как функции OpenType. Пакет Aspose.Words.Shaping.HarfBuzz обеспечивает поддержку функций OpenType в Aspose.Words с использованием механизма формирования текста HarfBuzz.

Aspose.Words способен использовать внешние объекты формирователя текста. Формирователь текста представляет шрифт и вычисляет информацию о формировании текста. В документе обычно используется несколько шрифтов, поэтому необходима фабрика формирователя текста. Этот пакет содержит реализацию фабрики формирователя текста, используемую свойством Aspose.Words.Layout.LayoutOptions.TextShaperFactory.

{{% alert color="primary" %}}

Формирование текста выполняется только при экспорте в форматы PDF или XPS.

{{% /alert %}}

В типичном приложении один экземпляр фабрики формирователя текста используется всеми экземплярами документа. Всякий раз, когда создается формирователь текста, осуществляется доступ к файлу шрифта. Анализ файла шрифта — дорогостоящая операция, поэтому рекомендуется кэширование. Aspose.Words реализует класс BasicTextShaperCache, который обертывает реализацию фабрики формирователя текста и кэширует экземпляры формирователя текста, возвращаемые обернутой фабрикой.
