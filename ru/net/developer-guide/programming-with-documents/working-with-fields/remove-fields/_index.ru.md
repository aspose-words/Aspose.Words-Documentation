Иногда необходимо удалить поле из документа. Это может произойти, когда его необходимо заменить другим типом поля или когда поле больше не требуется в документе. Например, `TOC` поле при сохранении в HTML.

Удалить поле, вставленное в документ с помощью [DocumentBuilder](https://reference.aspose.com/words/net/aspose.words/documentbuilder/).[InsertField](https://reference.aspose.com/words/net/aspose.words/documentbuilder/insertfield/), использовать возвращенный [Field](https://reference.aspose.com/words/net/aspose.words.fields/field/) объект, который обеспечивает удобное [Remove](https://reference.aspose.com/words/net/aspose.words.fields/field/remove/) Способ легкого удаления поля из документа.
