Вы можете легко узнать, создан ли документ с помощью Aspose.Words для .NET. Вы также можете найти номер версии Aspose.Words для .NET Используется для создания конкретного выходного документа. Aspose.Words непосредственно записывает эту информацию в сгенерированный документ.

Номер версии указывается в формате *YY.MM.N* или *YY.M.N*, Например, 23.12.0 или 24.1.0.<br/>
Здесь *YY* Это последние две цифры года выпуска, *M* или *MM* месяц выпуска (1-12) и *N* Это незначительная цифра выпуска. Обычно, *N* Это "0".

{{% alert color="primary" %}}

Обратите внимание, что вы можете поручить Aspose.Words удалить эту информацию из выходных документов.

{{% /alert %}}
