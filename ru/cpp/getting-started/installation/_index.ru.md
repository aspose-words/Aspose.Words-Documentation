Перед началом работы убедитесь, что ваш компьютер соответствует требованиям к системе [системные требования ](/words/cpp/system-requirements/).

В этой статье объясняется, как установить Aspose.Words for C++ на ваш компьютер.

## Установите или обновите Aspose.Words для C++ с помощью управления NuGet пакетами {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet это самый простой способ загрузки и установки [Aspose.Words для C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. Для этого выполните следующие действия:

1. Откройте *Microsoft Visual Studio* и *Manage NuGet Packages* в меню, чтобы открыть диспетчер пакетов
2. Выполните поиск "aspose" или "aspose.words", чтобы найти нужный Aspose API.<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. Выберите нужный Aspose API и нажмите "Установить"

Выбранный API будет загружен и на него будет дана ссылка в вашем проекте.

Вы также можете обновить Aspose.Words для C++ до нужной версии, используя *Manage NuGet Packages*.

## Установить или обновить Aspose.Words для C++ используя Package Manager Console

Вы также можете установить или обновить Aspose.Words для C++ APIs, используя *Package Manager Console*. Для этого выполните следующие действия:

1. Откройте *Microsoft Visual Studio* и *Package Manager Console* в меню, чтобы открыть консоль менеджера пакетов
2. Введите команду `Install-Package Aspose.Words.Cpp -Version 21.3.0` и нажмите Enter, чтобы установить последнюю полную версию в ваше приложение
3. После загрузки вы увидите подтверждающие сообщения<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. Теперь вы можете обнаружить, что Aspose.Words был успешно добавлен и на него есть ссылка в вашем приложении

В *Package Manager Console* вы также можете использовать команду `Update-Package Aspose.Words.Cpp`, чтобы проверить наличие обновлений для пакета Aspose.Words и установить их, если они есть.

## Устанавливать Aspose.Words для C++ Вручную

Вы можете скачать [Aspose.Words для C++ из NuGet галереи](https://www.nuget.org/packages/Aspose.Words.Cpp/), нажав на ссылку "Ручная загрузка" в разделе "Информация". Документацию по автономному API также можно найти в загруженном ZIP-пакете.

Использовать Aspose.Words для C++ в своих приложениях выполните следующие действия:

1. Загрузите последние версии файлов
2. Распакуйте папку в рабочую среду
3. Чтобы использовать Aspose.Words для C++, обратитесь к папкам Include и Lib в вашем проекте
