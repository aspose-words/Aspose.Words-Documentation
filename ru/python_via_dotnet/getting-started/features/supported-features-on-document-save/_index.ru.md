Aspose.Words обладает одной из самых продвинутых функций преобразования из всех распространенных форматов обработки текстов.

Все импортированные документы, загруженные в Aspose.Words, могут быть экспортированы в любой другой поддерживаемый формат. При преобразовании функции сохраняются. Преобразование между форматами дает одни из лучших результатов преобразования.

Aspose.Words позволяет конвертировать документ любого размера при условии, что для выполнения операции достаточно оперативной памяти. Кроме того, если доступной памяти недостаточно, вы можете выбрать временную папку при экспорте в формат DOC или DOCX. Это позволяет временно сохранять ресурсы во время преобразования на диске. Эти файлы удаляются после завершения преобразования. Это позволяет сохранять большие документы даже при ограниченном объеме доступной памяти.
