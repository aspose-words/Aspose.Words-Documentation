Чтобы лучше ознакомиться с возможностями и функциональностью Aspose.Words, мы приводим примеры, которые можно загрузить из нашего репозитория GitHub, запустить и изучить в деталях.

В этой статье вы можете ознакомиться с системными требованиями, а также с информацией о том, как запускать примеры.

## Требования к программному обеспечению и предварительные условия

Пожалуйста, убедитесь, что у вас установлена программа Python 3.5 или более поздняя версия на yout PC.

## Загрузите и запустите примеры

Все примеры Aspose.Words для Python .NET размещены на [GitHub](https://github.com/aspose-words/Aspose.Words-for-Python-via-.NET). Вы можете либо клонировать репозиторий с помощью вашего любимого клиента GitHub, либо загрузить [файл ZIP](https://github.com/aspose-words/Aspose.Words-for-Python-via-.NET/archive/master.zip).

Получив копию хранилища, вы можете обнаружить, что:

- Все примеры находятся в папке **Examples**.
- Все приведенные примеры являются модульными тестами.

Чтобы запустить примеры, откройте вашу любимую командную строку:

- Измените каталог на каталог с интересующим вас примером.
- Запустите пример с помощью Python, например:
{{< highlight python >}}
python .\base_conversions.py
{{< /highlight >}}
