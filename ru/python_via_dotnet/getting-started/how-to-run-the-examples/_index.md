Чтобы лучше познакомиться с возможностями и функциями Aspose.Words, мы предоставляем примеры, которые можно загрузить из нашего репозитория GitHub, запустить и изучить подробно.

В этой статье вы можете найти системные требования, а также информацию о том, как запускать примеры.

## Требования к программному обеспечению и предварительные условия

Убедитесь, что на вашем компьютере установлена версия Python 3.5 или новее.

## Загрузите и запустите примеры

Все примеры Aspose.Words для Python .NET размещены на [GitHub](https://github.com/aspose-words/Aspose.Words-for-Python-via-.NET). Вы можете либо клонировать репозиторий, используя свой любимый клиент GitHub, либо загрузить [ZIP-файл](https://github.com/aspose-words/Aspose.Words-for-Python-via-.NET/archive/master.zip).

Получив копию репозитория, вы можете обнаружить следующее:

- Все примеры находятся в папке **Examples**.
- Все примеры — модульные тесты.

Чтобы запустить примеры, откройте любимую командную строку:

- Измените каталог на каталог с интересующим примером.
- Запустите пример с Python, например:
{{< highlight python >}}
python .\base_conversions.py
{{< /highlight >}}
