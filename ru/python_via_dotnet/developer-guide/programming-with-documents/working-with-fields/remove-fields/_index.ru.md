Иногда необходимо удалить поле из документа. Это может произойти, когда он должен быть заменен другим типом поля или когда поле больше не требуется в документе. Например, a `TOC` поле при сохранении в HTML.

Удалить поле, вставленное в документ с помощью [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/), использовать возвращенный [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/) объект, который обеспечивает удобное [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) Способ легко удалить поле из документа.
