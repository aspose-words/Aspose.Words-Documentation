Aspose.Words теперь используются расширенные модели AI для улучшения обработки и анализа документов. Благодаря функциональности, основанной на Aspose.Words AI, разработчики могут получить доступ к функциям, которые используют AI для таких задач, как обобщение и анализ документов, интегрируя модели порождающего языка от лидеров отрасли.

Aspose.Words в настоящее время библиотека поддерживает интеграцию с моделями OpenAI и Google generative language. Эти модели предоставляют мощные возможности для работы с документами. Обеспечивая бесперебойное взаимодействие с этими инструментами AI, Aspose.Words упрощает сложные задачи и повышает производительность для разработчиков.

Вы можете создать экземпляр модели, используя метод [Create](https://reference.aspose.com/words/java/com.aspose.words/aimodel/#create-int) класса [AiModel](https://reference.aspose.com/words/java/com.aspose.words/aimodel/). Полный список поддерживаемых типов моделей приведен в перечислении [AiModelType](https://reference.aspose.com/words/java/com.aspose.words/aimodeltype/).

В следующем примере кода показано, как создать новый экземпляр класса **AiModel**.

{{< highlight java>}}
public static AiModel create(int modelType)
