Иногда необходимо удалить поле из документа. Это может произойти, когда он должен быть заменен другим типом поля или когда поле больше не требуется в документе. Например, a `TOC` поле при сохранении в HTML.

Удалить поле, вставленное в документ с помощью [DocumentBuilder](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/).[InsertField](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/#insertField-int-boolean), использовать возвращенный [Field](https://reference.aspose.com/words/java/com.aspose.words/field/) объект, который обеспечивает удобное [Remove](https://reference.aspose.com/words/java/com.aspose.words/field/#remove) Способ легкого удаления поля из документа.
