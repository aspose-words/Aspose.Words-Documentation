## Скачать из GitHub

Все примеры Aspose.Words для Java размещены на [Github](https://github.com/aspose-words/Aspose.Words-for-Java). Вы можете либо клонировать репозиторий с помощью своего любимого клиента Github, либо загрузить ZIP-файл. [здесь](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

Извлеките содержимое ZIP-файла в любую папку на вашем компьютере. Все примеры расположены в **Examples** папка.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

Проект использует Maven Построить систему. Любая современная IDE может легко открыть или импортировать проект. Ниже мы покажем вам, как использовать популярные IDE для создания и запуска примеров.

## IntelliJ IDEA

Нажмите на **File** меню и выбрать **Open**. Перейдите в папку проекта и выберите **xml** Файл.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

Он откроет проект и автоматически загрузит зависимости. Из вкладки "Проект" просмотрите примеры в **src/main/java** папка. Чтобы запустить пример, просто щелкните правой кнопкой мыши по файлу и выберите "Запустить...", пример будет выполнен, и вывод будет показан в встроенном окне вывода консоли.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

Нажмите на **File** меню и выбрать **Import**. Выбрать **Maven** - Существующий Maven Проекты.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

Просмотрите папку, которую вы клонировали или скачали из GitHub и выбрать **xml** Файл.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

Он откроет проект и автоматически загрузит зависимости. Из вкладки Package Explorer просмотрите примеры в **src/main/java** папка. Чтобы запустить пример, просто нажмите на файл и выберите **Run As** - **Java Application**, Пример будет выполнен, и выход будет показан в встроенном окне вывода консоли.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

Нажмите на **File** меню и выбрать **Open Project**. Просмотрите папку, которую вы клонировали или скачали из GitHub. Икона о **Examples** Показатель покажет, что его Maven Проект. Выберите примеры и откройте их.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

Он откроет проект и автоматически загрузит зависимости. Из вкладки "Проекты" просмотрите примеры в **исходные пакеты**. Чтобы запустить пример, просто нажмите на файл и выберите **Run File**, Пример будет выполнен, и выход будет показан в встроенном окне вывода консоли.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)

## Содействовать

Если вы хотите добавить или улучшить пример, мы рекомендуем вам внести свой вклад в проект. Все примеры и демонстрационные проекты в этом хранилище являются открытыми и могут свободно использоваться в ваших собственных приложениях.

Чтобы внести свой вклад, вы можете развязать репозиторий, отредактировать исходный код и создать запрос на вытягивание. Мы рассмотрим изменения и включим их в репозиторий, если они окажутся полезными.
