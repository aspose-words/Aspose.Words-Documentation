## Скачать с GitHub

Все примеры Aspose.Words для Java размещены на [Github](https://github.com/aspose-words/Aspose.Words-for-Java). Вы можете либо клонировать репозиторий, используя свой любимый клиент Github, либо загрузить файл ZIP с [здесь](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

Извлеките содержимое файла ZIP в любую папку на вашем компьютере. Все примеры находятся в папке **Examples**.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

В проекте используется система сборки Maven. Любой современный пользователь IDE может легко открыть или импортировать проект и его зависимости. Ниже мы покажем вам, как использовать популярную систему IDEs для сборки и запуска примеров.

## IntelliJ IDEA

Нажмите на меню **File** и выберите пункт **Open**. Перейдите в папку проекта и выберите файл **pom.xml**.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

Программа откроет проект и автоматически загрузит зависимости. На вкладке "Проект" просмотрите примеры в папке **src/main/java**. Чтобы запустить пример, просто щелкните правой кнопкой мыши на файле и выберите "Выполнить..", пример будет выполнен, а результат будет показан во встроенном окне вывода консоли.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

Нажмите на меню **File** и выберите **Import**. Выберите **Maven** - Существующие Maven проекта.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

Перейдите в папку, которую вы клонировали или загрузили из GitHub, и выберите файл **pom.xml**.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

Программа откроет проект и автоматически загрузит зависимости. На вкладке "Обозреватель пакетов" просмотрите примеры в папке **src/main/java**. Чтобы запустить пример, просто щелкните правой кнопкой мыши на файле и выберите **Run As** - **Java Application**, пример будет выполнен, а результат будет показан во встроенном окне вывода консоли.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

Нажмите на меню **File** и выберите **Open Project**. Перейдите к папке, которую вы клонировали или загрузили из GitHub. Значок папки **Examples** будет означать, что это проект Maven. Выберите Примеры и откройте его.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

Он откроет проект и автоматически загрузит зависимости. На вкладке "Проекты" просмотрите примеры в **source packages**. Чтобы запустить пример, просто щелкните правой кнопкой мыши на файле и выберите **Run File**, пример будет выполнен, а результат будет показан во встроенном окне вывода консоли.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)

## Способствовать

Если вы хотите добавить или улучшить пример, мы рекомендуем вам внести свой вклад в проект. Все примеры и демонстрационные проекты в этом репозитории имеют открытый исходный код и могут свободно использоваться в ваших собственных приложениях.

Чтобы внести свой вклад, вы можете выполнить форк репозитория, отредактировать исходный код и создать запрос на обновление. Мы рассмотрим изменения и добавим их в репозиторий, если сочтем полезными.
