## تثبيت أو تحديث Aspose.Words ل C++ باستخدام إدارة NuGet حزم {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet هو أسهل طريقة لتحميل وتثبيت [Aspose.Words ل C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. للقيام بذلك، اتبع الخطوات التالية:

1. افتح *Microsoft Visual Studio* و *Manage NuGet Packages* من القائمة لفتح مدير الحزم
2. ابحث عن" أسبوس "أو " أسبوس".الكلمات " للعثور على المطلوب أسبوس API<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. حدد المطلوب أسبوس API وانقر على "تثبيت"

سيتم تنزيل API المحدد والإشارة إليه في مشروعك.

يمكنك أيضا تحديث Aspose.Words ل C++ إلى الإصدار المطلوب باستخدام *Manage NuGet Packages*.

## تثبيت أو تحديث Aspose.Words ل C++ باستخدام Package Manager Console

يمكنك أيضا تثبيت أو تحديث Aspose.Words ل C++ APIs باستخدام *Package Manager Console*. للقيام بذلك، اتبع الخطوات التالية:

1. افتح *Microsoft Visual Studio* و *Package Manager Console* من القائمة لفتح وحدة تحكم مدير الحزم
2. اكتب الأمر `Install-Package Aspose.Words.Cpp -Version 21.3.0` واضغط على إدخال لتثبيت أحدث إصدار كامل في التطبيق الخاص بك
3. بمجرد التنزيل، سترى رسائل تأكيد<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. الآن قد تجد أن Aspose.Words تمت إضافتها بنجاح والمشار إليها في التطبيق الخاص بك بالنسبة لك

في *Package Manager Console*، يمكنك أيضا استخدام الأمر `Update-Package Aspose.Words.Cpp` للتحقق من وجود تحديثات لحزمة Aspose.Words وتثبيتها إذا كانت موجودة.
