في بعض الأحيان قد ترغب في إعطاء الوصول لضوابط الوصلات البينية التي تدير الرمز لتعديل الوثائق. Aspose.Words API يوفر `WebExtensions` (ج) حيز الاسم، الذي يمنح فئات مختلفة لتكييف العناصر وخصائص توسيع نطاق المفردات المخصصة لتمثيل المكاتب.

ويمكن تقسيم مساحة الاسم على شبكة الإنترنت بصورة مشروطة إلى:

* ♪ Objects for working with the task pane
* أهداف للعمل مع توسيع الشبكة

## استخدام الذباب

أغطية المهمة هي أسطح الواجهة التي تظهر على الجانب الأيمن من النافذة Microsoft Word. وتسمح مقلاة المهام للمستعملين بالوصول إلى ضوابط الوصلات البينية التي يمكن أن تستخدم الرمز لتعديل الوثائق.

## استخدام الإرشادات الشبكية

وتشكل عمليات توسيع الشبكة أدوات لتوسيع قدرات تطبيقات المكاتب والتفاعل مع محتوى وثائق المكتب. وتوفر عمليات الإرشاد على الشبكة وظائف إضافية لعملاء المكاتب من أجل تحسين خبرة المستعملين.
