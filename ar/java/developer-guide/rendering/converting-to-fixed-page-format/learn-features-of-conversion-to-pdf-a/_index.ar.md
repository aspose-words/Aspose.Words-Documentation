PDF is a fixed page format that is very popular among users and is widely supported by various applications, as a PDF document looks the same on any tool. ولهذا السبب، يشكل التحول إلى قوات الدفاع الشعبي سمة هامة من السمات Aspose.Words.

PDF is a complex format by itself, as it has a specific file structure, graphical model, font embedding, and some complex output functionity such as document structure tags, encryption, digital signatures, and editable forms. وبالإضافة إلى ذلك، يتطلب تحويل وثيقة إلى إدارة الدعم الميداني عدة مراحل حسابية معقدة وتستغرق وقتا طويلا.

وفي هذا الفرع، سننظر في المشاكل الرئيسية التي قد تنشأ عند العمل مع الوثائق الواردة في مختلف معايير PDF ونصف خيارات لحلها.

## الذي PDF المعيار Aspose.Words الدعم

Aspose.Words ويسمح الآن للمستعملين بالعمل مع صيغ PDF/A-1 و PDF/A-2 و PDF/A-4، فضلا عن PDF/UA-1:

* PDF/A-1 لديها قيود خطيرة مثل الشفافية وبعض خيارات الضغط المحظورة
* PDF/A-2 تلغي بعض القيود المفروضة على PDF/A-1، مثل دعم الشفافية وآثار الطبقات أو تجسيد الفواكه المفتوحة
* PDF/A-4 يفترض مستويات تطابق منقحة: تطابق منتظم بين PDF/A-4 يعادل تطابق الصيغ السابقة
* ينبغي توسيم محتوى PDF/UA-1 وتوحيده وفقاً لما ورد في ISO 32000-1: 2008

PDF/A is an ISO- النسخة الموحدة من PDF المراد استخدامها في حفظ الوثائق الإلكترونية وتخزينها على المدى الطويل. At the same time, PDF/UA is another ISO:: صيغة موحدة من PDF مصممة لضمان وصول الأشخاص ذوي الإعاقة الذين يستخدمون تكنولوجيا المساعدة. To specify the level of compliance with PDF standards, use the [Compliance](https://reference.aspose.com/words/java/com.aspose.words/pdfsaveoptions/#getCompliance) ملكية ونظراً لظروف التخزين، يجب أن تتضمن وثيقة PDF/A جميع العناوين والتشفير القابل للتحلل، في حين يجب على PDF/UA فقط أن تدمج جميع الفواكه.

وفي هذا الفرع، سنلقي نظرة أوثق على العمل مع وثائق PDF/A أو PDF/UA-1.

## ذات الصلة ISO for PDF المعايير

To learn more about the different PDF standards, check the following ISOS:

- PDF 1.7 = ISO-32000-1: 2008
- PDF 2.0 = ISO-32000-2: 2020
- PDF/A-1 = ISO-19005-1: 2005
- PDF/A-2 = ISO-19005-2: 2011
- PDF/A-4 = ISO-19005-4: 2020
- PDF/UA-1 = ISO-14289: 2014
