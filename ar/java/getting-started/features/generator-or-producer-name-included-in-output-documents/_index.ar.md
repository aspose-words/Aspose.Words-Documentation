يمكنك بسهولة معرفة ما إذا تم إنتاج مستند باستخدام Aspose.Words لـ Java. يمكنك أيضا العثور على رقم إصدار Aspose.Words لـ Java الذي تم استخدامه لإنشاء مستند إخراج معين. Aspose.Words يكتب هذه المعلومات مباشرة في المستند الذي تم إنشاؤه.

يتم تحديد رقم الإصدار بالتنسيق *YY.MM.N* أو *YY.M.N*، على سبيل المثال، 23.12.0 أو 24.1.0.<br />
هنا *YY* هو آخر رقمين من سنة الإصدار، *M* أو *MM* هو شهر الإصدار (1-12)، و *N* هو رقم الإصدار الثانوي. عادة، *N* هو "0".

{{% alert color="primary" %}}

لاحظ أنه لا يمكنك توجيه Aspose.Words لتغيير أو إزالة هذه المعلومات من مستندات الإخراج.

{{% /alert %}}
