문서에서 필드를 제거해야 하는 경우가 있습니다. 이는 다른 필드 유형으로 바꿔야 하거나 해당 필드가 문서에 더 이상 필요하지 않은 경우 발생할 수 있습니다. 예를 들어 HTML로 저장할 때 `TOC` 필드입니다.

[DocumentBuilder](https://reference.aspose.com/words/net/aspose.words/documentbuilder/).[InsertField](https://reference.aspose.com/words/net/aspose.words/documentbuilder/insertfield/)를 사용하여 문서에 삽입된 필드를 제거하려면 문서에서 필드를 쉽게 제거할 수 있는 편리한 [Remove](https://reference.aspose.com/words/net/aspose.words.fields/field/remove/) 방법을 제공하는 반환된 [Field](https://reference.aspose.com/words/net/aspose.words.fields/field/) 객체를 사용하십시오.
