Aspose.Words 다양한 기능을 사용자에게 제공합니다. 사용자는 문서를 지원되는 형식에서 다른 형식으로 변환하고 변환 프로세스 중에 해당 문서를 수정하는 것부터 구조화되고 시각적으로 매력적인 문서 작성 또는보고 자동화와 같은 비즈니스 작업에 이르기까지 엄청난 양의 문서 관련 작업을 수행 할 수 있습니다.

## 주요 특징

- 레이아웃 엔진(문서의 페이지 모델로 페이지를 구축 할 수있는 모듈)
- 문서 렌더링 PDF
- 이미지에 문서를 렌더링(이 기능이 포함되어 있습니다,그러나,우리는 여전히 개선에 노력하고 있습니다,그래서 베타 상태에주의하시기 바랍니다)
- 로드 및 저장 HTML 문서
- 저장 EPUB 문서
- 로드 및 저장 MHTML 문서
- 로드 중 MOBI 문서
- 문서 저장 XamlFlow 그리고 XamlFixed 형식
- 문서 로드 및 저장 ODT 형식
- 문서 저장 XPS 그리고 오픈XPS 형식
- 문서 저장 HtmlFixed (HTML 절대적으로 배치 된 요소를 사용하는 형식)형식
- 문서 저장 SVG 형식
- 문서 저장 PS, PCL, EMF 형식
- 삽입 HTML 문서에
- 삽입 SVG 문서에 이미지
- 삽입 온라인 비디오
- 창조 DML 차트
- 문서 레이아웃에 의존하는 필드 업데이트(예: `TOC`, `PAGE` 그 밖의
- 두 단어 문서 비교
- 읽기 및 쓰기 VBA 매크로

## 제한 사항

- 제한된 파일 형식 지원(*see [Supported Document Formats](/words/cpp/supported-document-formats/) for details*)
- 암호화 된 문서에 대한 지원 없음
- 인터넷에서 원격 리소스 다운로드에 대한 지원 없음
- 렌더링 기능에 대한 제한적이고 불안정한 지원
- 보고 기능에 대한 지원 없음
