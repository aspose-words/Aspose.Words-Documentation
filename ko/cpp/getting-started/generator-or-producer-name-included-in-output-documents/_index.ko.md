다음을 사용하여 문서가 생성되는지 쉽게 알 수 있습니다 Aspose.Words 에 대한 C++. 또한 버전 번호를 찾을 수 있습니다 Aspose.Words 에 대한 C++ 특정 출력 문서를 만드는 데 사용되었습니다. Aspose.Words 이 정보를 생성 된 문서에 직접 씁니다.

버전 번호는 다음과 같은 형식으로 지정됩니다 *YY.MM.N* 또는 *YY.M.N* 예를 들어, 23.12.0 또는 24.1.0.<br />
여기 *YY* 출시 연도의 마지막 두 자릿수입니다, *M* 또는 *MM* 릴리스 월(1-12)이며 *N* 마이너 릴리스 번호입니다. 보통, *N* 이 "0".

{{% alert color="primary" %}}

다음을 지시할 수 없습니다 Aspose.Words 출력 문서에서 이 정보를 변경하거나 제거합니다.

{{% /alert %}}
