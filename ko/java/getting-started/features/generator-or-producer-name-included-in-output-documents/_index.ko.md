Java에 대해Aspose.Words을 사용하여 문서가 생성되는지 쉽게 알 수 있습니다. 또한 특정 출력 문서를 만드는 데 사용된Java의Aspose.Words버전 번호를 찾을 수 있습니다. Aspose.Words이 정보를 생성된 문서에 직접 씁니다.

버전 번호는*YY.MM.N*또는*YY.M.N*형식(예:23.12.0또는24.1.0)으로 지정됩니다.<br />
여기서*YY*은 릴리스 연도의 마지막 두 자릿수,*M*또는*MM*는 릴리스 월(1-12),*N*은 마이너 릴리스 번호입니다. 일반적으로*N*은"0"입니다.

{{% alert color="primary" %}}

Aspose.Words에 출력 문서에서 이 정보를 변경하거나 제거하도록 지시할 수 없습니다.

{{% /alert %}}
