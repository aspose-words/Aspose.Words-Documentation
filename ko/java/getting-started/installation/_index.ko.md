당신의 기계를 만나십시오 [시스템 요구 사항](/words/ko/java/system-requirements/) 시작하기 전에.

이 문서는 설치 방법 설명 Aspose.Words 제품정보 Java 당신의 컴퓨터에.

## 설치하기 Aspose.Words 제품정보 Java 이름 * Maven 회사연혁 {#install-aspose-words-for-java-from-maven-repository}

모든 호스트 Java APIs 에 [Maven 관련 기사](https://releases.aspose.com/words/java/)· 쉽게 사용할 수 있습니다 Aspose.Words 제품정보 Java API 직접 당신의 Maven 간단한 윤곽을 가진 프로젝트:

1. 명세 먼저 Aspose를 지정해야 합니다. Maven 저장소 구성/위치 Maven 아래와 같이 pom.xml:
	{{< highlight html >}}
	<repositories>
		<repository>
			<id>AsposeJavaAPI</id>
			<name>Aspose Java API</name>
			<url>https://releases.aspose.com/java/repo/</url>
		</repository>
	</repositories>
	{{< /highlight >}}
2. 명세 그런 다음 정의 Aspose.Words 제품정보 Java API 다음과 같이 pom.xml의 의존도:
	{{< highlight html >}}
	<dependencies>
		<dependency>
			<groupId>com.aspose</groupId>
			<artifactId>aspose-words</artifactId>
			<version>22.11</version>
			<classifier>jdk17</classifier>
		</dependency>
		<dependency>
			<groupId>com.aspose</groupId>
			<artifactId>aspose-words</artifactId>
			<version>22.11</version>
			<classifier>javadoc</classifier>
		</dependency>
	</dependencies>
	{{< /highlight >}}
3. 명세 감사합니다! 당신은 성공적으로 정의했다 Aspose.Words 제품정보 Java 당신의 의지 Maven 프로젝트.
