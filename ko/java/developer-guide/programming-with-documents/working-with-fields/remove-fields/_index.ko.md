때때로 문서에서 필드를 제거해야 합니다. 다른 필드 유형으로 바꾸거나 문서에서 필드가 더 이상 필요하지 않을 때 발생할 수 있습니다. 예를 들어HTML에 저장할 때`TOC`필드입니다.

[DocumentBuilder](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/)를 사용하여 문서에 삽입된 필드를 제거합니다.[InsertField](https://reference.aspose.com/words/java/com.aspose.words/documentbuilder/#insertField-int-boolean),반환된[Field](https://reference.aspose.com/words/java/com.aspose.words/field/)개체를 사용하여 문서에서 필드를 쉽게 제거할 수 있는 편리한[Remove](https://reference.aspose.com/words/java/com.aspose.words/field/#remove)메서드를 제공합니다.
