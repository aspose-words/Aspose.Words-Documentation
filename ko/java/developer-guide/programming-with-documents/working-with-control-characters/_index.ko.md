Microsoft Word 문서는 특별한 의미를 가진 다양한 문자를 포함할 수 있습니다. 일반적으로 그들은 포맷 목적으로 사용되며 정상적인 모드에서 그려지지 않습니다. Standard toolbar에 위치한 Show/Hide Formatting Marks 버튼을 클릭하면 볼 수 있습니다.

때로는 텍스트에서 문자를 추가하거나 제거해야합니다. 예를 들어, 문서에서 text programmatically를 얻을 때, Aspose.Words 제어 문자의 대부분을 보존하므로이 텍스트로 작업해야 할 경우 문자를 제거하거나 교체해야합니다.

더 보기 [ControlChar](https://reference.aspose.com/words/java/com.aspose.words/controlchar/) 클래스는 문서에서 자주 발생하는 제어 문자를 나타내는 일정에 대한 저장소입니다. 동일한 상수의 char와 string 버전을 모두 제공합니다. 예제, 문자열 [LineBreak](https://reference.aspose.com/words/java/com.aspose.words/controlchar/#LINE-BREAK) 숯 [LineBreakChar](https://reference.aspose.com/words/java/com.aspose.words/controlchar/#LINE-BREAK_CHAR) 같은 값이 있습니다.
