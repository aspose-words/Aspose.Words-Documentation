내 계정 Aspose.Words, 범위는 문서의 다른 나무 같은 모델로 "flat window"입니다.

{{% /alert %}}

당신이 일한 경우 Microsoft Word 자동화, 당신은 아마 주요 도구 중 하나가 검사하고 문서 콘텐츠를 수정하는 것을 알고 `Range` 이름 * 범위는 문서 내용과 포맷으로 "window"과 같습니다.

Aspose.Words 또한 있다 [Range](https://reference.aspose.com/words/java/com.aspose.words/range/) 클래스와 비슷하게 관찰하고 행동하도록 설계되었습니다. **Range** 내 계정 Microsoft Word· 그러나 **Range** 문서의 임의 부분을 커버 할 수 없으며 **Start** · **End**, 모든 문서 노드에 접근할 수 있습니다. [Document](https://reference.aspose.com/words/java/com.aspose.words/document/) 이름 * 다른 말에서 각 노드는 자신의 범위를 가지고 있습니다. 더 보기 **Range** 객체는 범위 내에서 텍스트, 북마크 및 양식 필드에 액세스하고 수정할 수 있습니다.
