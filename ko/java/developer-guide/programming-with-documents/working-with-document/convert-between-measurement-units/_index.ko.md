대부분의 개체 속성에서 제공 Aspose.Words API 폭 또는 높이, 마진 및 다양한 거리와 같은 측정을 나타내는 것은 포인트의 값을 받아 1 인치가 72 점을 동일하게합니다. 때로는 편리하고 포인트는 다른 단위로 변환 할 필요가 없습니다.

Aspose.Words 제품정보 [ConvertUtil](https://reference.aspose.com/words/java/com.aspose.words/convertutil/) 다양한 측정 단위로 변환하는 돕기 기능을 제공하는 클래스. 인치, 픽셀 및 밀리미터를 포인트로 변환하고, 인치와 픽셀로 전환하고, 하나의 해상도에서 다른 픽셀로 변환 할 수 있습니다. 포인트로 픽셀을 변환하고 vice versa는 96dpi (인치당 점) 해상도 또는 지정된 dpi 해상도에서 수행 할 수 있습니다.

더 보기 **ConvertUtil** 클래스는 특히 유용합니다 때문에 다양한 페이지 속성을 설정할 때, 예를 들어, 인치는 포인트보다 더 일반적인 측정 단위입니다.
