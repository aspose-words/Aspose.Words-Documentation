OpenType은 PostScript와 TrueType에 비해 국제 언어 및 쓰기 시스템에 대한 더 나은 지원을 제공하기 위해 도입된 글꼴 형식입니다. OpenType의 레이아웃 기능은 일반적으로 OpenType 기능으로 알려져 있습니다. Aspose.Words. 모양.HarfBuzz 패키지는 OpenType 기능에 대한 지원을 제공합니다. Aspose.Words 사용 방법 `HarfBuzz` 텍스트 형성 엔진.

Aspose.Words text formr object를 사용하여 외부적으로 사용할 수 있습니다. 텍스트 셰이퍼는 텍스트에 대한 정보를 형성하는 글꼴과 computes를 나타냅니다. 문서는 일반적으로 여러 개의 글꼴을 나타냅니다 따라서 텍스트 셰이퍼 공장이 필요합니다. 이 패키지는 text shaper 공장의 구현을 포함 Aspose.Words.Layout.LayoutOptions.TextShaperFactory 속성.
