您可以轻松了解文档是否是使用 Aspose.Words for Python via .NET 生成的。您还可以找到用于创建特定输出文档的 Python via .NET 的 Aspose.Words 版本号。 Aspose.Words 直接将此信息写入生成的文档中。

版本号以 *YY.MM.N* 或 *YY.M.N* 格式指定，例如 23.12.0 或 24.1.0。<br />
这里 *YY* 是发布年份的最后两位数字，*M* 或 *MM* 是发布月份 (1-12)，*N* 是次要版本号。通常，*N* 为"0"。

{{% alert color="primary" %}}

请注意，您可以指示 Aspose.Words 从输出文档中删除此信息。

{{% /alert %}}
