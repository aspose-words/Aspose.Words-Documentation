亚洲打字法(Asian Typtography)是以亚洲语言书写的文件中的文本段落的一组选项.

Aspose.Words 支持使用 [ParagraphFormat](https://reference.aspose.com/words/net/aspose.words/paragraphformat/) 阶级及其一些属性。

## 自动调整亚洲文本和拉丁文本之间的空间或数字

如果您正在用东亚文本和拉丁文本设计一个模板,想要通过控制两种文本之间的空格来增强表格模板的外观,您可以配置您的表格模板,以自动调整这两种文本之间的空格. 要做到这一点,你可以使用 [AddSpaceBetweenFarEastAndAlpha](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getAddSpaceBetweenFarEastAndAlpha) 和 [AddSpaceBetweenFarEastAndDigit](https://reference.aspose.com/words/java/com.aspose.words/paragraphformat/#getAddSpaceBetweenFarEastAndDigit) 属性 `ParagraphFormat` 课。
