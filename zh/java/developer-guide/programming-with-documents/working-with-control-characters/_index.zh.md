Microsoft Word 文档可能包含具有特殊含义的各种字符。 通常它们用于格式化目的,不以正常模式绘制. 您可以单击标准工具栏上的显示/隐藏格式标记按钮,使其可见。

有时可能需要在文本中添加或删除字符 。 例如,在从文件中按程序获取文本时, Aspose.Words 保存大多数控件字符,所以如果需要使用此文本,您可能应该删除或替换这些字符。

那个 [ControlChar](https://reference.aspose.com/words/java/com.aspose.words/controlchar/) class 是代表文档中经常遇到的控制字符的常数的寄存器。 它既提供了字符集版本,也提供了相同常数的字符串版本. 例如,字符串 [LineBreak](https://reference.aspose.com/words/java/com.aspose.words/controlchar/#LINE-BREAK) 和 char 键 [LineBreakChar](https://reference.aspose.com/words/java/com.aspose.words/controlchar/#LINE-BREAK_CHAR) 具有同样的价值。
