大多数对象属性在 Aspose.Words API 表示一些测量,如宽度或高度、边距和各种距离,接受点值,1英寸等于72点。 有时这不方便,点数需要转换成其他单位.

Aspose.Words 提供 [ConvertUtil](https://reference.aspose.com/words/java/com.aspose.words/convertutil/) class 提供各种测量单位之间转换的辅助函数。 它能够将英寸,像素和毫米转换成点,将英寸和像素转换成点,并将像素从一个分辨率转换成另一个分辨率. 将像素转换成点,反之亦然,可以在96 dpi(每英寸)分辨率或指定的 dpi 分辨率下进行.

那个 **ConvertUtil** 分类在设置各种页面属性时特别有用,因为例如,英寸比分数更常见。
