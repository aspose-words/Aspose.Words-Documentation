## 临时文件文件夹

Aspose.Words (单位:千美元) Java 使用您的计算机上的临时目录,该目录通过 `java.io.tmpdir` 系统属性。 Aspose.Words (单位:千美元) Java 写入临时文件,然后在渲染完成后立即删除这些文件.

`java.io.tmpdir` 是一个标准 Java 系统属性,也用于基于磁盘的存储策略。 它决定了JVM撰写临时文件的地点.
