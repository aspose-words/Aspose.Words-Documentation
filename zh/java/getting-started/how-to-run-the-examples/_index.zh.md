## 从 GitHub

所有实例 Aspose.Words (单位:千美元) Java 被托管于 [吉图布](https://github.com/aspose-words/Aspose.Words-for-Java)。 。 。 您可以使用您最喜欢的 Github 客户端复制仓库, 或者下载 ZIP 文件 [这里](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip)。 。 。 。

提取 ZIP 文件的内容到您计算机上的任何文件夹 。 所有实例都位于 **Examples** 文件夹。

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

项目使用 Maven 建设系统. 任何现代的IDE都可以轻松地打开或导入项目及其依赖性. 下面我们教你如何使用流行的IDE来构建和运行实例.

## IntelliJ IDEA

点击 **File** 菜单和选择 **Open**。 。 。 。 浏览工程文件夹并选择 **pom.xml (英语)** 文档。

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

它会打开项目并自动下载依赖性. 从工程标签中浏览示例 **锡克/主/贾瓦** 文件夹。 要运行一个示例,只需右键点击文件并选择"Run...",该示例将被执行,输出将显示在控制台输出窗口的构建中.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

点击 **File** 菜单和选择 **Import**。 。 。 。 选择 **Maven** - 现有 Maven 项目。

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

浏览您克隆或下载的文件夹 GitHub 选择 **pom.xml (英语)** 文档。

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

它会打开项目并自动下载依赖性. 从软件包浏览器标签中浏览示例 **锡克/主/贾瓦** 文件夹。 要运行示例,只需右键单击文件并选择 **Run As** - 怎么样? **Java Application**, 示例将执行,输出将在控制台输出窗口中显示。

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

点击 **File** 菜单和选择 **Open Project**。 。 。 。 浏览您克隆或下载的文件夹 GitHub。 。 。 。 图标 **Examples** 文件夹将显示其 a Maven 项目。 选择示例并打开。

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

它会打开项目并自动下载依赖性. 从项目标签中浏览示例 **源软件包**。 。 。 。 要运行示例,只需右键单击文件并选择 **Run File**, 示例将执行,输出将在控制台输出窗口中显示。
