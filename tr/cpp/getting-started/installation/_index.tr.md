Makinenizin uygun olduğundan emin olun [sistem gereksinimleri](/words/cpp/system-requirements/) başlamadan önce.

Bu makalede, nasıl kurulacağı açıklanmaktadır Aspose.Words için C++ bilgisayarında.

## Yükleme veya Güncelleme Aspose.Words için C++ Manage NuGet Paketlerini Kullanma {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet indirmenin ve kurmanın en kolay yoludur [Aspose.Words için C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. Bunu yapmak için şu adımları izleyin:

1. Paket yöneticisi'ni açmak için menüden *Microsoft Visual Studio* ve *Manage NuGet Packages*'yi açın
2. "Aspose" veya "aspose" yi arayın.kelimeler" istenen yönü bulmak için API<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. İstediğiniz konumu seçin API ve "Yükle" ye tıklayın

Seçilen API indirilecek ve projenizde referans alınacaktır.

Ayrıca güncelleyebilirsiniz Aspose.Words için C++ *Manage NuGet Packages* kullanarak istediğiniz sürüme.

## Yükleme veya Güncelleme Aspose.Words için C++ Package Manager Console kullanarak

Ayrıca yükleyebilir veya güncelleyebilirsiniz Aspose.Words için C++ *Package Manager Console* kullanarak APIs. Bunu yapmak için şu adımları izleyin:

1. Paket yöneticisi konsolunu açmak için menüden *Microsoft Visual Studio* ve *Package Manager Console* 'yi açın
2. `Install-Package Aspose.Words.Cpp -Version 21.3.0` komutunu yazın ve uygulamanıza en son tam sürümü yüklemek için Enter tuşuna basın
3. İndirildikten sonra onay mesajlarını göreceksiniz<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. Artık Aspose.Words 'ın sizin için uygulamanıza başarıyla eklendiğini ve başvurulduğunu görebilirsiniz

*Package Manager Console* 'de, Aspose.Words paketindeki güncellemeleri kontrol etmek ve varsa bunları yüklemek için `Update-Package Aspose.Words.Cpp` komutunu da kullanabilirsiniz.

## Yüklemek Aspose.Words için C++ Manuel olarak

İndirebilirsin [Aspose.Words için C++ NuGet Galeriden](https://www.nuget.org/packages/Aspose.Words.Cpp/) bilgi bölümündeki "Manuel indirme" bağlantısını tıklayarak. Çevrimdışı API Belgeleri indirilen ZIP paketinde de bulunabilir.

Kullanmak için Aspose.Words için C++ uygulamalarınızda şu adımları izleyin:

1. En son dosyaları indirin
2. Klasörü bir üretim ortamına sıkıştırın
3. Kullanmak için Aspose.Words için C++, projenizdeki Ekle ve Lib klasörlerine bakın
