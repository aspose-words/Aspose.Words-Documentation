Python via .NET için Aspose.Words kullanarak bir belgenin üretilip üretilmediğini kolayca anlayabilirsiniz. Belirli bir çıktı belgesi oluşturmak için kullanılan Python via .NET için Aspose.Words sürüm numarasını da bulabilirsiniz. Aspose.Words bu bilgiyi doğrudan oluşturulan belgeye yazar.

Sürüm numarası *YY.MM.N* veya *YY.M.N* biçiminde belirtilir; örneğin 23.12.0 veya 24.1.0.<br />
Burada *YY*, yayın yılının son iki rakamıdır, *M* veya *MM*, yayın ayını (1-12) ve *N*, ikincil sürüm numarasını gösterir. Genellikle *N* "0"dır.

{{% alert color="primary" %}}

Aspose.Words'e bu bilgiyi çıktı belgelerinden kaldırması talimatını verebileceğinizi unutmayın.

{{% /alert %}}
