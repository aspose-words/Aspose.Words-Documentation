Bir belge Aspose.Words'ın kullanılarak üretildiğini kolayca bilirsiniz. Java Sadece belirli bir çıktı belgesi oluşturmak için kullanılan Aspose.Words sürüm numarasını da bulabilirsiniz Java. Aspose.Words doğrudan bu bilgiyi oluşturulan belgeye yazar.

Sürüm numarası *YY.MM.N* veya *YY.M.N* biçiminde belirtilmiştir, örneğin 23.12.0 veya 24.1.0.<br />
Burada *YY* yayın yılının son iki hanesidir, *M* veya *MM* yayın ayıdır (1-12) ve *N* küçük sürüm numarasıdır. Genellikle, *N* "0"

{{% alert color="primary" %}}

Şunu not edin: Aspose.Words 'nin bu bilgiyi çıktı belgelerinden değiştirmesi veya çıkarması için talimat veremezsiniz.

{{% /alert %}}
