The Credentials element contains information about user name and password to use to connect to a SharePoint site. You can omit it to use the current user credentials.Â Please note that storing user name and password in a report definition file may be insecure.

|Name|Cardinality|Has Children|Description|
| :- | :- | :- | :- |
|UserName|1|No|User name for connecting to a SharePoint server.|
|Password|1|No|Password for connecting to a SharePoint server.|
