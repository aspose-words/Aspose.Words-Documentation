「你可以輕易地知道一篇文件是否使用 Aspose.Words為 .NET 建立的。」 您也可以在 .NET 中找到特定輸出文件所用的 Aspose.Words 版本號。Aspose.Words 會直接將此資訊寫入生成的檔案中。

版本號以 *YY.MM.N* 或 *YY.M.N* 的格式指定，例如 23.12.0 或 24.1.0。<br />
這裡 *YY* 是發行年份的最後兩個數字， *M* 或 *MM* 是發行月份（1-12），以及 *N* 是次要版本號碼。 「通常，*N*是」0。「

{{% alert color="primary" %}}

請注意，您可以指示 Aspose.Words 从输出文档中删除此信息。

{{% /alert %}}
