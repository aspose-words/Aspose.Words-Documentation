為 .NET 的 Aspose.Words 不需要任何第三方產品，例如 Microsoft Word 來安裝。 Aspose.Words 本身是一個引擎，用於建立、修改、轉換及在各種格式的文件中呈现文件，包括 Microsoft Word 文檔。
