### שינוי שם הרישיון

שם הרישיון לא צריך להיות "Aspose.Words.Python.NETבבקשה." אתה יכול לקרוא את זה לאהוב ולהשתמש בשם זה בעת הגדרת רישיון ביישום שלך.

### ""לא יכול למצוא שם רישיון" למעט

בעת רכישת והורדה של רישיון, אתר As נניח שם את קובץ הרישיון *Aspose.Words.Python.NET.* אתה מוריד את קובץ הרישיון באמצעות הדפדפן שלך. במקרה זה, כמה דפדפנים מזהים את קובץ הרישיון כ-XML ונספח הסיומת .xml אליו, כך ששם הקובץ המלא במחשב שלך הופך *Aspose.Words.Python.NET.XML*

מתי Microsoft Windows מוגדר כדי להסתיר הרחבות עבור סוגי קבצים ידועים (ללא מאמץ, זה ברירת המחדל ברובם. Windows התקנות, קובץ הרישיון יופיע כ *Aspose.Words.Python.NET.* Windows Explorer אתה כנראה חושב שזה שם הקובץ האמיתי ולקרוא [set_license](https://reference.aspose.com/words/python-net/aspose.words/license/set_license/) עובר את זה *Aspose.Words.Python.NET.lic* אבל אין קובץ כזה, לכן יוצא הדופן.

כדי לפתור את הבעיה, שם מחדש את הקובץ כדי להסיר את הסיומת הבלתי נראית .xml. אנו גם ממליצים לך להשבית את האפשרות "הרחבות הייד" ב Microsoft Windows.

## שימוש במוצרים מרובים

אם אתה משתמש במוצרים מרובים של As נניח ביישום שלך, כגון Aspose.Words ו `Aspose.Cells`, הנה כמה טיפים שימושיים:

* הגדר את הרישיון עבור כל מוצר בנפרד. גם אם יש לך קובץ רישיון יחיד עבור כל המרכיבים, לדוגמה, "כאמור.ל." אתה עדיין צריך להתקשר [set_license](https://reference.aspose.com/words/python-net/aspose.words/license/set_license/) בנפרד עבור כל מוצר נניח שאתה משתמש ביישום שלך.
* השתמש ב שם הרישיון המלא. כל אחד נניח שלמוצר יש **License** בכיתה בשמה הפרטי שלו. לדוגמה, Aspose.Words יש [aspose.words.License](https://reference.aspose.com/words/python-net/aspose.words/license/) ו `Aspose.Cells` יש **נניח.תאים. רישיון רישיון רישיון** מעמד. באמצעות שם הכיתה מוסמך לחלוטין מאפשר לך להימנע בלבול לגבי איזה רישיון חל על איזה מוצר.
