לפעמים צריך להסיר שדה מן המסמך. זה עשוי להתרחש כאשר יש להחליף אותו בסוג שדה שונה או כאשר השדה כבר לא נחוץ במסמך. למשל `TOC` שדה בעת שמירת HTML

כדי להסיר שדה מוכנס לתוך מסמך באמצעות [DocumentBuilder.insert_field](https://reference.aspose.com/words/python-net/aspose.words/documentbuilder/insert_field/), שימוש חוזר [Field](https://reference.aspose.com/words/python-net/aspose.words.fields/field/) אובייקט, המספק את הנוחות [remove](https://reference.aspose.com/words/python-net/aspose.words.fields/field/remove/#default) שיטה להסיר בקלות את השדה מן המסמך.
