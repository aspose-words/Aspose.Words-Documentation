ברוכים הבאים Aspose.Words עבור Java!

Aspose.Words היא ספרייה בכיתה חוצה פלטפורמות המאפשרת היישום שלך לבצע מגוון רחב של משימות עיבוד מסמכים. אנו קוראים לזה עורך המסמכים הניתן לתוכנה.

שימוש Aspose.Words, אתה יכול ליצור, לשנות, להמיר, להפעיל ולהדפיס מסמכים מבלי להסתמך על יישומים של צד שלישי, כגון: Microsoft Word, או Office Automation.

ראוי לציין כי אתה יכול להמיר מסמך מכל פורמט עומס נתמך לכל פורמט הצלה נתמך, למשל,. Microsoft Word פורמטים ל- PDF, HTML או Markdown, HTML ל- DOCX וכן הלאה. קרא עוד על המרת מסמכים [המרת מסמך](/words/he/java/convert-a-document/) סעיף.

אפשרות פופולרית נוספת בספריה שלנו היא *דיווחים ודואר*. המשתמשים שלנו מייצרים מאוד מורכבים באופן אישי מותאם פיננסית, רפואית ועוד *Reports*. בעשותו כן, המסמך שנוצר נראה כמו שיצרתם את הדו"ח באופן ידני באמצעות עורך טקסט רגיל. אפשרות נוספת, *Mail*, מספק את היכולת לייצר במהירות אלפי ומיליוני מסמכים פשוטים אך עדיין מבוססים על תבניות אישיות.

במאמץ לעשות את API ברור וברור ככל האפשר, החלטנו להכיר ולכבד את המשותף. Java שיטות פיתוח פלטפורמה כך, Aspose.Words עבור Java עקבו אחרי coding Guidelines Accepted By Java מפתחים.

## היתרונות של שימוש Aspose.Words

שימוש Aspose.Words עבור Java הפרויקט שלך נותן לך את היתרונות הבאים:

* ביצועים ודרגות
* כרך עשיר
* עצמאות פלטפורמה
* עצמאות מיישומים של צד שלישי
* עקומת למידה מינימלית

הדיאגרמה הבאה מציגה את התכונות העיקריות של Aspose.Words עבור Java ואיך הם מתייחסים אחד לשני.

### סדרה עשירה של תכונות

Aspose.Words עבור Java ניתן לחלק את התכונות לארבעה תחומים עיקריים.

#### הגייה

היכולת להמיר בקלות וביעילות מסמכים היא אחת התכונות העיקריות של Aspose.Words.

מקיף [לטעון, לחסוך ולהמיר](/words/he/java/loading-saving-and-converting/) עם [מסמך](/words/he/java/supported-document-formats/) מאפשר למשתמשים להמיר מסמכים [פורמט פופולרי](https://reference.aspose.com/words/java/com.aspose.words/loadformat/) to to [another](https://reference.aspose.com/words/java/com.aspose.words/saveformat/), לדוגמה, מ- DOCX ל- PDF, JPEG, או Markdown.

#### Document Object Model (DOM)

גישה מתודולוגית דרך עשיר API לכל רכיבי המסמך והפורמטיבי מאפשר ליצור, לשנות, לחלץ, להעתיק, לפצל, להצטרף ולהחליף את תוכן המסמך. גלה עוד ב [Aspose.Words Document Object Model (DOM)](/words/he/java/aspose-words-document-object-model/) סעיף.

#### Rendering

אנו משתמשים במונח [Rendering](/words/he/java/rendering/) כדי לתאר את התהליך של המרת מסמך לתוך פורמט קובץ או מדיום, דמיין או שיש לו מושג של דפים.

ניתן להמיר מסמכים שלמים או דפים ספציפיים ל- PDF או XPS כדי ליצור מסמכים בצד השרת, להדפיס אותם, או ליצור דפים. וכל זה עם נאמנות גבוהה - בדיוק כמו Microsoft Word היה עושה את זה.

#### דיווח

ליצור מסמכים או דוחות מאפס או על ידי מילוי תבניות עם נתונים ממקורות נתונים או אובייקטים עסקיים. לקבלת מידע נוסף על הדיווח והבאת נתונים ממאגרי מידע, ראה את [Mail Merge דיווח ו](/words/java/mail-merge-and-reporting/) ו [תגית:LINQ](/words/java/linq-reporting-engine/) חלקים.

### פלטפורמות עצמאות

Aspose.Words עבור Java מכסה את כל Java סביבות פיתוח ופלטפורמות פריסה. שלנו API ניתן להשתמש כדי לפתח יישומים עבור מגוון רחב של מערכות הפעלה, כגון Windows, Linux, Mac OS Android, פלטפורמות שונות.

אם הפלטפורמה שלך יכולה לרוץ סטנדרטי <span notrans="<span notrans=" Java"=""></span>" זה יכול לרוץ Aspose.Words עבור Java. אם היישום שלך יכול להתקשר סטנדרטי Java <span notrans="<span notrans=" API"=""></span>" אין לך בעיות להתקשר Aspose.Words עבור Java API מתוך היישום שלך.

כרגע הבסיס שלנו הוא JDK1.7 וחדש. אנחנו עדיין תומכים ב-JDK1.6.

יש לנו שלוש הודעות שונות בהתבסס על אותו בסיס קוד:

* Aspose.Words עבור Java
* Aspose.Words עבור Android באמצעות Java
* Aspose.Words עבור JasperReports

בעוד Java ו Android הודעות נספו עבור הפלטפורמה המקבילה, Aspose.Words עבור JasperReports הוא רק עטיפה עבה סביב "רגיל" Aspose.Words עבור Java. זה רק לנוחות. אתה יכול גם לשלב באופן ידני שלך JasperReports במילים נניח - באופן ספציפי שמתאים לצרכים שלך.

### עצמאות מבקשות אחרות

Aspose.Words אין צורך באפליקציות של צד שלישי, למשל. Microsoft יש להתקין את המשרד כדי לעבוד. כל המרכיבים הם עצמאיים לחלוטין. זה עושה Aspose.Words אלטרנטיבה נהדרת לאוטומציה מבחינת אבטחה, יציבות, קיבולת / מהירות, מחיר ותכונות לעבודה עם מסמכים ומשימות קשורות.

### תגית: Scalability

Aspose.Words נועד לבצע רבות הן בשרת או לקוח. Aspose.Words הוא יחיד Java צנצנת שניתן לפרוס עם כל Java יישום פשוט העתק אותו. ניתן גם להוסיף `HarfBuzz` תוסף עבור Advanced תכונות פונטיות אם אתה צריך את זה (קשור בצנצנת נפרדת). תקן Standard Java ImageIO הרחבה של JAI יכולה גם להוסיף לפלטפורמה שלך כדי לתמוך נוסף ImageIO קודים (אין צורך במודרניזציה) Java סביבות).

עם Aspose.Words, אתה יכול פשוטו כמשמעו לייצר אלפי מסמכים תוך דקות, אשר כוללים פתיחת מסמך, שינוי תוכן ופורמטיבי או מפולג אותו עם נתונים והצלתו. אפילו המסמכים המורכבים ביותר נפתחים וננצלים במכונת RAM של P42.4Ghz 1Gb בשבריר שנייה.

Aspose.Words הוא רב-קריאה בטוחה כל עוד חוט אחד עובד על מסמך בזמן. זהו תרחיש טיפוסי שיש לו חוט אחד העובד על מסמך אחד. חוטים שונים יכולים לעבוד בבטחה על מסמכים שונים בו זמנית.

### לימוד מינימלי

למרות שיש יותר מ-150 כיתות ציבוריות והערכות <span notrans="<span notrans=" Aspose.Words"=""></span>" עקומת הלמידה היא מינימלית כי Aspose.Words API תוכנן בקפידה עם המטרות הבאות:

* Borrow Best Practice from Anotherידוע APIכמו Microsoft Word אוטומציה אוטומציה
* Borrow Best Practices from the Java המונחים Design
* • לספק איזון של קלות שימוש ומניפולציה מפורטת של רכיבי מסמך

מפתחים שהשתמשו בעבר Microsoft Word אוטומציה בפרויקטים שלהם תמצא את השמות וההתנהגות של כיתות רבות, שיטות ונכסים מוכרים.

[Document](https://reference.aspose.com/words/java/com.aspose.words/document/), [Paragraph](https://reference.aspose.com/words/java/com.aspose.words/paragraph/), [Bookmark](https://reference.aspose.com/words/java/com.aspose.words/bookmark/), [Range](https://reference.aspose.com/words/java/com.aspose.words/range/), [Section.PageSetup](https://reference.aspose.com/words/java/com.aspose.words/section/#getPageSetup) ו [Paragraph.ParagraphFormat](https://reference.aspose.com/words/java/com.aspose.words/paragraph/#ParagraphFormat) הם חלק Aspose.Words שיעורים. במקביל, Aspose.Words הוא שונה מאוד מן Microsoft Word מודל אובייקטים בו הוא מייצג מסמך כעץ של אובייקטים, כמו XML DOM עץ. אם עבדת עם XML DOM ספריה, תמצאו קל להבין ולעבוד עם Aspose.Words.
