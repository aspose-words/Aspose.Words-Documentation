אתה יכול בקלות לדעת אם מסמך מיוצר על ידי שימוש Aspose.Words עבור Java. ניתן גם למצוא את מספר הגירסה של Aspose.Words עבור Java זה שימש ליצירת מסמך פלט מסוים. Aspose.Words כותב ישירות את המידע הזה במסמך שנוצר.

מספר הגירסה מוגדר בפורמט *YY.MM.N* או *YY.M.N*, 23.12.0 או 24.1.0.<br/>
כאן *YY* שתי הספרות האחרונות של שנת השחרור *M* או *MM* חודש השחרור (1–12) *N* הוא מספר השחרור הקטן. בדרך כלל, *N* זה "0".

{{% alert color="primary" %}}

שים לב שאי אפשר להורות Aspose.Words כדי לשנות או להסיר מידע זה ממסמכים.

{{% /alert %}}
