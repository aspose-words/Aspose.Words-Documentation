## להורדה GitHub

כל הדוגמאות Aspose.Words עבור Java מתארחים על [Github](https://github.com/aspose-words/Aspose.Words-for-Java). אתה יכול גם לשחזר את המאגר באמצעות הלקוח האהוב עליך Github או להוריד את קובץ ZIP מ [כאן](https://github.com/aspose-words/Aspose.Words-for-Java/archive/master.zip).

קבל את התוכן של קובץ ZIP לכל תיקיה במחשב שלך. כל הדוגמאות ממוקמות **Examples** תיקיה.

![how-to-run-examples-aspose-words-java-1](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-1.png)

הפרויקט משתמש Maven לבנות מערכת כל IDE מודרני יכול בקלות לפתוח או לייבא את הפרויקט ואת התלויות שלו. להלן אנו מראים לך כיצד להשתמש ב- IDEs פופולריים כדי לבנות ולהפעיל את הדוגמאות.

## IntelliJ IDEA

לחץ על **File** תפריט ובחר **Open**. רוץ לתיקיית הפרויקט ובחר את **pom.xml** קובץ.

![how-to-run-examples-aspose-words-java-2](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-2.png)

הוא יפתח את הפרויקט ויורד את התלויות באופן אוטומטי. מתוך כרטיסיית הפרויקט, עיין בדוגמאות ב **src/main/java** תיקיה. כדי להפעיל דוגמה, רק לחיצה נכונה על הקובץ ולבחור "רוץ"., הדוגמה תיעשה והפלט יופיע בחלון הפלט שנבנה בקונסולה.

![how-to-run-examples-aspose-words-java-3](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-3.png)

## Eclipse

לחץ על **File** תפריט ובחר **Import**. בחר בחר **Maven** קיים Maven פרויקטים.

![how-to-run-examples-aspose-words-java-4](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-4.png)

הזדרזו לתיקיה שאתם משטפים או מורידים ממנה GitHub ובחר **pom.xml** קובץ.

![how-to-run-examples-aspose-words-java-5](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-5.png)

הוא יפתח את הפרויקט ויורד את התלויות באופן אוטומטי. מכרטיסית החבילה Explorer, עיין בדוגמאות ב **src/main/java** תיקיה. כדי להפעיל דוגמה, פשוט ללחוץ על הקובץ ולבחור **Run As** - - **Java Application**, הדוגמה תבוצע והפלט יופיע בחלון הפלט שנבנה בקונסולה.

![how-to-run-examples-aspose-words-java-6](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-6.png)

## NetBeans

לחץ על **File** תפריט ובחר **Open Project**. הזדרזו לתיקיה שאתם משטפים או מורידים ממנה GitHub. הסמל של **Examples** התיקיה תוכיח שהיא Maven פרויקט. בחרו דוגמאות ולפתוח אותן.

![how-to-run-examples-aspose-words-java-7](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-7.png)

הוא יפתח את הפרויקט ויורד את התלויות באופן אוטומטי. מכרטיסיית הפרויקטים, עיין בדוגמאות **חבילות מקור**. כדי להפעיל דוגמה, פשוט ללחוץ על הקובץ ולבחור **Run File**, הדוגמה תבוצע והפלט יופיע בחלון הפלט שנבנה בקונסולה.

![how-to-run-examples-aspose-words-java-8](/words/java/how-to-run-the-examples/how-to-run-examples-aspose-words-java-8.png)
