{{% alert color="primary" %}}

Aspose.Words יש ממשק ליצירת ברקודים מותאמים אישית שהופך אותו מאוד קל לשימוש [Aspose.Words](https://products.aspose.com/words/java/) ו [תגית. ברקוד](https://products.aspose.com/barcode/java/) יחד כדי להפוך תמונות ברקוד במסמכים פלט. לדוגמה, באפשרותך לטעון מסמך DOC, OOXML או RTF המכיל מסמך המכיל `DISPLAYBARCODE` שדה לתוך Aspose.Wordsלספק את היישום שלך של גנרטור ברקוד מותאם אישית ולהציל פורמטי דף קבועים כגון PDF, XPS וכו'. טיפוסי `DISPLAYBARCODE` שדה יש את הסינמס הבא:

{ `DISPLAYBARCODE` "SomeData" QR \h 720 }

{{% /alert %}}

להלן מחולל קוד מדגם המנף את `Aspose.BarCode` API. דוגמה זו מראה כיצד להוסיף תמונות ברקוד `DISPLAYBARCODE` מיקום השדה במסמך Word באמצעות Aspose.Words ו `Aspose.BarCode` API.
