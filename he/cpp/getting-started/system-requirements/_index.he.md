## סביבת פיתוח

### תלות נוספת ב Linux

Aspose.Words עבור C++ על Linux תלוי ב [פונטקונפיג](https://www.freedesktop.org/wiki/Software/fontconfig/) קבצים בינאריים הן ספרייה דינמית וכלי. בבקשה להתקין אותו לפני השימוש:

1. התקנת גופן על Ubuntu או Debian</br>
`sudo apt install libfontconfig fontconfig`
2. התקנת גופן על Fedora או CentOs</br>
   `sudo yum install fontconfig`

### תלות נוספת ב macOS

Aspose.Words עבור C++ על macOS תלוי ב [פונטקונפיג](https://www.freedesktop.org/wiki/Software/fontconfig/) קבצים בינאריים הן ספרייה דינמית וכלי. בבקשה להתקין אותו לפני השימוש:

התקנת פונקונפיג ב macOS עם [homebrew](https://brew.sh)</br>
`brew install fontconfig`
