לוודא שלך מכונה פוגש את [דרישות מערכת](/words/cpp/system-requirements/) לפני שתתחיל.

מאמר זה מסביר כיצד להתקין Aspose.Words עבור C++ במחשב שלך.

## התקן או עדכן Aspose.Words עבור C++ באמצעות ניהול NuGet חבילות {#install-or-update-aspose-words-for-cpp-using-nuget}

NuGet היא הדרך הקלה ביותר להוריד ולהתקין [Aspose.Words עבור C++](https://www.nuget.org/packages/Aspose.Words.Cpp) APIs. לשם כך, בצע את השלבים הבאים:

1. פתח *Microsoft Visual Studio* ו *Manage NuGet Packages* מהתפריט כדי לפתוח את מנהל החבילות
2. חפש "אספוז" או "אספוז".מילים " כדי למצוא את האספוז הרצוי API<br>
   <img src="aspose-words-nuget.png" alt="aspose-words-cpp-nuget" style="width:800px"/>
3. בחר את האספוז הרצוי API ולחץ על "התקן"

API שנבחר יוריד ויופנה לפרויקט שלך.

ניתן גם לעדכן Aspose.Words עבור C++ לגרסה הרצויה באמצעות *Manage NuGet Packages*.

## התקן או עדכן Aspose.Words עבור C++ באמצעות Package Manager Console

ניתן גם להתקין או לעדכן Aspose.Words עבור C++ APIs באמצעות *Package Manager Console*. לשם כך, בצע את השלבים הבאים:

1. פתח *Microsoft Visual Studio* ו *Package Manager Console* מהתפריט כדי לפתוח את מסוף מנהל החבילות
2. הקלד את הפקודה `Install-Package Aspose.Words.Cpp -Version 21.3.0` ולחץ להיכנס כדי להתקין את המהדורה המלאה האחרונה ביישום שלך
3. לאחר ההורדה, תראה הודעות אישור<br>
   {{% alert color="primary" %}}If you are unfamiliar with the [Aspose EULA](https://about.aspose.com/legal/eula/), it is recommended to read the license referenced in the URL.{{% /alert %}}
4. עכשיו אתה עשוי לגלות כי Aspose.Words נוספה בהצלחה הפניה ביישום שלך בשבילך

ב *Package Manager Console*, אתה יכול גם להשתמש בפקודה `Update-Package Aspose.Words.Cpp` כדי לבדוק אם קיימים עדכונים לחבילה Aspose.Words ולהתקין אותם אם קיימים.

## התקן Aspose.Words עבור C++ באופן ידני

ניתן להוריד [Aspose.Words עבור C++ מ NuGet גלריה](https://www.nuget.org/packages/Aspose.Words.Cpp/) על ידי לחיצה על הקישור "הורדה ידנית" בקטע מידע. לא מקוון API תיעוד ניתן למצוא גם בחבילה שהורדת ZIP.

כדי להשתמש ב Aspose.Words עבור C++ ביישומים שלך, בצע את השלבים הבאים:

1. הורד את הקבצים העדכניים ביותר
2. פתח את התיקיה לסביבת ייצור
3. כדי להשתמש ב Aspose.Words עבור C++, עיין בתיקיות כלול וליב בפרויקט שלך
