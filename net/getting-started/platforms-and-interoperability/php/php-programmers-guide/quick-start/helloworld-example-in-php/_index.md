## Aspose.Words - HelloWorld Example

Hello World Example

**PHP Code**

{{< highlight php >}}
        $dataDir = '.';
        $doc = new \COM("Aspose.Words.Document");
        $builder = new \COM("Aspose.Words.DocumentBuilder");

        $builder->Write("Hello world!");
        $doc->Save($dataDir . "/HelloWorld Out.docx");
